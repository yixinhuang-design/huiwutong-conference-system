## 📝 集成方式

### 最小集成 (仅消除错误)
```html
<head>
  <!-- 添加这行 -->
  <script src="../js/extension-handler.js"></script>
</head>
```

### 完整集成 (包括模态框)
```html
<head>
  <script src="../js/extension-handler.js"></script>
  <link rel="stylesheet" href="../css/modal-styles.css">
  <script type="module">
    import Modal from '../js/components/Modal.js';
    app.component('Modal', Modal);
  </script>
</head>
```

### 使用模态框
```vue
<Modal 
  :visible="showModal"
  title="确认操作"
  @confirm="handleConfirm"
  @cancel="handleCancel"
>
  <p>确定要执行此操作吗？</p>
</Modal>
```

## 🚨 故障排除

| 问题 | 解决方案 |
|------|---------|
| 仍然看到 404 | 1. 清除缓存 (Ctrl+Shift+Delete)<br>2. 强制刷新 (Ctrl+F5)<br>3. 重启服务器 |
| Extension Handler 未加载 | 1. 检查脚本标签位置<br>2. 检查路径是否正确<br>3. 打开控制台查看错误 |
| 模态框样式混乱 | 1. 确认 modal-styles.css 已加载<br>2. 检查 CSS 加载顺序<br>3. 使用浏览器检查器查看样式 |
