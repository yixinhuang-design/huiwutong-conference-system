# 座位布局自定义配置 - 功能说明

**实现日期**: 2026-01-24
**页面**: seating-mgr.html (智能排座系统)

---

## 🎯 功能概述

实现了完整的座位布局自定义配置功能，用户可以：
1. 创建新的座位区域
2. 编辑现有座位区域
3. 自定义行数、列数
4. 设置旋转角度
5. 配置过道位置和宽度
6. 设置固定座位
7. 实时预览布局效果

---

## 📋 详细功能说明

### 1. 座位区配置弹窗

**触发方式**：
- 点击"添加座位区"按钮
- 点击已有区域的"编辑"按钮

**弹窗包含**：
- 基本信息配置
- 尺寸配置
- 角度配置
- 高级选项
- 实时预览
- 统计信息

---

### 2. 基本信息配置

#### 区域名称
- 输入框：文本输入
- 示例：A区 (VIP区域)、B区、C区
- 必填项

#### 区域类型
- 下拉选择：普通区域 / VIP区域
- VIP区域座位显示为粉色
- 影响前排座位的分配优先级

---

### 3. 尺寸配置

#### 行数配置
- **范围**: 1-20 行
- **控件**: `[-] [输入框] [+]`
- **默认**: 10 行
- **说明**: 使用+/-按钮调整，或直接输入数字

#### 列数配置
- **范围**: 1-30 列
- **控件**: `[-] [输入框] [+]`
- **默认**: 15 列
- **说明**: 使用+/-按钮调整，或直接输入数字

---

### 4. 角度配置

#### 角度滑块
- **范围**: 0° - 360°
- **步长**: 15°
- **控件**: 滑块 + 数字显示
- **实时预览**: 显示旋转后的效果

#### 快捷角度按钮
```
[0°] [45°] [90°] [180°]
```
- 点击快速设置常用角度
- 激活状态高亮显示

#### 角度预览
- 小方框显示旋转效果
- 实时响应滑块变化

---

### 5. 过道配置

#### 启用过道
- 复选框：勾选后启用过道配置
- 默认：关闭

#### 过道位置
- **左侧**: 过道在座位区左侧
- **中间**: 过道在座位区中间
- **右侧**: 过道在座位区右侧

#### 过道宽度
- **范围**: 1-5 列
- 默认：1列
- 说明：过道占用的列数

#### 过道显示
- 过道座位显示为灰色
- 状态标记为 `aisle`
- 不参与人员分配

---

### 6. 固定座位设置

#### 启用固定座位
- 复选框：勾选后启用固定座位配置
- 默认：关闭

#### 固定座位位置
- **输入框**: 文本输入
- **支持格式**:
  - 单个座位：`1排1号`
  - 多个座位：`1排1号, 1排2号, 2排5号`
  - 范围：`1排1-5号` （1排的1-5号）

#### 固定座位显示
- 固定座位显示黄色边框
- 边框加粗显示
- 标记为 `fixed`

---

### 7. 实时预览

#### 预览功能
- **布局预览**: 显示座位排列
- **旋转预览**: 按角度旋转显示
- **状态标识**:
  - VIP座位：粉色
  - 过道座位：灰色
  - 固定座位：黄色边框

#### 预览更新
- 任何配置变更都会实时更新预览
- 无延迟响应
- 帮助用户快速决策

---

### 8. 统计信息

#### 实时统计
```
总座位数: 150    (行 × 列)
实际座位: 145     (扣除过道)
过道占用: 5       (过道座位数)
```

#### 统计说明
- **总座位数**: 行数 × 列数
- **实际座位**: 总座位 - 过道占用
- **过道占用**: 标记为过道的座位数量

## 🎮 使用流程

### 创建新座位区

1. 点击"添加座位区"按钮
2. 弹窗打开，显示默认配置
3. 配置区域名称、类型、尺寸等
4. 可选：配置角度、过道、固定座位
5. 查看实时预览和统计信息
6. 点击"创建座位区"保存

### 编辑座位区

1. 在座位区列表中点击"编辑"按钮
2. 弹窗打开，加载现有配置
3. 修改配置参数
4. 查看实时预览效果
5. 点击"保存修改"更新

### 删除座位区

1. 在座位区列表中点击"删除"按钮
2. 确认删除操作
3. 座位区被移除

## 💡 配置示例

### 示例1：普通会议厅（无角度）
```
区域名称：主会场
区域类型：普通区域
行数：10
列数：15
角度：0°
过道：无
固定座位：无
```

### 示例2：VIP区域（前排居中）
```
区域名称：A区 (VIP区域)
区域类型：VIP区域
行数：5
列数：10
角度：0°
过道：无
固定座位：无
```

### 示例3：U型会议（45度旋转）
```
区域名称：U型讨论区
区域类型：普通区域
行数：8
列数：12
角度：45°
过道：无
固定座位：无
```

### 示例4：带过道的会议厅
```
区域名称：B区
区域类型：普通区域
行数：10
列数：20
角度：0°
过道：启用，中间，宽度2列
固定座位：无
```

### 示例5：带固定座位的主席台
```
区域名称：主席台
区域类型：VIP区域
行数：2
列数：8
角度：0°
过道：无
固定座位：启用
固定位置：1排1-8号（全部）
```
