### 5. 💻 seat-number-generator.js ⭐⭐⭐
**重要性：最高 | 代码量：400+行**

**包含的类**：

#### SeatNumberGenerator（座位编号生成器）
```javascript
new SeatNumberGenerator(rows, cols, mode)
  .generate(row, col)          // 单个座位号
  .generateAll()               // 批量生成
  .setMode(newMode)           // 改变模式
  .getModeName()              // 获取模式名称
```

#### SeatNumberConverter（座位编号转换器）
```javascript
new SeatNumberConverter(rows, cols)
  .convert(fromMode, toMode, seatNumber)  // 单个转换
  .convertAll(fromMode, toMode)           // 批量转换
  .getSeatPosition(mode, seatNumber)      // 查找位置
```

#### SeatingAreaBuilder（座位区域构建器）
```javascript
new SeatingAreaBuilder(id, name, rows, cols, type)
  .setNumberingMode(mode)    // 设置编号方式
  .addAisle(type, position)  // 添加过道
  .setStyle(styles)          // 设置样式
  .build()                   // 生成配置
```

**特点**：
- 完整注释和文档字符串
- 5个完整的使用示例
- 错误处理和验证
