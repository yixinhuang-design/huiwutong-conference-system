# 座位图动态展示改进说明

**改进日期**: 2026-01-24
**页面**: seating-mgr.html (智能排座系统)

---

## 🎯 改进目标

确保座位图根据用户添加的座位区布局动态展示，而不是使用硬编码的默认数据。

---

## ✅ 实施的改进

### 1. 座位区配置数据完整性

**问题**: `generateAreaFromConfig` 方法没有保存角度、过道等配置到座位区对象中。

**解决方案**: 在返回座位区对象时，保存所有配置信息：

```javascript
return {
    id: areaId,
    name: areaName,
    type: areaType,
    angle: config.angle || 0,              // ✅ 新增：角度配置
    hasAisle: config.hasAisle || false,    // ✅ 新增：是否有过道
    aislePosition: config.aislePosition || 'center',  // ✅ 新增：过道位置
    aisleWidth: config.aisleWidth || 1,    // ✅ 新增：过道宽度
    hasFixedSeats: config.hasFixedSeats || false,  // ✅ 新增：是否有固定座位
    rows: rows
};
```

---

### 2. 默认座位区数据完善

**问题**: `createDefaultSeatingAreas` 函数创建的默认座位区缺少配置字段。

**解决方案**: 为每个默认座位区添加完整的配置信息：

```javascript
{
    id: 'area_a',
    name: 'A区 (VIP区域)',
    type: 'vip',
    angle: 0,                    // ✅ 新增
    hasAisle: false,             // ✅ 新增
    aislePosition: 'center',     // ✅ 新增
    aisleWidth: 1,               // ✅ 新增
    hasFixedSeats: false,        // ✅ 新增
    rows: generateRows('A', 5, 15)
}
```

### 3. 座位区动态样式展示

**新增方法**: `getAreaStyle(area)`

**功能**: 根据座位区的配置动态生成样式

**实现**:
```javascript
getAreaStyle(area) {
    const style = {};

    // 应用角度旋转
    if (area.angle && area.angle !== 0) {
        style.transform = `rotate(${area.angle}deg)`;
        style.transformOrigin = 'center center';
    }

    return style;
}
```

**使用**:
```html
<div class="area-section"
     :style="getAreaStyle(area)">
    <!-- 座位区内容 -->
</div>
```

### 4. 单个座位动态样式

**新增方法**: `getSeatStyle(seat, area)`

**功能**: 根据座位状态和座位区类型动态生成单个座位的样式

**实现**:
```javascript
getSeatStyle(seat, area) {
    const style = {};

    // 过道座位样式
    if (seat.isAisle) {
        style.background = 'rgba(128, 128, 128, 0.2)';
        style.border = '2px dashed rgba(128, 128, 128, 0.4)';
        style.cursor = 'not-allowed';
        style.opacity = '0.5';
        return style;
    }

    // 已占用座位：样式在getSeatClass中处理
    if (seat.occupant) {
        return style;
    }

    // 预留座位
    if (seat.isReserved) {
        style.background = 'linear-gradient(135deg, #fbc2eb 0%, #a6c1ee 100%)';
        return style;
    }

    // 固定座位
    if (seat.isFixed) {
        style.border = '2px solid #FFD700';
        style.background = 'rgba(255, 215, 0, 0.1)';
        return style;
    }

    // VIP区域的空座位
    if (area.type === 'vip') {
        style.background = 'linear-gradient(135deg, rgba(240, 147, 251, 0.3) 0%, rgba(245, 87, 108, 0.3) 100%)';
        style.border = '1px solid rgba(240, 147, 251, 0.5)';
    }

    return style;
}
```

**使用**:
```html
<div class="seat"
     :style="getSeatStyle(seat, area)"
     :class="getSeatClass(seat)">
    {{ seat.number }}
</div>
```

## 🔧 技术实现细节

### 数据流程

```
用户配置座位区 (areaConfig)
        ↓
保存配置 (saveAreaConfig)
        ↓
生成座位区对象 (generateAreaFromConfig)
        ↓
添加到 seatingAreas 数组
        ↓
Vue 响应式更新视图
        ↓
座位图展示新座位区
```

### Vue响应式

- `seatingAreas` 是响应式数组
- 添加/编辑/删除座位区会自动触发视图更新
- 座位区的配置字段也是响应式的

### 样式优先级

1. CSS类（`:class`）处理基本状态：
   - `available` - 空座位
   - `occupied` - 已占用
   - `selected` - 已选中
   - `vip` - VIP座位
   - `reserved` - 预留座位
   - `aisle` - 过道座位
   - `locked` - 锁定座位

2. 内联样式（`:style`）处理动态配置：
   - 背景颜色
   - 边框样式
   - 旋转角度
   - 不透明度
