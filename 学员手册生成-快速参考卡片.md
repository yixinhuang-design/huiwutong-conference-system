## 💡 使用技巧

### 工作人员的10个技巧

1. **使用默认值节省时间**
   - 第1步直接用默认值，点"继续"

2. **编辑前预览后台数据**
   - 确保数据在后台已正确输入

3. **仅编辑必要项目**
   - 只改"培训须知"和"册尾"，其他项看看就行

4. **发现错误立即返回**
   - 不满意可随时返回第2步修改

5. **每次预览都检查名册**
   - 确保人数和分组正确

6. **下载后检查文件**
   - 生成后立即打开检查，及时发现问题

7. **分享时生成新链接**
   - 不要重复分享同一个链接

8. **使用标准模板更稳定**
   - 详细模板可能有排版问题

9. **保存备份很重要**
   - 至少保存2份副本

10. **遇到问题清除缓存**
    - 浏览器缓存可能导致问题

### 学员的7个技巧

1. **看目录快速定位**
   - 手册有5个部分，目录告诉你位置

2. **日程要看清时间**
   - 注意会议时间和地点

3. **名册可筛选分组**
   - 按钮可快速查看各分组人员

4. **下载后离线使用**
   - 下载PDF可在没有网络时查看

5. **分享给家人或同事**
   - 分享链接7天内有效

6. **如无法查看要求重新分享**
   - 链接过期需要重新生成

7. **需要打印选择彩色**
   - 彩色打印效果更好

## ⚠️ 常见错误及修复

### 错误1：编辑内容没保存
```
❌ 问题：点击编辑框编辑后，发现内容没保存
✅ 修复：
  1. 确认点击了"保存"按钮
  2. 检查浏览器是否禁用localStorage
  3. 尝试清除缓存重新编辑
```

### 错误2：PDF生成失败
```
❌ 问题：点击"生成PDF"后一直转圈
✅ 修复：
  1. 检查网络连接
  2. 刷新页面重试
  3. 用标准模板而非详细模板
  4. 联系技术支持
```

### 错误3：预览显示空白
```
❌ 问题：点击预览后弹窗显示空白
✅ 修复：
  1. 检查第2步是否完成编辑
  2. 返回第1步重新选择模板
  3. 清除浏览器缓存
```

### 错误4：名册人数不对
```
❌ 问题：预览名册时人数不是142人
✅ 修复：
  1. 回到后台检查学员数据
  2. 检查分组是否完整
  3. 在后台重新同步数据后重生成
```

### 错误5：分享链接失效
```
❌ 问题：分享给别人的链接打不开
✅ 修复：
  1. 检查是否超过7天有效期
  2. 重新点击"分享链接"生成新链接
  3. 确保收者网络正常
```

## 📈 性能优化建议

### 加快生成速度

1. **使用标准模板** - 比详细模板快10%
2. **减少编辑项** - 只编辑必要的2项
3. **清除浏览器缓存** - 定期清理
4. **关闭其他标签** - 释放内存

### 改善用户体验

1. **添加加载提示** - 告诉用户在等待
2. **分段生成** - 大型名册分几个手册
3. **预设模板** - 省去选择步骤
4. **快捷键支持** - 加快操作速度
