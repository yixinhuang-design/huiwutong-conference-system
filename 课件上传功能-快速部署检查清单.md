```
检查清单:

### 如果上传失败

```
检查清单:
- [ ] Token是否有效
- [ ] scheduleId是否存在

- [ ] 文件类型是否在白名单中
- [ ] 是否是重复上传（同一文件）
- [ ] 查看后端日志中的具体错误信息
```

### 如果下载失败

```
检查清单:
- [ ] 文件是否物理存在
