## 📋 使用指南

### 如何查看 API 文档
1. 启动 conference-auth 服务
2. 访问 http://localhost:8081/swagger-ui.html
3. 在 "租户管理" 标签下查看所有 API
4. 可以直接在 Swagger UI 中测试 API

### 如何使用管理后台
1. 访问 http://localhost:8080/pages/customer-management.html
2. 在浏览器控制台设置 Token: `localStorage.setItem('auth_token', 'YOUR_TOKEN')`
3. 刷新页面，即可看到租户列表
4. 可以进行所有 CRUD 操作

### 如何使用 APP 端
1. 访问 http://localhost:3000/app/staff/tenant-manage.html（或直接打开 HTML 文件）
2. 在浏览器控制台设置 Token 和租户 ID
3. 刷新页面，即可看到租户信息和协管员列表

### 如何调试前端
1. 打开浏览器开发者工具 (F12)
2. 查看 Console 标签的日志输出
3. 查看 Network 标签的 API 请求
4. 使用 Sources 标签设置断点调试

### 如何查看数据库
1. 使用 MySQL 客户端连接到数据库
2. 使用数据库: `USE conference_db;`
3. 查看表: `SHOW TABLES;`
4. 查询数据: `SELECT * FROM sys_tenant;`
