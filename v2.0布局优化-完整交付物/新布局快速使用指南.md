## 📍 快捷操作位置

### 1. 插入过道（新方式）

#### 步骤 1️⃣
找到你想要插入过道的排号（例如"3排"）

#### 步骤 2️⃣
**右键点击**排号标签

#### 步骤 3️⃣
菜单弹出，显示两个选项：
- **⬆ 在上方插入过道** - 在当前排上方插入过道
- **⬇ 在下方插入过道** - 在当前排下方插入过道

#### 步骤 4️⃣
点击选项，菜单自动关闭，过道插入完成

## 💡 使用技巧

### 技巧 1：快速多个过道插入
```
如果需要在第2和第5行之间插入过道：
1. 在第2排下方插入过道
2. 在新过道下方继续操作
3. 继续在相应位置插入其他过道
```

### 技巧 2：响应式屏幕
```
不同屏幕宽度下：
- 座位区宽度自动调整
- 底部信息面板列数自动调整
- 座位大小自动缩放（宽屏下）
```

### 技巧 3：查看历史
```
如果不小心插入了过道，可以：
1. 点击底部的"变更历史"卡片
2. 查看最近的操作
3. 点击"撤销"按钮恢复
```

## 🖱️ 鼠标交互

### 左键点击
- **座位方块**：选择/分配座位
- **座位号**：可用座位分配，已占座位查看信息

### 右键点击
- **排号标签**：显示插入过道菜单
- **其他位置**：菜单消失

### 鼠标悬停
- **排号标签**：提示"右键菜单：在N排的上下方插入过道"
- **菜单项**：背景变亮，显示可点击状态

## 🔧 故障排除

### 问题：菜单未出现
**解决方案**：
1. 确保右键点击在排号标签上
2. 检查浏览器是否禁用右键菜单
3. 刷新页面重试

### 问题：座位显示异常
**解决方案**：
1. 按 `Ctrl+F5` 硬刷新清除缓存
2. 检查浏览器开发者工具（F12）是否有红色错误
3. 尝试用其他浏览器

### 问题：底部面板显示不全
**解决方案**：
1. 向下滚动页面查看更多内容
2. 减小浏览器缩放比例（Ctrl + "-"）
3. 使用更大的屏幕

## 🎓 完整工作流程示例

### 场景：为500人会议布置座位

```
1. 创建座位区
   └─ 点击"添加第一个区域"
   └─ 配置行数、列数、布局类型

2. 导入参会人员
   └─ 点击"从报名导入"
   └─ 或点击"Excel导入"上传名单

3. 配置过道（如需要）
   └─ 右键点击排号
   └─ 选择"在下方插入过道"
   └─ 根据需要重复操作

4. 执行自动排座
   └─ 选择排座算法
   └─ 点击"执行排座"
   └─ 系统自动分配座位

5. 调整和优化
   └─ 点击座位进行手动调整
   └─ 使用"撤销"恢复错误操作

6. 发送通知
   └─ 点击"发送座位通知"
   └─ 参会者收到APP推送

7. 生成输出
   └─ 导出座位表
   └─ 生成LED导示图
   └─ 生成查座二维码
```
