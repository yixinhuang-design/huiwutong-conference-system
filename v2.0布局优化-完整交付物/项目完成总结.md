3. 交互方式不友好（隐藏的小按钮）

## 🔧 技术亮点

### 1. 响应式网格布局
```css
.bottom-panels {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    /* 自动适应屏幕宽度，根据空间创建 N 列 */
}
```

### 2. CSS 伪元素虚线
```css
.aisle-row::before {
    background: repeating-linear-gradient(
        90deg,
        #ff9800 0px, #ff9800 8px,    /* 8px 实线 */
        transparent 8px, transparent 16px /* 8px 透明 */
    );
    /* 用一个虚线代替多个座位，减少 DOM */
}
```

### 3. Vue 响应式菜单管理
```javascript
const contextMenu = reactive({
    visible: false,
    x: 0, y: 0,
    type: null,
    area: null,
    rowIndex: null
    // 完整的菜单状态管理
});
```

### 4. 事件驱动架构
```javascript
// 右键点击 → 显示菜单
@contextmenu.prevent="showRowContextMenu($event, area, rowIndex)"

// 菜单点击 → 执行操作 → 关闭菜单
@click="insertRowAisle(...); hideContextMenu()"
```
