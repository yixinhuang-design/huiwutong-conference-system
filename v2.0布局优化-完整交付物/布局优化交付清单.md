# 智能排座系统 - 布局优化交付清单

## ✅ 实现验证清单

### 1. 页面布局改进

- [x] **主容器改为单列布局**
  - 位置：CSS `#main-content` 或 `.main-content`
  - 改动：`grid-template-columns: 2fr 1fr` → `grid-template-columns: 1fr`
  - 效果：座位区宽度从 66% 扩展到 100%

- [x] **座位区占据全宽**
  - 位置：`.seating-chart-section`
  - 属性：`grid-column: 1; grid-row: 1;`
  - 效果：座位区现在占据整个上半部分

- [x] **底部面板响应式布局**
  - 位置：`.side-panel` 或 `.bottom-panels`
  - 属性：`grid-template-columns: repeat(auto-fit, minmax(300px, 1fr))`
  - 效果：控制面板在底部自适应多列显示

### 2. 过道显示优化

- [x] **过道行虚线显示**
  - 位置：`.aisle-row::before` 伪元素
  - 实现：`repeating-linear-gradient` 虚线图案
  - 效果：虚线分隔符替代灰色座位网格

- [x] **隐藏过道行座位方块**
  - 位置：`.aisle-row .seat`
  - 属性：`display: none`
  - 效果：过道行不显示座位格子

- [x] **过道行标签样式**
  - 位置：`.aisle-label`
  - 样式：背景色 + 虚线边框 + "过道" 文本
  - 效果：清晰标识过道区域

- [x] **HTML 模板中添加过道标签**
  - 位置：`<div v-if="row.isAisle" class="aisle-row">`
  - 内容：`<span class="aisle-label">--- 过道 ---</span>`
  - 效果：在 UI 中显示过道标签

### 3. 交互方式升级

- [x] **右键菜单数据结构**
  - 位置：setup() 函数中 `contextMenu` 响应式对象
  - 字段：visible, x, y, type, area, rowIndex, colIndex
  - 效果：完整的菜单状态管理

- [x] **右键菜单 HTML 容器**
  - 位置：主模板中 `<div v-if="contextMenu.visible" class="context-menu">`
  - 内容：两个菜单模板（行菜单和列菜单）
  - 效果：菜单在页面中正确渲染

- [x] **右键菜单 CSS 样式**
  - 位置：`.context-menu` 和 `.context-menu-item`
  - 样式：固定定位 + 阴影 + 悬停效果
  - 效果：菜单外观美观专业

- [x] **右键菜单 JavaScript 方法**
  - `showRowContextMenu()` - 显示行菜单
  - `showSeatContextMenu()` - 显示座位菜单（保留扩展）
  - `hideContextMenu()` - 隐藏菜单
  - 效果：完整的菜单交互逻辑

- [x] **移除旧的按钮控制**
  - 移除：`.row-control` 行操作按钮
  - 移除：`.seating-columns-control` 列操作按钮
  - 移除：`.column-insert-marker` 列标记
  - 效果：界面清爽，减少视觉混乱

- [x] **HTML 模板中添加右键触发**
  - 位置：过道行和座位行上方
  - 属性：`@contextmenu.prevent="showRowContextMenu($event, area, rowIndex)"`
  - 效果：右键点击可触发菜单

### 4. 代码集成

- [x] **在返回对象中导出新方法**
  - `contextMenu` - 菜单状态
  - `showRowContextMenu` - 行菜单处理
  - `showSeatContextMenu` - 座位菜单处理
  - `hideContextMenu` - 隐藏菜单
  - 效果：模板可以访问所有方法

- [x] **保持现有功能不变**
  - `insertRowAisle()` 接口不变
  - `insertColumnAisle()` 接口不变
  - 菜单点击后自动调用这些方法
  - 效果：向后兼容，功能完整

### 5. 视觉外观

- [x] **过道标签背景** → 使用 `--warning-bg` (淡橙色 #fff3e0)

- [x] **菜单阴影** → 清晰的阴影效果，Z-index 10000
- [x] **菜单悬停** → 背景变亮，文字变为主色
