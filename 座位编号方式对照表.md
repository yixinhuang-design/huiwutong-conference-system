# 座位编号方式完整对照表

## 一、五种编号方式汇总

### 📊 对于 3行×7列 的座位区，五种方式的编号效果对比

#### 方式1：中间左右（Center-Outward）
```
从中心向两侧扩散，体现对称和尊重

行号\列号   1    2    3    4    5    6    7
  1        6    4    2    1    3    5    7
  2       13   11    9    8   10   12   14
  3       20   18   16   15   17   19   21

特点：
- 核心逻辑：中心位置编号最小，向两侧递增
- 中心列：ceil(cols/2) = 4
- 左侧：距离中心越远编号越大
- 右侧：距离中心越远编号越大
- 应用：VIP座区、贵宾位

数学公式：
centerCol = ceil(cols / 2)
distance = |col - centerCol|
if col == centerCol:
    num = (row-1) × cols + centerCol
else if col < centerCol:
    num = (row-1) × cols + (centerCol - distance×2 + 1)
else:
    num = (row-1) × cols + (centerCol + distance×2 - 1)
```

#### 方式2：由左至右（Left-to-Right）
```
最标准的编号方式，从左到右顺序，逐行递增

行号\列号   1    2    3    4    5    6    7
  1        1    2    3    4    5    6    7
  2        8    9   10   11   12   13   14
  3       15   16   17   18   19   20   21

特点：
- 核心逻辑：行序*列数 + 列号
- 最简单直观
- 应用：普通座位区、标准票证

数学公式：
num = (row - 1) × cols + col
```

#### 方式3：左右右左（Alternating-LTR-Start）
```
蛇形编号：奇数行从左到右，偶数行从右到左
便于座位中间通道设置

行号\列号   1    2    3    4    5    6    7
  1        1    2    3    4    5    6    7
  2       14   13   12   11   10    9    8
  3       15   16   17   18   19   20   21

特点：
- 核心逻辑：行交替方向
- 奇数行(1,3,5...)：从左到右
- 偶数行(2,4,6...)：从右到左
- 最小化行间过道
- 应用：剧院式座位、通道居中

数学公式：
baseNum = (row - 1) × cols
if row % 2 == 1:
    num = baseNum + col
else:
    num = baseNum + (cols - col + 1)
```

#### 方式4：由右至左（Right-to-Left）
```
从右向左的标准编号，逐行递增

行号\列号   1    2    3    4    5    6    7
  1        7    6    5    4    3    2    1
  2       14   13   12   11   10    9    8
  3       21   20   19   18   17   16   15

特点：
- 核心逻辑：行内从右到左编号
- 与左至右相反
- 应用：右侧主入口、特殊场景

数学公式：
num = (row - 1) × cols + (cols - col + 1)
```

#### 方式5：右左左右（Alternating-RTL-Start）
```
蛇形编号：奇数行从右到左，偶数行从左到右
反向蛇形布局

行号\列号   1    2    3    4    5    6    7
  1        7    6    5    4    3    2    1
  2        8    9   10   11   12   13   14
  3       21   20   19   18   17   16   15

特点：
- 核心逻辑：行交替方向（反向）
- 奇数行(1,3,5...)：从右到左
- 偶数行(2,4,6...)：从左到右
- 最小化行间过道
- 应用：左侧主入口、反向剧院式

数学公式：
baseNum = (row - 1) × cols
if row % 2 == 1:
    num = baseNum + (cols - col + 1)
else:
    num = baseNum + col
```

## 三、JavaScript 实现对照

### 统一接口

```javascript
class SeatNumberGenerator {
  constructor(rows, cols, mode) {
    this.rows = rows;
    this.cols = cols;
    this.mode = mode;
  }

  // 根据行列号生成座位号
  generate(row, col) {
    switch(this.mode) {
      case 'center_outward':
        return this._centerOutward(row, col);
      case 'left_to_right':
        return this._leftToRight(row, col);
      case 'alternating_ltr_start':
        return this._alternatingLTRStart(row, col);
      case 'right_to_left':
        return this._rightToLeft(row, col);
      case 'alternating_rtl_start':
        return this._alternatingRTLStart(row, col);
    }
  }

  // ============ 具体实现 ============

  _centerOutward(row, col) {
    const centerCol = Math.ceil(this.cols / 2);
    const distance = Math.abs(col - centerCol);
    const baseNum = (row - 1) * this.cols;

    if (col === centerCol) {
      return baseNum + centerCol;
    } else if (col < centerCol) {
      // 左侧：递减递增（6,4,2,1,...）
      return baseNum + (centerCol - distance * 2 + 1);
    } else {
      // 右侧：递增（...,1,3,5,7）
      return baseNum + (centerCol + distance * 2 - 1);
    }
  }

  _leftToRight(row, col) {
    return (row - 1) * this.cols + col;
  }

  _alternatingLTRStart(row, col) {
    const baseNum = (row - 1) * this.cols;
    return row % 2 === 1
      ? baseNum + col
      : baseNum + (this.cols - col + 1);
  }

  _rightToLeft(row, col) {
    return (row - 1) * this.cols + (this.cols - col + 1);
  }

  _alternatingRTLStart(row, col) {
    const baseNum = (row - 1) * this.cols;
    return row % 2 === 1
      ? baseNum + (this.cols - col + 1)
      : baseNum + col;
  }

  // 批量生成全部座位
  generateAll() {
    const seats = [];
    for (let r = 1; r <= this.rows; r++) {
      for (let c = 1; c <= this.cols; c++) {
        seats.push({
          row: r,
          col: c,
          number: this.generate(r, c),
          displayText: `${r}排${c}列`,
          displayNumber: `第${this.generate(r, c)}座`
        });
      }
    }
    return seats;
  }
}
```

### 使用示例

```javascript
// 创建 3行×7列 的座位区，使用"左右右左"编号
const gen = new SeatNumberGenerator(3, 7, 'alternating_ltr_start');

// 获取单个座位编号
console.log(gen.generate(1, 4)); // 输出：4
console.log(gen.generate(2, 4)); // 输出：11

// 生成全部座位
const allSeats = gen.generateAll();
console.log(allSeats[0]); // { row: 1, col: 1, number: 1, ... }

// 创建 HTML 表格预览
function renderPreview(seats, rows, cols) {
  let html = '<table class="seat-preview"><tbody>';
  for (let r = 1; r <= rows; r++) {
    html += '<tr>';
    for (let c = 1; c <= cols; c++) {
      const seat = seats.find(s => s.row === r && s.col === c);
      html += `<td>${seat.number}</td>`;
    }
    html += '</tr>';
  }
  html += '</tbody></table>';
  return html;
}
```

## 四、实际应用场景选择指南

### 🎯 选择建议

1. **中间左右** ← 选这个
   - 条件：VIP贵宾区、领导座位、对称布局
   - 特点：突出主位、体现尊重、对称美感
   - 例子：主席台、演讲嘉宾区

2. **由左至右** ← 推荐，最常用
   - 条件：普通观众区、展览会、标准票券
   - 特点：最简单直观、易于售票系统
   - 例子：电影院、报告厅

3. **左右右左** ← 推荐，剧院场景
   - 条件：剧院式座位、需要居中通道
   - 特点：蛇形布局、最小化通道占用
   - 例子：会议厅、表演厅

4. **由右至左** ← 特殊场景
   - 条件：右侧为主入口、建筑特殊
   - 特点：反向编号、适应建筑流线
   - 例子：右侧入场的场所

5. **右左左右** ← 特殊场景
   - 条件：左侧为主入口、反向蛇形布局
   - 特点：蛇形反向、左侧通道
   - 例子：左侧入场的场所

## 五、转换算法

### 同一座位在不同编号方式下的转换

```javascript
class SeatNumberConverter {
  constructor(rows, cols) {
    this.rows = rows;
    this.cols = cols;
    this.generators = {
      center_outward: new SeatNumberGenerator(rows, cols, 'center_outward'),
      left_to_right: new SeatNumberGenerator(rows, cols, 'left_to_right'),
      alternating_ltr_start: new SeatNumberGenerator(rows, cols, 'alternating_ltr_start'),
      right_to_left: new SeatNumberGenerator(rows, cols, 'right_to_left'),
      alternating_rtl_start: new SeatNumberGenerator(rows, cols, 'alternating_rtl_start')
    };
  }

  // 根据某种方式的座位号，查找其他方式的号码
  convert(fromMode, toMode, seatNumber) {
    // 1. 找到座位的行列位置
    let seat = null;
    for (let r = 1; r <= this.rows && !seat; r++) {
      for (let c = 1; c <= this.cols && !seat; c++) {
        if (this.generators[fromMode].generate(r, c) === seatNumber) {
          seat = { row: r, col: c };
        }
      }
    }

    if (!seat) return null;

    // 2. 用目标方式生成号码
    return this.generators[toMode].generate(seat.row, seat.col);
  }

  // 批量转换
  convertAll(fromMode, toMode) {
    const mapping = {};
    for (let r = 1; r <= this.rows; r++) {
      for (let c = 1; c <= this.cols; c++) {
        const fromNum = this.generators[fromMode].generate(r, c);
        const toNum = this.generators[toMode].generate(r, c);
        mapping[fromNum] = toNum;
      }
    }
    return mapping;
  }
}

// 使用示例
const converter = new SeatNumberConverter(10, 15);

// 某人在"由左至右"中的座位号是42，在"左右右左"中是多少？
const newNumber = converter.convert('left_to_right', 'alternating_ltr_start', 42);
console.log(`原座位号42转换后为: ${newNumber}`);

// 获取两种方式之间的完整映射
const mapping = converter.convertAll('left_to_right', 'alternating_ltr_start');
```

## 六、性能对比

| 操作 | 时间复杂度 | 说明 |
|------|---------|------|
| 单个座位号生成 | O(1) | 直接数学计算 |
| 批量生成（rows×cols座位） | O(r×c) | 线性遍历 |
| 座位号转换（单个） | O(r×c) | 需要搜索行列位置 |
| 座位号转换（批量） | O(r×c) | 遍历全部座位 |
| 编号方式切换 | O(r×c) | 需要重新编号全部 |
