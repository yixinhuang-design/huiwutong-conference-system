## 🔧 技术栈对比

### 学员端
```
框架：UniApp (Vue 3)
样式：内置组件 + CSS
状态：Reactive refs
通信：Fetch API
存储：localStorage
兼容性：iOS/Android/Web
```

### 管理端
```
框架：UniApp (Vue 3)
样式：内置组件 + SCSS
表单：HTML input + picker
通信：uni.request / Fetch API
存储：localStorage
兼容性：iOS/Android/Web
```

### Web后台
```
框架：Vue 3 (CDN)
样式：HTML + CSS/SCSS
表单：HTML form + modal
通信：Fetch API
存储：localStorage
兼容性：现代浏览器
```

### 后端
```
框架：Spring Boot 3.2.3
ORM：MyBatis Plus 3.5.5
数据库：MySQL 9.6
服务发现：支持多租户
认证：Bearer Token
```
