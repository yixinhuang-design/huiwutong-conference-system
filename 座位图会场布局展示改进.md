# 座位图可视化展示改进说明

**改进日期**: 2026-01-24
**页面**: seating-mgr.html (智能排座系统)

---

## 🎯 改进目标

将简单的网格排列改为**真实的会场平面图**展示，参考市面上最佳排座系统的可视化方式。

---

## 📐 新的座位图展示方式

### 设计理念

参考 Azaxi、开会排座、Cvent 等主流排座系统的最佳实践：
- ✅ **2D平面图** - 模拟真实会场空间布局
- ✅ **空间关系** - 座位区在会场中的相对位置
- ✅ **入场标识** - 入口、紧急出口等标识
- ✅ **主席台** - 大屏幕、讲台的直观展示
- ✅ **动态布局** - 支持不同会议类型

## 🎨 座位区展示方式

### 布局策略

**第1个座位区** (通常是VIP区):
- 位置：前排居中
- 对齐：`top: 20px, left: 50%, translateX(-50%)`
- 特点：离主席台最近

**第2个座位区**:
- 位置：右侧
- 对齐：`top: 20px, right: 20px`
- 特点：会场右侧区域

**第3个座位区**:
- 位置：左侧
- 对齐：`top: 20px, left: 20px`
- 特点：会场左侧区域

**超过3个座位区**:
- 使用网格布局
- 每行最多3个区
- 自动换行

### VIP区域特殊样式

**边框**: 粉色 (3px solid #f093fb)
**背景**: 淡粉色渐变
**徽章**: "VIP" 标签（粉色渐变）
**空座位**: 淡粉色背景

**普通区域**:
**边框**: 白色虚线 (2px dashed #dee2e6)
**背景**: 白色
**空座位**: 半透明白色

## 📏 座位区定位算法

### 方法: `getAreaPositionStyle(area, index)`

**前3个座位区** (预设位置):
```javascript
// VIP区 (第1个)
{
    top: '20px',
    left: '50%',
    transform: 'translateX(-50%)'
}

// 右侧区 (第2个)
{
    top: '20px',
    right: '20px'
}

// 左侧区 (第3个)
{
    top: '20px',
    left: '20px'
}
```

**超过3个座位区** (网格布局):
```javascript
const cols = 3;
const row = Math.floor(index / cols);
const col = index % cols;

return {
    top: `${20 + row * 220}px`,
    left: `${50 + (col - 1) * 33}%`,
    transform: col === 1 ? 'translateX(-50%)' : 'none'
};
```

## 📝 代码变更总结

### HTML结构变化

**之前**:
```html
<div class="seating-chart">
  <div class="stage">主席台</div>
  <div class="seating-area">
    <div class="area-section" v-for="area in seatingAreas">
      <!-- 座位 -->
    </div>
  </div>
</div>
```

**现在**:
```html
<div class="seating-chart">
  <div class="venue-container">
    <div class="venue-background">
      <!-- 会场框架、入口、出口 -->
    </div>
    <div class="stage-area">
      <!-- 主席台 -->
    </div>
    <div class="seating-chart-viewport">
      <div class="seating-areas-container">
        <div class="area-section"
             v-for="area in seatingAreas"
             :style="getAreaPositionStyle(area, index)">
          <!-- 座位 -->
        </div>
      </div>
    </div>
  </div>
</div>
```

### 新增CSS类

- `.venue-container` - 会场容器
- `.venue-background` - 会场背景
- `.venue-walls` - 会场墙壁
- `.venue-entrance` - 入口标识
- `.emergency-exit` - 紧急出口
- `.stage-area` - 主席台区域
- `.stage-platform` - 主席台平台
- `.seating-chart-viewport` - 座位视图容器
- `.seating-areas-container` - 座位区容器
- `.area-label-banner` - 区域标签横幅
- `.area-stats-overlay` - 区域统计浮层

### 新增方法

- `getAreaPositionStyle(area, index)` - 计算座位区位置
- `getRowStyle(row, area)` - 获取行样式（处理过道边距）
