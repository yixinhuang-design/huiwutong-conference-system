## ❌ 完全缺失的流程页面

### 流程01：多租户管理流程

**流程概述**：系统管理员创建和管理租户，租户管理员登录并管理自己的会议。

**缺失页面**：
1. **租户管理页面** `staff/tenant-manage.html`
   - **功能需求**：
     - 创建新租户（填写租户基本信息）
     - 编辑租户信息
     - 删除租户
     - 锁定/解锁租户
     - 查看租户列表
   - **字段清单**：
     - 租户基本信息：租户名称、租户编码、联系人、联系电话、邮箱、状态
     - 租户参数：租户类型、权限范围、使用期限、最大用户数、最大会议数、存储配额
     - 租户管理员：账号、初始密码、管理员姓名、管理员手机

**流程步骤**：
```
1. 系统管理员登录系统
2. 点击进入租户管理页面
3. 创建新租户/编辑/删除/锁定/解锁
   ├─ 填写租户基本信息
   ├─ 配置租户参数
   └─ 设置租户管理员
4. 租户管理员首次登录
   ├─ 选择租户
   ├─ 输入账号密码
   └─ 获取JWT Token
```

### 流程18：预警提示流程

**流程概述**：配置预警规则，实时监控关键指标，触发预警后发送通知。

**缺失页面**：
1. **预警配置页面** `staff/alert-config.html`
   - **功能需求**：
     - 选择会议/培训
     - 选择预警指标
     - 设置预警阈值
     - 设置预警级别（高/中/低）
     - 设置通知方式（短信、系统通知）
     - 查看预警规则列表
     - 编辑/删除预警规则

2. **预警处理页面** `staff/alert-handle.html`
   - **功能需求**：
     - 查看预警列表（待处理、处理中、已处理、已忽略）
     - 查看预警详情（指标、阈值、实际值、级别、时间、原因）
     - 处理预警事件
     - 标记预警状态
     - 记录处理记录
     - 查看预警统计（总数、各级别数、已处理数、未处理数）

**流程步骤**：
```
1. 配置预警规则
   ├─ 选择会议/培训
   ├─ 选择预警指标（报名率、签到率、任务完成率等）
   ├─ 设置预警阈值
   ├─ 设置预警级别
   └─ 设置通知方式

2. 实时监控数据
   ├─ 监控关键指标
   ├─ 检测数据变化
   └─ 计算指标值

3. 触发预警规则
   ├─ 阈值监控
   ├─ 异常检测
   ├─ 规则匹配
   └─ 生成预警事件

4. 预警通知
   ├─ 发送预警通知
   ├─ 通知渠道
   └─ 记录预警日志

5. 预警处理
   ├─ 查看预警详情
   ├─ 处理预警事件
   ├─ 标记预警状态
   └─ 记录处理记录

6. 预警统计与分析
   ├─ 查看预警统计
   ├─ 分析预警趋势
   ├─ 优化预警规则
   └─ 预警效果评估
```

### 流程05：名册生成流程

**当前覆盖情况**：
- ✅ 报名管理页面 `staff/registration-manage.html` - 已有报名列表、审核功能
- ✅ 分组管理页面 `staff/grouping-manage.html` - 已有分组功能
- ✅ 学员指南页面 `learner/guide.html` - 已有培训须知展示

**缺失功能**：
1. **名册生成页面** `staff/handbook-generate.html`
   - **功能需求**：
     - 进行名册分组（可选）
     - 进行名册排序（按单位/拖动/序号）
     - 配置名册格式
       - 选择名册模板（标准、详细、自定义）
       - 上传自定义封面图
       - 编辑培训须知（富文本编辑）
       - 设置会议日程
       - 设置学员名册
       - 编辑自定义册尾内容
     - 预览名册内容
       - 查看封面预览
       - 查看学员手册预览
       - 查看名册列表预览
     - 生成名册（生成PDF格式）
     - 保存名册模板
     - 下载或分享名册
       - 下载到本地
       - 分享链接（有效期7天）

**流程步骤**：
```
1. 管理员进入名册生成页面

2. 进行名册分组（可选）
   ├─ 设置分几个组、各组名称
   ├─ 指定各组组长、各组工作人员
   ├─ 设置各组成员：拖动分组/设置序号分组
   └─ 自动生成各组人员群

3. 进行名册排序
   ├─ 选择全部已审核通过人员
   ├─ 按单位排序/拖动排序/设置序号排序
   └─ 如分组，对组内人员排序

4. 配置名册格式
   ├─ 选择名册模板
   ├─ 上传自定义封面图
   ├─ 编辑培训须知
   ├─ 设置会议日程
   ├─ 设置学员名册
   └─ 编辑自定义册尾内容

5. 预览名册内容
   ├─ 查看封面预览
   ├─ 查看学员手册预览
   └─ 查看名册列表预览

6. 生成名册
   ├─ 生成PDF格式
   └─ 保存名册模板

7. 下载或分享名册
   ├─ 下载到本地
   └─ 分享链接
```

## 📝 技术实现要点

### 流程01 多租户管理
- **前端组件**：租户列表、租户表单、租户详情
- **后端接口**：租户CRUD、租户状态管理、租户管理员设置
- **权限控制**：仅系统管理员可访问

### 流程05 名册生成
- **前端组件**：富文本编辑器、PDF预览器、拖拽排序
- **后端接口**：名册数据组装、PDF生成、文件分享
- **第三方库**：PDF生成库、富文本编辑器

### 流程17 数据分析
- **前端组件**：数据表格、图表组件、导出组件
- **后端接口**：历史数据查询、数据聚合分析、数据导出
- **性能优化**：大数据量分页、异步导出

### 流程18 预警提示
- **前端组件**：预警规则表单、预警列表、预警详情弹窗
- **后端接口**：预警规则CRUD、实时监控、预警通知
- **实时通信**：WebSocket推送预警通知
