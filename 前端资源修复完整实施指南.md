## 🔧 集成步骤

### 步骤 1: 在 HTML 中引入新脚本

在 `<head>` 标签中添加:

```html
<!-- Extension 处理脚本 (必须) -->
<script src="../js/extension-handler.js"></script>

<!-- Vue 模态框组件 (可选 - 如需使用模态框) -->
<script type="module">
  import Modal from '../js/components/Modal.js';
  
  // 在应用中注册组件
  app.component('Modal', Modal);
</script>

<!-- 模态框样式 (如使用 Vue 模态框) -->
<link rel="stylesheet" href="../css/modal-styles.css">
```

### 步骤 2: 在 HTML 中使用模态框

```html
<!-- Vue 应用中使用 Modal 组件 -->
<Modal 
  :visible="showConfirmModal"
  title="确认删除"
  confirmText="删除"
  cancelText="取消"
  @confirm="deleteItem"
  @cancel="showConfirmModal = false"
>
  <p>确定要删除这条记录吗？此操作无法撤销。</p>
</Modal>
```

### 步骤 3: 在 Vue 组件中处理事件

```javascript
export default {
  data() {
    return {
      showConfirmModal: false,
      selectedItem: null
    }
  },
  methods: {
    openDeleteDialog(item) {
      this.selectedItem = item;
      this.showConfirmModal = true;
    },
    deleteItem() {
      // 执行删除操作
      console.log('删除项目:', this.selectedItem);
      this.showConfirmModal = false;
      
      // 调用 API
      fetch(`/api/items/${this.selectedItem.id}`, {
        method: 'DELETE',
        headers: {
          'Authorization': 'Bearer ' + this.getToken()
        }
      }).then(response => {
        if (response.ok) {
          // 刷新列表
          this.loadItems();
        }
      });
    }
  }
}
```

## 🚨 常见问题排查

### 问题 1: 仍然看到 404 错误

**症状**: 
```
GET http://localhost:8080/pages/css/modules/laydate/default/laydate.css 404
```

**解决方案**:
1. 确认文件已创建: `Test-Path` 检查
2. 清除浏览器缓存: `Ctrl+Shift+Delete`
3. 强制刷新页面: `Ctrl+F5`
4. 重启 Web 服务器

### 问题 2: Extension Handler 未加载

**症状**:
```
ReferenceError: window.ExtensionHandler is not defined
```

**解决方案**:
1. 检查脚本标签位置 (应在 `<head>` 中)
2. 检查脚本路径是否正确
3. 打开控制台，查看是否有加载错误
4. 尝试在控制台手动运行: `location.reload()`

### 问题 3: 模态框样式不正确

**症状**:
```
模态框显示但样式混乱、位置不对或按钮不可见
```

**解决方案**:
1. 确认 CSS 文件已加载: F12 → Network → modal-styles.css
2. 检查 CSS 加载顺序 (modal-styles.css 应在其他样式之后)
3. 清除浏览器缓存
4. 检查是否有 CSS 冲突 (使用浏览器检查器)

### 问题 4: 模态框组件未注册

**症状**:
```
[Vue warn]: Failed to resolve component: Modal
```

**解决方案**:
1. 确认 Modal.js 已正确导入
2. 检查 app.component('Modal', Modal) 是否执行
3. 确认 Vue 应用实例可访问
4. 检查浏览器控制台获取更多错误信息
