## 🔗 页面跳转

### 报名管理 → 智能排座

**位置**: registration-mgr.html

```javascript
goToSeating() {
    const conferenceId = new URLSearchParams(window.location.search).get('conferenceId');
    const url = conferenceId
        ? `seating-mgr.html?conferenceId=${conferenceId}`
        : 'seating-mgr.html';

    // 保存报名数据到localStorage
    this.saveRegistrationData();

    // 跳转到排座系统
    window.location.href = url;
}

saveRegistrationData() {
    const data = {
        registrations: this.registrations,
        timestamp: Date.now(),
        conference: this.conference
    };
    localStorage.setItem('registrationData', JSON.stringify(data));
}
```
