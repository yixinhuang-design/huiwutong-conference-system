## 📝 使用说明

### 快速开始

1. **编译项目**
   ```bash
   mvn clean compile
   ```

2. **启动应用**
   ```bash
   mvn spring-boot:run
   ```

3. **测试API**
   ```bash
   curl -X POST "http://localhost:8080/api/seating/seats/swap/1/2" \
     -H "Authorization: Bearer {token}" \
     -H "Content-Type: application/json"
   ```

### API端点

```
POST /api/seating/seats/swap/{seatId1}/{seatId2}
```

**成功响应** (HTTP 200):
```json
{
  "code": 200,
  "message": "座位交换成功",
  "data": {
    "success": true,
    "seatId1": 1,
    "seatId2": 2,

    "afterSeat1Assignee": "李四",
    "afterSeat2Assignee": "张三",
```
