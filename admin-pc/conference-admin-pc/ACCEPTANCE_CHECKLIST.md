### 第三部分: Token管理

#### 3.1 Token存储
- [ ] 打开浏览器开发者工具 (F12)
- [ ] 切换到Application/Storage标签
- [ ] 找到localStorage选项卡
- [ ] 成功登录后，能看到以下键值:
  - [ ] `authToken` - 包含有效的访问令牌
  - [ ] `refreshToken` - 包含刷新令牌
  - [ ] `tokenType` - 值为 "Bearer"
  - [ ] `tokenExpireTime` - 时间戳
  - [ ] `expiresIn` - 秒数
  - [ ] `userId` - 用户ID
  - [ ] `username` - 用户名为 "admin"
  - [ ] `userType` - 用户类型为 "admin"
  - [ ] `tenantId` - 租户ID
  - [ ] `tenantCode` - 租户编码为 "default"

#### 3.2 Token格式验证
- [ ] 复制authToken值
- [ ] 访问 https://jwt.io 网站
- [ ] 粘贴token到"Encoded"区域
- [ ] 验证token能正确解码
- [ ] 检查exp字段表示过期时间
- [ ] 检查其他字段包含用户信息

#### 3.3 Token生命周期
- [ ] 成功登录3秒后，检查localStorage
- [ ] Token值应该保持不变
- [ ] 刷新页面，Token仍然保存
- [ ] 关闭浏览器标签页，再打开
- [ ] Token仍然保存（同浏览器）
