## 六、常见问题排查

### Q1: 页面加载时出现模块导入错误
```
错误信息如：
Uncaught SyntaxError: Cannot use import statement outside a module
或
Failed to load module from ...

解决：
1. 确保seating-mgr.html的<script>标签有 type="module"
2. 确保三个模块文件在正确的位置：
   - modules/seating-numbering-system.js
   - modules/seating-layout-grid.js
   - modules/seating-schedule-manager.js
3. 检查文件路径是否相对正确（当前是../modules/...）
```

### Q2: scheduleManager未定义
```
错误信息：
ReferenceError: useScheduleManager is not defined

解决：
1. 检查import语句是否正确
2. 检查module export是否正确导出useScheduleManager
3. 检查是否使用了type="module"
```

### Q3: Grid布局函数报错
```
错误如：
Cannot read property 'autoLayoutAreas' of undefined

解决：
1. 检查import seatingLayoutGrid是否成功
2. 确保seating-layout-grid.js的export语句正确
3. 在console中验证：console.log(seatingLayoutGrid)
```

### Q4: 日程数据未加载
```
现象：
schedulesList为空，currentSchedule为null

解决：
1. 等待一段时间让异步初始化完成
2. 在console中手动调用：vm.loadSchedules()
3. 检查localStorage是否被清空
```

// 检查所有导入
