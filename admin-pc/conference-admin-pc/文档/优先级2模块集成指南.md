## 🐛 故障排除

### 错误1: "模块找不到"
```
Uncaught SyntaxError: import not found
```
**解决方案**:
- 检查文件路径是否正确
- 确保模块文件确实存在
- 检查文件名是否拼写正确

### 错误2: "XX is not defined"
```
Uncaught ReferenceError: importMethods is not defined
```
**解决方案**:
- 确保模块已初始化
- 检查初始化时是否提供了所有依赖
- 检查返回对象中是否正确导出了方法

### 错误3: "找不到依赖"
```
Cannot read property 'value' of undefined
```
**解决方案**:
- 检查传给 init() 的依赖对象
- 确保所有必需的依赖都已提供
- 检查依赖的类型是否正确（ref vs reactive）

### 错误4: "导入循环"
```
Circular dependency detected
```
**解决方案**:
- 不要在模块中相互导入
- 如果需要交互，通过返回对象和参数传递
- 使用依赖注入模式
