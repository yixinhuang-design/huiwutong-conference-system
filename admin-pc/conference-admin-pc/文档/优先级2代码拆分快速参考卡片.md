## 🔌 集成方式

### 方式1: 在setup中初始化（推荐）

```javascript
// 导入模块
import { SeatingImportAttendeesModule } from '../modules/seating-import-attendees.js';

// 在setup中使用
const importMethods = SeatingImportAttendeesModule.init(vm, {
    attendees,
    currentScheduleId,
    loading,
    saveData,
    updateStatistics,
    isVipByPosition
});

// 导出到模板
return {
    ...importMethods,
    // 其他返回值
};
```

### 方式2: 直接使用（如果不需要响应式）

```javascript
// 导入模块
import { SeatingImportAttendeesModule } from '../modules/seating-import-attendees.js';

// 调用static方法
const importAttendees = async () => {
    const methods = SeatingImportAttendeesModule.init(vm, deps);
    await methods.handleExcelImport(event);
};
```

## ⚠️ 注意事项

### 数据依赖
- 所有模块都需要 `saveData()` 来持久化数据
- 修改后必须调用 `updateStatistics()` 更新统计信息
- 使用 `recordSeatChange()` 记录操作历史用于撤销

### 兼容性
- 需要 Vue 3.3+
- 依赖 InteractionUtils 全局工具类
- 需要 localStorage 支持
- 需要 XLSX 库用于Excel解析

### 错误处理
- 所有异步操作都应该用 try-catch 包装
- 显示用户友好的错误提示
- 在 finally 中清理加载状态
