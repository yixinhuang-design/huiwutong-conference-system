# 座位布局过道直接插入功能使用指南

## 📋 功能说明

在座位管理系统的主座位布局显示区中，用户现在可以**直接操作座位图来添加和管理过道**，无需打开配置弹窗。这种方式类似于Excel表格编辑，提供了更直观、更快速的过道配置体验。

## 🎯 关键优势

| 特性 | 说明 |
|------|------|
| **直观操作** | 在看到的座位图上直接点击按钮，即时看到结果 |
| **速度快** | 无需打开弹窗，减少多步交互 |
| **座位编号保持连贯** | 添加过道不会影响现有座位的编号 |
| **灵活配置** | 可以随时在任何位置添加或调整过道 |
| **实时保存** | 所有操作自动保存，无需手动确认 |

## 📍 界面布局

### 操作按钮位置

```
                  ⬅ ⬅ ⬅ ⬅ (每列一个)
                   ↓  ↓  ↓  ↓
        ⬆ 过 座座座座座 ➡
        ⬇ 道 1  2  3  4  5 → 行末右边插入过道

        ⬆ 1  座座座座座 ➡
        ⬇ 排 1  2  3  4  5

        ⬆ 2  座座座座座 ➡
        ⬇ 排 1  2  3  4  5

        ⬆ 3  座座座座座 ➡
        ⬇ 排 1  2  3  4  5
```

**说明：**
- 顶部 `⬅` 按钮：在该列左边插入过道列
- 左侧 `⬆` 按钮：在该行上方插入过道行
- 左侧 `⬇` 按钮：在该行下方插入过道行
- 右侧 `➡` 按钮：在该行的最后一列右边插入过道列

## 🚀 使用步骤

### 步骤1：打开座位管理页面

1. 登录智能排座系统
2. 进入"座位管理"页面
3. 选择或创建一个会议

### 步骤2：创建座位区域

1. 点击 **"添加区域"** 按钮
2. 填写区域信息：
   - **区域名称**：如"普通区"、"VIP区"等
   - **区域类型**：选择"普通"或"VIP"
   - **行数**：设置初始行数（如10）
   - **列数**：设置初始列数（如15）
   - **会场布局**：选择"普通式"、"U型"或"圆形"
3. **可选**：选择过道模式
   - 无：不创建过道
   - 左侧：在左边创建过道
   - 右侧：在右边创建过道
   - 中间：在中间创建过道
   - 双边：左右两边各一条过道
   - 三边：左中右三条过道
   - 十字：纵向和横向各一条过道
4. 点击 **"保存"** 创建区域

### 步骤3：在主座位图上添加过道

在座位区域显示区中，现在可以看到每行左侧有操作按钮，每列顶部有操作按钮：

#### 操作1：在某行的上方插入过道

```
现状：
第1排 □□□□□
第2排 □□□□□

操作：在第2排上方点击 ⬆ 按钮

结果：
第1排 □□□□□
过道  ░░░░░
第2排 □□□□□
```

**步骤：**
1. 找到要在上方插入过道的行
2. 点击该行左侧的 `⬆` 按钮
3. 一条新的过道行会出现在该行的上方
4. 其他行的编号不变

#### 操作2：在某行的下方插入过道

```
现状：
第2排 □□□□□
第3排 □□□□□

操作：在第2排下方点击 ⬇ 按钮

结果：
第2排 □□□□□
过道  ░░░░░
第3排 □□□□□
```

**步骤：**
1. 找到要在下方插入过道的行
2. 点击该行左侧的 `⬇` 按钮
3. 一条新的过道行会出现在该行的下方
4. 其他行的编号不变

#### 操作3：在某列的左边插入过道

```
现状：
□ □ □ □ □
□ □ □ □ □
□ □ □ □ □

操作：在第2列左边点击 ⬅ 按钮（顶部）

结果：
□ ░ □ □ □
□ ░ □ □ □
□ ░ □ □ □
```

**步骤：**
1. 在座位区域上方的列操作栏找到要操作的列
2. 点击该列上方的 `⬅` 按钮
3. 一条新的过道列会出现在该列的左边
4. 所有行的座位编号保持不变

#### 操作4：在某列的右边插入过道

```
现状：
□ □ □ □ □
□ □ □ □ □
□ □ □ □ □

操作：在最后一列右边点击 ➡ 按钮（行末）

结果：
□ □ □ □ ░
□ □ □ □ ░
□ □ □ □ ░
```

**步骤：**
1. 在每行的最后一个座位的右侧点击 `➡` 按钮
2. 一条新的过道列会出现在最后一列的右边
3. 所有行的座位编号保持不变

## 📊 座位编号示例

### 初始状态（4行×5列）
```
   1  2  3  4  5
1: □  □  □  □  □
2: □  □  □  □  □
3: □  □  □  □  □
4: □  □  □  □  □
```

### 在第2行下方插入过道行后
```
   1  2  3  4  5
1: □  □  □  □  □
2: □  □  □  □  □
   ░  ░  ░  ░  ░  (过道行，无编号)
3: □  □  □  □  □
4: □  □  □  □  □
```
**注意：** 第3行仍编号为"3排"，不会变成"4排"

### 在第3列左边插入过道列后
```
   1  ░  2  3  4  5
1: □  ░  □  □  □  □
2: □  ░  □  □  □  □
3: □  ░  □  □  □  □
   ░  ░  ░  ░  ░  ░ (过道行)
4: □  ░  □  □  □  □
```
**注意：** 所有座位的编号保持不变，只是在座位间插入了过道

## 💡 最佳实践

### 情景1：大型会议，需要多条走道

1. 创建座位区域（如12排×20列）
2. **在第6排下方插入过道**（分割上下两部分）
3. **在第10列右边插入过道**（分割左右两部分）
4. **在第3排和第9排下方各插入过道**（创建进出通道）

结果：形成"米"字型的走道布局，方便参会人员进出和分散

### 情景2：VIP区单独通道

1. 创建VIP区域（如3排×8列）
2. **在第2排下方插入过道**（VIP区与普通区分隔）
3. **在第4列右边插入过道**（VIP通道）

结果：VIP区有专用的进出通道和休息空间

### 情景3：圆形/U型布局特殊需求

- 对于U型布局，在合适位置插入过道行可以改善视线和通行
- 对于圆形布局，不需要过度使用过道，保持开放感

## ⚠️ 注意事项

### 不会发生什么

❌ **座位编号不会改变**
- 添加过道前：1排有座位1-20号
- 添加过道后：1排仍然有座位1-20号（编号不变）

❌ **其他行的编号不会改变**
- 添加过道行不会导致后续行重新编号
- 添加过道列不会影响座位号

❌ **已分配的参会人员座位不会变**
- 如果座位A5已分配给某个参会人员
- 添加过道不会影响该分配

### 可以改变什么

✅ **座位在视觉上的位置**
- 插入过道后，显示位置会改变
- 但逻辑编号保持不变

✅ **通行路线**
- 过道的增加会改善会场流动

✅ **视觉布局**
- 过道会通过灰色显示，区分座位区域

## 🔄 撤销操作

目前版本中，如果不小心添加了过道，处理方法：

1. **刷新页面**：会回到上次保存的状态
2. **编辑区域**：重新配置过道模式
3. **删除区域**：删除整个区域重新创建

> ℹ️ 提示：后续版本将支持"撤销"（Ctrl+Z）功能

## 🆘 常见问题

### Q1：为什么过道行没有编号？

**A：** 过道行不占用"排"的编号，这样座位编号就保持连贯。如果过道行也占用编号，那么插入过道就会导致现有座位编号改变，造成混乱。

### Q2：过道座位会参与统计吗？

**A：** 不会。过道座位（编号为null）不计入座位总数、可用座位数等统计信息。只有真实的座位（有编号的）才会被计算。

### Q3：能否调整过道的宽度？

**A：** 当前版本中，每次插入都是单列/单行过道。未来版本将支持调整过道宽度。

### Q4：能否删除已插入的过道？

**A：** 当前版本不支持直接删除过道。可以通过编辑区域配置或删除整个区域重新创建来处理。

### Q5：过道能否进行座位分配？

**A：** 不能。过道座位在前端显示时会显示为灰色虚线样式，无法被点击和分配。
