# 手动座位调整功能说明

## 功能概述

智能排座系统现已支持6种手动座位调整方式，方便会务人员灵活调整座位分配。

---

## 1. 拖拽式调整

### 使用方法
1. 点击工具栏中的"拖拽模式"按钮切换到拖拽模式
2. 将已有人员的座位拖动到目标座位
3. 释放鼠标完成调整

### 支持操作
- **移动到空座位**：将人员移动到空座位
- **交换座位**：将两个人的座位互换

### 视觉反馈
- 拖拽中的座位会半透明并放大
- 目标座位会显示虚线边框

---

## 2. 点击选择式

### 使用方法
1. 点击第一个座位选中
2. 点击第二个座位执行操作

### 支持操作
- **单击选择**：选中/取消选中座位
- **Ctrl+点击**：添加到多选/从多选移除
- **Shift+点击**：范围添加到选择
- **选中后点击空座位**：移动人员到空座位
- **选中后点击已占座位**：交换两人座位

### 视觉反馈
- 选中的座位会显示高亮边框

---

## 3. 搜索定位式

### 使用方法
1. 在搜索框中输入姓名、部门或手机号
2. 从下拉列表选择目标人员
3. 系统自动定位并高亮其座位

### 功能特点
- 支持模糊搜索
- 显示搜索结果的座位信息
- 自动滚动到目标座位位置
- 闪烁高亮3次便于定位

---

## 4. 批量调整式

### 使用方法
1. 使用Ctrl+点击或Shift+点击选择多个座位
2. 使用工具栏的批量操作按钮

### 支持操作
- **清空座位**：批量清空选中座位的分配
- **交换座位**：选中两个座位后可交换人员
- **填充剩余**：将未分配人员填入选中的空座位

---

## 5. 表单编辑式

### 使用方法
1. **双击**任意座位打开详情面板
2. 在右侧面板中编辑座位信息
3. 点击保存完成修改

### 可编辑项
- **入座人员**：从下拉列表选择或清空
- **VIP状态**：设置/取消VIP座位
- **备注**：添加座位备注信息

---

## 6. 快捷键操作

### 快捷键列表

| 快捷键 | 功能 |
|--------|------|
| Click | 选择/取消选择座位 |
| Ctrl+Click | 多选座位 |
| Shift+Click | 范围选择 |
| 双击 | 打开座位详情面板 |
| Delete | 清空选中座位 |
| Ctrl+X | 剪切座位人员 |
| Ctrl+V | 粘贴到座位 |
| Escape | 取消选择/关闭面板 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |

### 查看快捷键
点击工具栏中的"快捷键"按钮可显示快捷键提示面板。

---

## 界面元素说明

### 搜索定位工具栏
位于座位图上方，包含：
- 搜索输入框
- 编辑模式切换按钮（点选/拖拽/框选）
- 快捷键提示按钮

### 批量操作工具栏
选中座位后自动显示，包含：
- 已选座位数量显示
- 取消选择按钮
- 清空座位按钮
- 交换座位按钮（选中2个座位时显示）
- 填充剩余按钮

### 座位详情面板
双击座位后从右侧滑出，包含：
- 座位基本信息
- 入座人员选择
- VIP状态设置
- 备注输入
- 保存/取消按钮

### 编辑模式指示器
切换到拖拽或框选模式时，页面顶部显示当前模式提示。

---

## 注意事项

1. **自动保存**：所有座位调整会自动保存到本地存储
2. **变更记录**：所有操作都会记录到变更历史中，可查看和撤销
3. **右键菜单**：右键点击座位可访问更多操作选项（插入过道等）
4. **性能考虑**：大量座位时建议使用搜索定位功能快速找到目标
