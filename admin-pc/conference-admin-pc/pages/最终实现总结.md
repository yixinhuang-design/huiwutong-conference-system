  ```javascript
  功能：在指定行的上方或下方插入过道行
  参数：area（区域对象），rowIndex（行索引），direction（'above'或'below'）

    1. 验证参数有效性

  ```

- **函数2：insertColumnAisle(area, colIndex, direction)**
  ```javascript
  功能：在指定列的左边或右边插入过道列（影响所有行）

  流程：
    1. 验证参数有效性
    2. 根据direction确定插入位置
    3. 遍历area.rows中的每一行：
       - 创建过道座位对象（id, number=null, col, row, isAisle=true）
       - 在insertPosition位置插入该座位
       - 更新该行后续座位的col属性（+1）
    4. 调用saveData()保存数据
    5. 调用updateStatistics()更新统计
    6. 输出debug日志
  ```

## 🔧 技术实现细节

### 关键设计决策

#### 1. 过道行的number设为null
**原因：** 防止座位编号混乱
- 如果过道行也占用number（如编号为"2.5"），会造成编号不连贯
- 设为null清晰表示过道行不参与编号
- 其他行仍保持原来的编号（1,2,3...）

```javascript
// 插入过道后，编号仍为：
Row 1: seats 1-20
Row 2: seats 1-20
Row null (aisle): no seats
Row 3: seats 1-20 ← 仍然是"第3排"，不变成"第4排"
```

#### 2. 过道座位的number设为null
**原因：** 与座位数统计分离
- 过道座位不计入座位总数、可用座位数等统计
- 前端显示时过道座位无编号，用户无法在上面分配
- 保证统计数据的准确性

#### 3. 使用isAisle标志
**原因：** 多层次标识
- `row.isAisle = true` - 标记整行为过道
- `seat.isAisle = true` - 标记单个座位为过道
- 支持行级和座位级的过道标识

#### 4. 重新编号逻辑
**原因：** 保证编号连贯
```javascript
let rowNumber = 1;
area.rows.forEach((row, idx) => {
  if (!row.isAisle) {           // 跳过过道行
    row.number = rowNumber;     // 顺序编号
    rowNumber++;                // 继续计数
  }
  // 过道行的number保持null
});
```

#### 5. 列号更新逻辑
**原因：** 保证列号准确
```javascript
for (let i = insertPosition + 1; i < row.seats.length; i++) {
  row.seats[i].col = i + 1;    // 更新插入位置之后的所有座位
}
```

## 📊 数据流示例

### 初始状态
```javascript
area = {
  rows: [
    { number: 1, isAisle: false, seats: [{number: 1, col: 1, ...}, ...] },
    { number: 2, isAisle: false, seats: [{number: 1, col: 1, ...}, ...] },
    { number: 3, isAisle: false, seats: [{number: 1, col: 1, ...}, ...] }
  ]
}
```

### 在第2行下方插入过道行后
```javascript
area = {
  rows: [
    { number: 1, isAisle: false, seats: [...] },
    { number: 2, isAisle: false, seats: [...] },
    { number: null, isAisle: true, seats: [{number: null, ...}, ...] },  ← 新过道行
    { number: 3, isAisle: false, seats: [...] }  ← 仍然是number: 3
  ]
}
```

### 在第3列左边插入过道列后（对所有行）
```javascript
area = {
  rows: [
    {
      seats: [
        {number: 1, col: 1, ...},
        {number: 2, col: 2, ...},
        {number: null, col: 3, isAisle: true, ...},  ← 新过道座位
        {number: 3, col: 4, ...},  ← col从3更新到4
        {number: 4, col: 5, ...}   ← col从4更新到5
      ]
    },
    // ... 其他行相同
  ]
}
```

### 时间复杂度
| 操作 | 复杂度 | 实际耗时 |
|------|-------|---------|
| insertRowAisle | O(n) | < 1ms (n=行数) |
| insertColumnAisle | O(n×m) | < 2ms (n×m=座位数) |
| 完整操作 | O(n×m) | < 5ms (含保存) |

| 操作 | 复杂度 | 说明 |
|------|-------|------|
| 新过道行 | O(m) | m个座位对象 |
| 新过道座位 | O(n) | n个行对象 |
| 数据保存 | O(n×m) | 整个座位区 |
