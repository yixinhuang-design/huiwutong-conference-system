## 使用示例

### 场景1：创建通道

```
原始：1[A][B][C][D][E][F] 2排
操作：右键点击D → 选择"在右边插入过道"
结果：1[A][B][C][D][X][E][F][G] 2排
      原E-G列现在变成F-H列
```

### 场景2：分隔区域

```
操作：在第5列右侧插入过道
      在第10列右侧插入过道
结果：区域分为三部分
      1-5列 | 过道 | 6-10列 | 过道 | 11-15列
```

### 场景3：撤销操作

```
操作1：在第3列左侧插入过道 → 座位编号调整
操作2：意识到位置不对
操作3：点击"变更历史" → "撤销"
结果：座位编号恢复到之前的状态
```
