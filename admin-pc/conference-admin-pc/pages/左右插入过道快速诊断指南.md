# 左右插入过道 - 快速验证指南

## 🔍 问题诊断

如果你右键点击座位后仍然看不到"在左边/在右边插入过道"选项，请按照以下步骤诊断：

## ✅ 快速验证步骤

### 第1步：打开浏览器开发者工具

按 **F12** 打开开发者工具，进入 **Console** 标签

### 第2步：右键点击座位

在座位图中，用**鼠标右键点击任意座位**（不是行号，是座位本身）

### 第3步：查看 Console 日志

应该看到类似的日志输出：

```
[showSeatContextMenu] 座位右键菜单 - 列级操作 {
  type: "column",
  col: X,
  row: Y,
  areaName: "...",
  visible: true,
  x: ...,
  y: ...
}
```

### 第4步：检查菜单是否显示

**如果看到上面的日志，说明方法被正确调用了，现在检查：**

1. **菜单是否在鼠标位置显示？**
   - 应该有一个白色的菜单框出现在右键位置

2. **菜单内容是否正确？**
   - 应该有两个选项：
     - ← 在左边插入过道
     - → 在右边插入过道

3. **是否有左右箭头图标？**
   - 如果没有图标，可能是 Font Awesome 图标库没有加载

---

## 🐛 常见问题排查

### 问题1：菜单没有显示，也没有日志

**原因：** 座位右键事件没有被正确触发

**解决方法：**
1. 在 Console 中运行：`console.log('showSeatContextMenu方法存在:', typeof showSeatContextMenu)`
2. 确保你点击的是座位 div，而不是座位外面的空白区域
3. 尝试刷新页面

### 问题2：菜单显示但只有上下选项

**原因：** 可能是菜单的 `type` 仍然是 `'row'` 而不是 `'column'`

**解决方法：**
1. 在 Console 中运行：`console.log('contextMenu对象:', contextMenu)`
2. 查看 `type` 的值是否为 `'column'`
3. 如果是 `'row'`，说明方法没有正确更新类型
4. 尝试刷新页面

### 问题3：右键点击座位没有任何反应

**原因：** 事件可能被其他东西阻止了，或者座位元素没有正确绑定事件

**解决方法：**
1. 打开开发者工具的 Elements/Inspector 标签
2. 右键点击座位并选择"检查"或"检查元素"
3. 查看 HTML 中是否有 `@contextmenu.prevent="showSeatContextMenu(...)"` 事件
4. 检查是否有 `pointer-events: none` 阻止了事件

## 🧪 手动测试

### 方法1：在 Console 中直接调用

```javascript
// 模拟座位对象
const mockSeat = { col: 5, id: 'test' };
const mockArea = { name: 'A区', id: 'area_a' };
const mockRow = { number: 3 };
const mockEvent = { pageX: 100, pageY: 200 };

// 创建伪事件对象
mockEvent.preventDefault = () => console.log('preventDefault called');
mockEvent.stopPropagation = () => console.log('stopPropagation called');

// 调用方法
showSeatContextMenu(mockEvent, mockSeat, mockArea, mockRow, 0);

// 检查结果
console.log('contextMenu.visible:', contextMenu.visible);
console.log('contextMenu.type:', contextMenu.type);
console.log('contextMenu.colIndex:', contextMenu.colIndex);
```

### 方法2：查看响应式数据

```javascript
// 检查 contextMenu 对象的当前状态
console.log('完整的contextMenu对象:', contextMenu);

// 也可以查看特定属性
console.log('菜单是否可见:', contextMenu.visible);
console.log('菜单类型:', contextMenu.type);
console.log('列索引:', contextMenu.colIndex);
```

## 💡 调试技巧

### 添加更多日志

如果需要追踪问题，可以在 Console 中运行：

```javascript
// 监听 contextMenu 变化（如果支持）
watch(() => contextMenu.visible, (newVal) => {
    console.log('菜单可见性变化:', newVal);
});

watch(() => contextMenu.type, (newVal) => {
    console.log('菜单类型变化:', newVal);
});
```

### 检查事件绑定

```javascript
// 找到第一个座位元素
const seatElement = document.querySelector('.seat');
console.log('座位元素:', seatElement);
console.log('座位HTML:', seatElement?.outerHTML.substring(0, 200));
```
