常见问题排查

问题1: 400 Bad Request
→ 请求参数格式不正确，或缺少必要的认证令牌
→ 解决: 检查是否有有效的 Authorization header
问题2: 404 Not Found
→ API 端点不存在，检查URL是否正确
→ 后端服务可能没有启动
问题3: CORS 错误
→ 跨域请求被浏览器阻止
→ 解决: 检查后端是否配置了 CORS
问题4: 403 Forbidden
→ 用户权限不足，需要超级管理员权限
→ 解决: 确保使用了有效的超级管理员 token

🚀 快速开始

1. 启动后端服务:
cd backend/conference-backend
mvn spring-boot:run -pl conference-auth

2. 检查后端连接:
点击上面的"检查后端"按钮，看是否返回200状态码

3. 获取租户列表:
如果连接成功，点击"获取列表"按钮查看所有租户

4. 如遇到错误:
- 查看浏览器 F12 Console 标签的详细错误信息
- 检查后端服务日志
- 确认数据库连接正常
