#### 1. 通知管理移动版
**文件名**：`notice-mobile.html`  
**路径**：`app/staff/notice-mobile.html`  
**大小**：约8KB  
**功能**：协管员快速发送各类通知

**核心功能**：
```html
✅ 快捷统计卡片      - 总参会者 142人、待发送 6条、已发送 23条
✅ 快捷发送网格      - 6个常用通知类型（2×3布局）
✅ 模板库面板        - 5个预设模板（可折叠展开）
✅ 紧凑通知列表      - 状态分组（待发送、已发送、草稿）
✅ 一键快速发送      - 每条通知旁都有立即发送按钮
✅ 状态筛选          - 4个快速筛选按钮（全部、待发、已发、草稿）
```

**代码亮点**：
- 响应式设计，完美适配移动屏幕
- 紧凑卡片设计，一屏显示多条信息
- 快速交互，无需进入编辑页面
- 预设值减少用户输入

## 🎨 设计亮点

### 1. 紧凑卡片设计
```html
后台（详细）：
<div class="card">
  <div class="card-header">标题</div>
  <div class="assistant-item">项目1</div>
  <div class="assistant-item">项目2</div>
  <div class="assistant-item">项目3</div>
  <div class="assistant-item">项目4</div>
  <div class="cta-bar">
    <button>按钮1</button>
    <button>按钮2</button>
  </div>
</div>

移动端（紧凑）：
<div class="notice-item-compact">
  <div class="notice-title">标题</div>
  <div class="notice-desc">简要说明</div>
  <button class="quick-send-btn">操作</button>
</div>
```

高度对比：后台 220px → 移动端 70px，节省 **68%** 空间

### 2. 快捷操作设计
```html
后台：无快捷按钮，需要进入详情页才能发送

移动端：
<button onclick="quickSend('seat')">
  <i class="fas fa-chair"></i> 座位通知
</button>
```

减少操作步数：5步 → 1步，**减少80%** 操作

### 3. 颜色优先设计
```html
预警优先级视觉化：

🔴 高优先级：红色背景 + 脉冲指示器 + 加粗文字
🟠 中优先级：黄色背景 + 普通指示器
🔵 低优先级：蓝色背景 + 普通指示器
🟢 已处理：绿色背景 + 半透明

用户可以通过颜色一眼识别优先级
```

### 4. 一屏多信息设计
```html
通知列表：
- 每个通知项高约70px
- 一屏可显示 5-6 条通知
- 支持滑动查看更多
- 核心信息（标题、描述、操作）全在一行内

预警列表：
- 每个预警项高约80px
- 一屏可显示 4-5 条预警
- 优先级颜色突出显示
- 包含操作按钮
```
