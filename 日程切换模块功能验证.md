## 🎯 功能流程

### 1️⃣ 页面加载时
```
onMounted() 
  → initializeConference()
  → loadSchedules()
    → 加载示例日程数据 (3个)
    → 自动选择第一个日程
    → 触发 selectSchedule()
      → 填充当前日程信息
      → 显示日程选择区域 (schedule-selector-area)
```

### 2️⃣ 用户点击"切换日程"按钮
```
openScheduleSelector()
  → showScheduleModal.value = true
  → 显示日程选择弹窗 (modal-overlay)
  → 列出所有可用日程 (schedulesList)
  → 当前日程显示徽章标记
```

### 3️⃣ 用户选择一个日程
```
selectSchedule(schedule)
  → 填充确认对话框数据
    - scheduleId, scheduleName, venueName, capacity
    - attendeeCount, assignedCount
  → 关闭日程列表弹窗
  → 显示确认对话框 (scheduleChangeConfirm.show = true)
```

### 4️⃣ 用户确认日程切换
```
confirmScheduleChange()
  → 保存当前日程的排座结果
  → 加载新日程的排座数据
  → 更新 currentScheduleId
  → 关闭确认弹窗
  → 刷新UI显示新日程
```
