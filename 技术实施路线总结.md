## 🔑 关键技术点

### 1. 多租户数据隔离
- **策略**：数据库级隔离（独立数据库）
- **实现**：动态数据源切换 + MyBatis拦截器
- **性能**：连接池按租户隔离，缓存策略优化

### 2. 微服务注册与发现
- **注册中心**：Nacos
- **配置中心**：Nacos Config
- **服务调用**：OpenFeign + 负载均衡

### 3. 分布式事务
- **方案**：Seata（AT模式）
- **场景**：创建会议时同时创建群组和发送通知

### 4. 消息队列异步处理
- **队列**：Kafka
- **场景**：报名事件、通知发送、数据统计
- **流处理**：Flink实时计算

### 5. 认证授权
- **认证**：JWT Token
- **授权**：RBAC权限模型
- **网关**：统一鉴权拦截

## 🚀 快速开始

### 步骤1：创建项目结构

```bash
# 创建项目目录
mkdir -p backend/conference-backend
cd backend/conference-backend

# 创建Maven父项目
# 参考：技术实施详细指南.md 第1.1节
```

### 步骤2：部署Nacos

```bash
# Docker方式（推荐）
docker run -d \
  --name nacos \
  -p 8848:8848 \
  -e MODE=standalone \
  nacos/nacos-server:v2.2.3

# 访问 http://localhost:8848/nacos
# 默认账号：nacos/nacos
```

### 步骤3：启动第一个服务

```bash
# 启动Gateway
cd conference-gateway
mvn spring-boot:run

# 验证
curl http://localhost:8080/actuator/health
```
