## 🚀 使用指南

### 1. 工作人员使用流程（handbook-mobile.html）

#### 第1步：快速选择（1-2分钟）

```
1. 打开"学员手册生成"页面
2. 在模板卡片中选择一个模板
   - 推荐：标准模板（适合大多数场景）
   - 详细模板：需要更多信息时选择
3. 选择排序方式
   - 按单位：默认方式
   - 按序号：数字序列排序
4. 确认继续到第2步
```

**最佳实践**：
- ✅ 首次使用选择"标准模板"
- ✅ 大多数场景选择"按单位"排序
- ✅ 不确定时不用改，用默认值

---

#### 第2步：内容编辑（2-3分钟）

```
1. 点击"培训须知"开始编辑
   - 弹出富文本编辑框
   - 编辑培训要点（已预填325字）
   - 点击"保存"确认

2. 点击"册尾内容"添加或编辑
   - 可添加致辞、备注、附注等
   - 可不编辑（点击"保存"即可跳过）

3. 点击"会议日程"快速查看
   - 显示3天的日程安排
   - 如需修改，返回后台编辑

4. 点击"学员名册"快速查看
   - 显示142人名单
   - 显示6个分组信息
   - 如需修改，返回后台编辑
```

**最佳实践**：
- ✅ 只编辑必要的2项（须知+册尾）
- ✅ 其他项目如需改动，返回后台编辑后重新生成
- ✅ 编辑完成后立即保存

---

#### 第3步：预览检查（1-2分钟）

```
1. 点击"封面预览"查看手册封面
   - 确保标题、时间、LOGO正确

2. 点击"手册预览"查看完整手册
   - 检查培训须知内容是否正确

3. 点击"名册预览"查看学员名单
   - 检查人数、分组、排序是否正确

4. 点击"完整预览"查看全部内容
   - 检查整个手册的页面数和排列
```

**最佳实践**：
- ✅ 至少预览一次确保内容正确
- ✅ 如发现错误，返回第2步修改
- ✅ 预览满意后继续生成

---

#### 第4步：生成导出（1分钟）

```
1. 点击"生成PDF名册"按钮
   - 开始生成，显示加载动画
   - 约20-30秒后完成
   - 显示"生成成功"提示

2. 点击"下载到本地"
   - PDF文件下载到设备
   - 可随时打开和打印

3. 点击"分享链接"
   - 生成7天有效的分享链接
   - 可发送给其他人查看
   - 限制下载次数为5次（防止泄露）
```

**最佳实践**：
- ✅ 生成后立即下载保存到本地
- ✅ 分享链接仅发给需要的人
- ✅ 过期链接需要重新生成

## 🐛 常见问题与解决方案

### Q1: 编辑后的内容没有保存怎么办？

**解决方案**：
1. 确保点击了"保存"按钮
2. 检查浏览器是否禁用了localStorage
3. 尝试清除浏览器缓存后重新编辑

### Q2: PDF生成失败怎么办？

**解决方案**：
1. 确保网络连接正常
2. 检查生成中的内容是否过大
3. 尝试使用默认模板而不是详细模板

### Q3: 生成的PDF名册人数不对？

**解决方案**：
1. 返回后台检查学员名单是否完整
2. 确认分组设置是否正确
3. 重新点击"预览" → "名册预览"验证

### Q4: 分享链接如何延长有效期？

**解决方案**：
1. 点击"分享链接"重新生成新链接
2. 新生成的链接有效期为7天
3. 每次分享时都可以生成新链接

### Q5: 如何修改已生成的手册？

**解决方案**：
1. 返回编辑页面
2. 修改相应的内容
3. 点击"预览"检查修改效果
4. 点击"生成PDF名册"重新生成
