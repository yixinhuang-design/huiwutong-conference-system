## 💡 实际使用示例

### 最简单的使用方式

```javascript
// 1. 生成座位号（单个）
const gen = new SeatNumberGenerator(5, 10, 'left_to_right');
const number = gen.generate(2, 5);  // 输出：15

// 2. 批量生成座位配置
const seats = gen.generateAll();
console.table(seats);

// 3. 创建座位区域（集成版）
const area = createSeatingArea(
  'area_a',
  'A区',
  'vip',
  5,
  10,
  0,
  'center_outward'  // ← 指定编号方式
);
```

### 在Vue中使用

```javascript
// 在 seating-mgr.html 的 setup() 中
const seatingAreas = ref([
  createSeatingArea('area_stage', '主席台', 'stage', 1, 8, 0, 'center_outward'),
  createSeatingArea('area_a', 'A区', 'vip', 5, 10, 0, 'center_outward'),
  createSeatingArea('area_b', 'B区', 'regular', 10, 15, 0, 'left_to_right'),
]);
```

### 在HTML模板中显示

```html
<!-- 显示座位号（而不是行列位置） -->
<div v-for="seat in area.seats" :key="seat.id" class="seat">
  <span class="seat-number">{{ seat.number }}</span>
</div>
```

## 🎯 选择合适的编号方式

### 按会场类型选择

**📊 会议厅 / 报告厅**
```
推荐: 由左至右 或 左右右左
理由: 易理解，便于通道
```

**🎭 剧院 / 演讲厅**
```
推荐: 左右右左 或 右左左右
理由: 蛇形，最小化通道占用
```

**🎖️ VIP区 / 贵宾区**
```
推荐: 中间左右
理由: 对称美感，中央最重要
```

**🎤 舞台 / 主席台**
```
推荐: 中间左右
理由: 演讲者居中，编号1在正中央
```
