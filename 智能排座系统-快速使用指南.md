**版本**: v2.0

**适用对象**: 会议管理员、会场工作人员

## 🚀 快速开始（5分钟上手）

### 第一次使用

#### 第一步：创建座位布局（2分钟）

1. 打开 `seating-mgr.html` 页面
2. 点击 **"添加区域"** 按钮
3. 配置座位区：
   - 区域名称：例如 "A区 (VIP区域)"
   - 区域类型：选择 "VIP区域"
   - 行数：输入 5，列数：输入 10
   - 点击 **"创建座位区"**

4. 重复步骤2-3，创建更多区域（如 "B区"、"C区"）

#### 第二步：导入参会人员（1分钟）

**方式一：从报名系统导入**
1. 在报名管理页面审核通过报名
2. 点击 **"从报名系统导入"** 按钮
3. 系统自动导入已审核通过的人员

**方式二：Excel导入**
1. 准备Excel文件（包含姓名、单位、职位等列）
2. 在快捷操作区域选择导入模式：
   - "仅导入名单" - 导入后自动排座
   - "导入名单+座位号" - 按Excel中的座位号分配
3. 点击 **"导入Excel"** 按钮

#### 第三步：执行智能排座（1分钟）

1. 在工具栏选择排座算法：
   - **智能推荐**：VIP优先 + 按部门集中 + 职级排序
   - **按部门排座**：部门内部集中
   - **按职级排座**：职级从高到低
   - **按分区排座**：均匀分配到各区

2. 点击 **"执行排座"** 按钮
3. 确认排座，系统自动分配座位

#### 第四步：导出和分享（1分钟）

1. 生成LED导示图：
   - 点击 **"LED导示图"** 按钮
   - 自动下载高清图片（用于现场大屏）

2. 导出座位表：
   - 点击 **"导出座位表"** 按钮
   - 选择导出格式（Excel/PDF）

3. 生成查座二维码：
   - 点击 **"生成查座二维码"** 按钮
   - 复制查座链接分享给参会者

## 📝 详细操作指南

### 一、座位布局配置

#### 1.1 创建座位区

**操作步骤**：
```
1. 点击"添加区域"按钮
2. 填写区域信息：
   - 区域名称：必填，如"A区 (VIP区域)"
   - 区域类型：VIP区域 / 普通区域
   - 会场布局：普通式 / U型 / 圆桌
3. 配置尺寸：
   - 行数：1-20行（使用 +/- 按钮或直接输入）
   - 列数：1-30列
4. 配置角度（可选）：
   - 拖动滑块或点击快捷按钮（0°/45°/90°/180°）
5. 配置过道（可选）：
   - 勾选"启用过道"
   - 选择过道位置（左侧/中间/右侧）
   - 设置过道宽度（1-5列）
6. 配置固定座位（可选）：
   - 勾选"设置固定座位"
   - 输入固定位置，如："1排1号, 1排2号, 2排1-5号"
7. 查看实时预览和统计
8. 点击"创建座位区"保存
```

**提示**：
- 实时预览会显示当前配置的座位布局
- 统计信息显示总座位数、实际座位数、过道占用数
- VIP区域座位会显示为粉色

#### 1.2 编辑座位区

**暂未实现编辑功能**，如需修改：
1. 删除现有座位区
2. 重新创建

### 二、参会人员管理

#### 2.1 从报名系统导入

**前提条件**：
- 已在报名管理页面添加参会人员
- 参会人员状态为"已审核通过"

**操作步骤**：
```
1. 点击"从报名系统导入"按钮
2. 系统自动导入已审核通过的人员
3. 自动判断VIP身份（根据职位关键词）
4. 显示导入成功提示
```

**VIP职位关键词**：
- 书记、市长、局长、主任、部长、主席、副主席、秘书长、副秘书长

#### 2.2 Excel导入

**模式一：仅导入名单**
- 用途：导入参会人员信息，后续自动排座
- Excel格式要求：
  ```
  姓名    | 单位        | 职位     | 手机号       | 邮箱
  张三    | XX局        | 处长     | 13800138001 | zhang@xx.com
  李四    | XX公司      | 经理     | 13800138002 | li@xx.com
  ```

**模式二：导入名单+座位号**
- 用途：按Excel中的座位号直接分配（所见即所得）
- Excel格式要求：
  ```
  姓名    | 单位    | 职位   | 区域   | 排 | 号
  张三    | XX局    | 处长   | A区    | 1  | 5
  李四    | XX公司  | 经理   | B区    | 2  | 8
  ```

**支持的列名格式**：
- 姓名 / name / Name
- 单位 / 部门 / department
- 职位 / 职务 / position
- 区域 / 座位区
- 排 / 行 / row
- 号 / 列 / col / seat

**操作步骤**：
```
1. 选择导入模式
2. 点击"导入Excel"按钮
3. 选择Excel文件
4. 等待导入完成
```

### 三、智能排座

#### 3.1 选择排座算法

**智能推荐算法**（推荐）
- VIP优先分配（前排居中）
- 按部门集中分配
- 部门内按职级排序

**按部门排座算法**
- 部门内部集中
- 按部门人数排序分配区域

**按职级排座算法**
- 职级从高到低排序
- 前排优先高职级

**按分区排座算法**
- 将参会者均匀分配到各个座位区
- 适用于多区域场景

#### 3.2 执行排座

**操作步骤**：
```
1. 确保已创建座位区域
2. 确保已导入参会人员
3. 选择排座算法
4. 点击"执行排座"按钮
5. 确认排座
6. 等待系统自动分配
```

**注意事项**：
- 排座会清空现有座位分配
- VIP优先分配到VIP区域前排
- 无法分配座位的参会者会被提示

**操作步骤**：

```
   - 房间号：如"301"

   - 容量：可容纳人数
   - 入住时间：选择日期和时间
   - 退房时间：选择日期和时间
```

**操作步骤**：

## 💡 常见问题

### Q1: 如何修改已创建的座位区？

**A**: 当前版本暂不支持编辑，请删除后重新创建。删除操作会清空该区域的所有座位分配。

### Q2: VIP是如何判断的？

**A**: 根据职位关键词自动判断：
- 书记、市长、局长、主任、部长等
- 可在代码中修改 `isVipByPosition()` 方法添加更多关键词

### Q3: 导入Excel时提示找不到文件？

**A**: 确保：
- Excel文件格式正确（.xlsx 或 .xls）
- 文件包含必要的列（姓名、单位、职位等）
- 列名格式正确（支持多种格式）

### Q4: 排座后还可以调整吗？

**A**: 可以！支持：
- 手动移动参会者到其他座位
- 交换两个参会者的座位
- 批量清空座位
- 填充剩余座位
- 所有变更都会记录在变更历史中

### Q5: 辅助安排是如何分配的？

**A**:
- **乘车**：按部门分组，优先同部门人员同车
- **住宿**：考虑性别和房间类型（单人间/双人间）
- **讨论室**：按部门分配，优先完整部门同室

### Q6: 如何撤销错误的座位变更？

**A**:
1. 在辅助安排区域找到"变更历史"卡片
2. 点击"撤销最后一次变更"按钮
3. 确认撤销

**注意**：当前版本仅记录变更历史，完整的状态恢复功能需要进一步开发。

### Q7: LED导示图分辨率不够高怎么办？

**A**: 当前版本生成1920x1080高清图片。如需更高分辨率：
- 调整代码中的容器尺寸（line 2747-2750）
- 增加 scale 参数（当前为2）

### Q8: 手机查座时找不到座位怎么办？

**A**: 请检查：
1. 姓名输入是否正确
2. 是否使用报名时的姓名
3. 联系会议管理员确认

## 🎯 最佳实践

### 会议准备流程

**会前3天**：
```
1. 创建座位布局（根据会场实际情况）
2. 在报名系统审核参会者报名
3. 从报名系统导入参会人员
```

**会前1天**：
```
1. 执行智能排座（选择合适的算法）
2. 手动调整特殊座位（如有需要）
3. 添加车辆、房间、讨论室
4. 执行智能分配所有
5. 导出座位表（Excel/PDF）
6. 生成LED导示图
7. 生成查座二维码
```

**会议当天**：
```
1. 导示图显示在LED大屏
2. 参会者扫码或输入姓名查座
3. 根据实际情况微调座位
4. 记录所有变更
```

### 数据备份建议

**定期备份**：
- 每次重要操作后导出数据
- 保存导出的Excel和PDF文件
- 定期清理历史数据

**数据清理**：
- 会议结束后导出变更历史
- 清空本地数据（准备下一场会议）
- 保存重要的配置模板
