/* 特征 */

✅ 边框效果: 1px glass-border
✅ 阴影效果: 0 20px 60px rgba(0,0,0,0.3)

```css
/* 四层断点 */
1920px+ (桌面)     → 完整显示，900px 宽弹窗

768px (小平板)     → 全宽模态，单列表单
375px (手机)       → 全屏，堆叠按钮

/* 共同特性 */
✅ max-width: 95vw (防止超出屏幕)
✅ max-height: 90vh (防止超过视口)
✅ 可滚动内容 (overflow-y: auto)
✅ 灵活的按钮布局
```
