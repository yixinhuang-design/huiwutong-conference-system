# 报名系统修复 - 一页纸总结

## 问题
用户报名提交成功，但无法查询到报名信息

## 根因
前端 `localStorage` 中租户ID使用了不同的键名：
- 登录时保存为 `'tenant_id'`
- 报名时读取 `'tenantId'` → **找不到！**
- 导致报名提交和查询使用了不同的租户ID

## 解决方案
统一租户ID的读取优先级为：`'tenant_id'` → `'tenantId'` → `'current_tenant_id'` → 默认值
