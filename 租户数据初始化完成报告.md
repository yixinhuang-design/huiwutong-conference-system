### 多租户隔离

- 每个租户有独立的用户、角色、权限
- 用户只能访问自己租户的数据
- 通过 `tenant_id` 实现数据隔离

## 📞 常见问题

### Q: 如何重置所有数据?

A: 执行以下命令清空所有用户和角色，然后重新初始化：

```sql
DELETE FROM sys_role;
DELETE FROM sys_user;
DELETE FROM sys_tenant WHERE id > 2;
-- 然后重新运行初始化脚本
```

### Q: 如何修改用户密码?

A: 使用 BCrypt 生成新密码后更新：

```sql
-- 密码需要使用 BCrypt 加密
UPDATE sys_user SET password = '$2a$10$...' WHERE username = 'admin';
```
