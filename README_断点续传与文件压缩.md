## 🎯 5分钟快速开始

### 1️⃣ 准备代码 (3分钟)
将以下9个文件复制到项目中：
- 4个DTO文件
- 1个配置类
- 1个工具类
- 2个服务类
- 1个控制器
- 1个JavaScript库

### 2️⃣ 配置项目 (1分钟)
编辑 `application.properties`：
```properties
file.upload.max-file-size=104857600
file.upload.chunk-size=5242880
file.upload.enable-image-compress=true
```

### 3️⃣ 启动项目 (1分钟)
```bash
mvn clean compile && java -jar target/*.jar
```

## 📊 关键信息

### 代码量
- 后端: 1,305行 Java (9个类)
- 前端: 600行 JavaScript (1个库)
- 文档: 2,500+行 Markdown (5份)

### API端点 (7个)
| 方法 | 路径 | 功能 |
|------|------|------|
| POST | /api/schedule/chunk/init | 初始化 |
| POST | /api/schedule/chunk/upload | 上传分块 |
| GET | /api/schedule/chunk/progress/{id} | 查询进度 |
| POST | /api/schedule/chunk/finish | 完成上传 |
| DELETE | /api/schedule/chunk/{id} | 取消上传 |
| GET | /api/schedule/chunk/stats/{id} | 获取统计 |
| GET | /api/schedule/chunk/health | 健康检查 |

### 配置项 (9个)
```properties
file.upload.max-file-size=104857600           # 最大100MB
file.upload.chunk-size=5242880                # 5MB/块
file.upload.enable-image-compress=true        # 启用压缩
file.upload.image-quality=85                  # 质量85%
file.upload.temp-file-expire-time=86400000    # 24小时清理
file.upload.enable-resume-upload=true         # 支持断点续传
file.upload.max-retry-count=3                 # 最多重试3次
file.upload.retry-delay=1000                  # 重试延迟1秒
```

### 性能指标
| 文件大小 | 预计耗时 | 分块数 |
|---------|---------|--------|
| 5MB | <10秒 | 1 |
| 50MB | 40-80秒 | 10 |
| 100MB | 80-160秒 | 20 |
