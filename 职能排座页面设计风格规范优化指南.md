## 快速开始

### 引入新样式文件

在 `seating-mgr.html` 的 `<head>` 中添加：

```html
<!-- 职能排座弹窗标准样式 -->
<link rel="stylesheet" href="../css/seating-function-modal.css">
```

### 基础用法

```html
<!-- 弹窗遮罩层 -->
<div class="modal-overlay">
    <!-- 弹窗容器 -->
    <div class="modal-content large">
        <!-- 弹窗头部 -->
        <div class="modal-header">
            <h3><i class="fas fa-magic"></i> 智能排座</h3>
            <button class="close-btn"><i class="fas fa-times"></i></button>
        </div>

        <!-- 弹窗主体 -->
        <div class="modal-body">
            <!-- 内容... -->
        </div>

        <!-- 弹窗页脚 -->
        <div class="modal-footer">
            <button class="btn-outline">取消</button>
            <button class="btn-primary">确认</button>
        </div>
    </div>
</div>
```

## 设计原则

### 1. **玻璃态效果（Glassmorphism）**

所有弹窗采用半透明毛玻璃效果，与全局设计系统保持一致。

```css
/* 遮罩层：深色模糊背景 */
background: rgba(0, 0, 0, 0.55);
backdrop-filter: blur(5px);

/* 弹窗内容：轻微透明度 */
background: rgba(255, 255, 255, 0.95);
border: 1px solid var(--glass-border);
```

**效果特点**：
- ✅ 视觉层级清晰
- ✅ 背景内容可见，保持上下文
- ✅ 现代简约风格
- ✅ 无障碍性好

### 2. **色彩系统**

使用全局CSS变量，确保主题一致性：

```css
/* 主色 */
--primary-color: #667eea

/* 状态色 */
--success-color: #10b981        /* 绿色 - 成功 */
--warning-color: #f59e0b        /* 黄色 - 警告 */
--danger-color: #ef4444         /* 红色 - 危险 */
--info-color: #3b82f6           /* 蓝色 - 信息 */

/* 中性色 */
--gray-100 ~ --gray-900         /* 灰色系 */
```

**应用规范**：
- 主操作按钮：`--primary-color`
- 次操作按钮：`--gray-300`
- 成功提示：`--success-color`
- 错误提示：`--danger-color`

### 3. **排版规范**

```css
/* 标题 */
--font-size-lg: 1.125rem        /* 弹窗标题 */
--font-size-base: 1rem          /* 常规文本 */
--font-size-sm: 0.875rem        /* 描述文本 */
--font-size-xs: 0.75rem         /* 辅助文本 */

/* 字重 */
Font-weight: 600 ~ 700          /* 标题、标签 */
Font-weight: 500 ~ 600          /* 强调文本 */
Font-weight: 400                /* 常规文本 */
```

### 4. **间距规范**

```css
--space-1: 0.25rem     /* 2px    - 微小 */
--space-2: 0.5rem      /* 4px    - 极小 */
--space-3: 0.75rem     /* 6px    - 小 */
--space-4: 1rem        /* 8px    - 正常 */
--space-5: 1.5rem      /* 12px   - 中 */
--space-6: 2rem        /* 16px   - 大 */
--space-8: 3rem        /* 24px   - 更大 */
```

### 5. **圆角规范**

```css
--radius-sm: 4px        /* 输入框 */
--radius-md: 8px        /* 卡片 */
--radius-lg: 12px       /* 弹窗 */
--radius-xl: 16px       /* 大容器 */
--radius-full: 9999px   /* 圆形 */
```

### 6. **过渡效果规范**

```css
--transition-fast: 150ms cubic-bezier(0.4, 0, 0.2, 1)     /* 快速交互 */
--transition-normal: 300ms cubic-bezier(0.4, 0, 0.2, 1)   /* 标准过渡 */
--transition-slow: 500ms cubic-bezier(0.4, 0, 0.2, 1)     /* 缓慢过渡 */
```

## 组件标准

### 1. 弹窗容器

#### 尺寸规范

```html
<!-- 紧凑型（600px）- 简单确认对话框 -->
<div class="modal-content compact">

<!-- 中等型（700px）- 普通表单 -->
<div class="modal-content medium">

<!-- 大型（900px）- 复杂配置 -->
<div class="modal-content large">

<!-- 全屏（95vw x 95vh）- 高级操作 -->
<div class="modal-content full">
```

#### 样式特点

- **背景渐变**：从白色到浅蓝色，增加深度感
- **阴影**：`0 20px 60px rgba(0, 0, 0, 0.3)`，表现出浮动感
- **边框**：微妙的1px浅色边框，与玻璃态融合
- **动画**：`slideUp 0.35s cubic-bezier(0.34, 1.56, 0.64, 1)` - 弹性进入

### 2. 弹窗头部

**结构**：
```html
<div class="modal-header">
    <div>
        <h3><i class="fas fa-icon"></i> 标题</h3>
        <div class="modal-header-subtitle">副标题（可选）</div>
    </div>
    <button class="close-btn"><i class="fas fa-times"></i></button>
</div>
```

**设计特点**：
- 蓝色渐变背景（`--primary-gradient`）
- 白色文本
- 圆形关闭按钮，悬停时旋转
- 图标：12px左右间距
- 副标题：可选，用于说明当前步骤或描述

**响应式处理**：
- 桌面：`padding: 1.5rem`
- 平板：`padding: 1rem`
- 手机：`padding: 0.75rem`，标题转为单行

### 3. 表单组件

#### 输入框组

```html
<div class="form-group">
    <label>
        座位区名称
        <span class="required">*</span>
        <span class="hint">（选填）</span>
    </label>
    <input type="text" placeholder="例如：A区 VIP" />
    <span class="input-hint">输入2-20个字符</span>
</div>
```

**样式规范**：
- **默认状态**：`border: 1px solid #d1d5db`
- **聚焦状态**：`border-color: #667eea; box-shadow: 0 0 0 3px rgba(102, 126, 234, 0.1)`
- **错误状态**：`border-color: #ef4444; box-shadow: 0 0 0 3px rgba(239, 68, 68, 0.1)`
- **禁用状态**：`background: #f3f4f6; color: #9ca3af; cursor: not-allowed`

#### 数字输入器

```html
<div class="number-input">
    <button>−</button>
    <input type="number" value="10" min="1" max="30" />
    <button>+</button>
</div>
```

**功能**：
- 按钮：36px正方形，蓝色悬停效果
- 输入：80px宽，居中对齐
- 实时验证：超出范围时提示
- 快捷键：方向键可调整数值

#### 复选框与开关

```html
<label class="checkbox-label">
    <input type="checkbox" />
    <span>启用VIP座位设置</span>
</label>
```

**样式**：
- 复选框：18px正方形，使用 `accent-color: var(--primary-color)`
- 标签文本：18号字体
- 悬停：整体变蓝

### 4. 配置网格

```html
<div class="config-grid two-column">
    <div class="config-section">
        <h4><i class="fas fa-info-circle"></i> 基本信息</h4>
        <!-- 表单项... -->
    </div>
    <div class="config-section active">
        <h4><i class="fas fa-cogs"></i> 尺寸配置</h4>
        <!-- 表单项... -->
```

**布局规则**：
- 默认：自适应列数（最小320px）
- `.two-column`：固定2列
- `.three-column`：固定3列
- 平板及以下：自动变为单列

**样式特点**：
- 边框：2px solid #e5e7eb（悬停变蓝）
- 背景：半透明白色
- 圆角：`var(--radius-lg)` = 12px
- 过渡：`all 150ms ease-out`
- 活跃状态：背景色变浅蓝，边框变蓝

### 5. 预览区域

```html

    <h4><i class="fas fa-eye"></i> 座位布局预览</h4>
```

- 高度：最小200px，自适应内容
- 统计行：用 `|` 分隔不同项

## 常见问题

### Q: 弹窗显示不完整，内容超出屏幕怎么办？

**A:** 检查以下几点：

```css
/* 确保最大高度设置正确 */
.modal-content {
    max-height: 90vh;
    overflow: hidden;
}

.modal-body {
    overflow-y: auto;
    flex: 1;
}
```

### Q: 如何自定义弹窗宽度？

**A:** 使用 CSS 变量或自定义类：

```css
/* 方法1：使用预设尺寸类 */
<div class="modal-content compact">   <!-- 600px -->
<div class="modal-content medium">   <!-- 700px -->
<div class="modal-content large">    <!-- 900px -->

/* 方法2：自定义样式 */
<div class="modal-content" style="width: 1200px;">

/* 方法3：添加自定义类 */
<style>
.modal-content.custom-wide {
    width: 1200px;
}
</style>
```

### Q: 深色模式下样式是否支持？

**A:** 完全支持！样式已包含深色模式适配：

```css
@media (prefers-color-scheme: dark) {
    /* 深色模式样式已定义 */
}
```

用户系统设置深色模式时会自动应用。

### Q: 如何修改主色（蓝色）为其他颜色？

**A:** 修改 CSS 变量：

```css
/* 在 conference-theme.css 中修改 */
:root {
    --primary-color: #667eea;  /* 改为你想要的颜色 */
}
```

所有使用 `var(--primary-color)` 的组件都会自动更新。

### Q: 弹窗背景半透明度如何调整？

**A:** 修改 `modal-overlay` 的背景颜色：

```css
.modal-overlay {
    background: rgba(0, 0, 0, 0.55);  /* 改第四个数字，范围0-1 */
}

/* 深色背景 */
.modal-overlay.dark {
    background: rgba(0, 0, 0, 0.65);
}

/* 浅色背景 */
.modal-overlay.light {
    background: rgba(0, 0, 0, 0.45);
}
```

### Q: 如何禁用弹窗动画？

**A:** 添加 `no-animation` 类：

```css
.modal-content.no-animation {
    animation: none;
}
```

或在 HTML 中：

```html
<div class="modal-content no-animation">
```

## 设计检查清单

实施前，请确保满足以下标准：

### 视觉一致性
- [ ] 所有弹窗使用统一的配色方案
- [ ] 按钮样式一致
- [ ] 字体和大小符合规范
- [ ] 间距符合间距规范

### 交互体验
- [ ] 弹窗打开/关闭动画流畅
- [ ] 按钮有悬停状态反馈
- [ ] 表单有验证反馈
- [ ] 禁用状态清晰可见

### 可访问性
- [ ] 支持键盘导航（Tab、Enter、Esc）
- [ ] 颜色不是唯一的信息传递方式
- [ ] 文本对比度 >= 4.5:1
- [ ] 支持屏幕阅读器

### 响应式
- [ ] 桌面（1920px+）
- [ ] 平板（768px - 1024px）
- [ ] 手机（< 768px）
- [ ] 极小屏幕（< 480px）

### 性能
- [ ] 页面加载时间 < 3s
- [ ] 动画帧率 >= 60fps
- [ ] 没有布局抖动
- [ ] 内存占用合理
