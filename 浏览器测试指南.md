**方法一：直接打开HTML文件**

**方法二：通过本地服务器（推荐）**

```bash
cd G:\huiwutong新版合集\admin-pc\conference-admin-pc

# 使用Python启动本地服务器
python -m http.server 8080
```

## 🔍 控制台检查要点

### 应该看到的日志

```javascript
// 正常初始化日志
Vue应用已挂载，开始初始化...
会议信息已加载: {id: 'demo_conf_001', name: '示例会议'}
座位区域数量: 3
✓ 已从报名系统自动导入 50 名参会人员 (VIP: 5人)
初始化完成，准备就绪

// 操作日志（如果启用）
排座方式已切换到: excel
算法已切换到: 智能推荐算法
```

### 不应该看到的错误

```javascript
// ❌ 这些错误表示有问题
Uncaught TypeError: Cannot read property 'xxx' of undefined
Uncaught ReferenceError: xxx is not defined
Uncaught SyntaxError: Unexpected token
Error: [Vue warn]: Property or method "xxx" is not defined
```

### 常见错误和原因

| 错误信息 | 可能原因 | 解决方案 |
|---------|---------|---------|
| `Cannot read property 'length' of undefined` | 数组未正确初始化 | 检查data()中数组是否初始化为[] |
| `xxx is not a function` | 方法未定义或拼写错误 | 检查methods中是否有该方法 |
| `Unexpected token` | JavaScript语法错误 | 检查代码中的括号、逗号等 |
| `Failed to load resource` | CSS/JS文件路径错误 | 检查文件引用路径 |

## 💡 测试技巧

1. **逐步测试**: 按照顺序逐个测试，不要一次测试所有功能
2. **记录问题**: 发现问题立即记录，包括截图和错误信息
3. **清除缓存**: 每次修改代码后，刷新页面时按 `Ctrl + F5` 强制刷新
4. **查看网络**: 在Network标签检查CSS/JS文件是否正确加载
5. **断点调试**: 在Chrome DevTools的Sources标签中设置断点调试
