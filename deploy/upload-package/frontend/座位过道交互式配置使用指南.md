# 座位过道交互式配置使用指南

## 功能概述

现在你可以直接在座位布局图上**点击行号或列号**来快速添加或移除过道，操作非常直观！

## 核心功能

### 1. 点击列号添加纵向过道

**操作步骤**：
1. 在座位布局顶部找到**列号**（1, 2, 3, ... 12）
2. **点击某个列号**（如点击列号"6"）
3. 整列会立即被标记为过道（显示虚线框）
4. 列号会高亮显示蓝色

**视觉效果**：
```
点击列号"6"后：

### 2. 点击行号添加横向过道

**操作步骤**：
1. 在座位布局左边找到**行号**（1, 2, 3, ... 8）
2. **点击某个行号**（如点击行号"4"）
3. 整行会立即被标记为过道（显示虚线框）
4. 行号会高亮显示蓝色

**视觉效果**：
```

### 3. 取消过道

**操作步骤**：
- 要取消某列的过道：再次点击对应的**列号**
- 要取消某行的过道：再次点击对应的**行号**
- 立即生效，无需额外确认

### 4. 组合多个过道

可以组合纵向和横向过道，创建十字或网格式的通道：

## 快速模式 vs 自定义模式

### 快速模式按钮
在预览上方有 7 个快速模式按钮：

| 按钮 | 效果 | 说明 |
|------|------|------|
| **无过道** | 所有座位可用 | 没有通道 |
| **左** | 左侧1列过道 | 适合单边通行 |
| **右** | 右侧1列过道 | 适合单边通行 |
| **中** | 中间1列过道 | 最常用，两侧均衡 |
| **左右** | 左右两侧过道 | 双边通行 |
| **左中右** | 左、中、右3列过道 | 大型会场 |
| **十字** | 中间行+中间列 | 网格式通行 |

### 自定义模式
如果快速模式不满足需求，可以：
1. 选择任何快速模式作为基础（如"中"）
2. 直接在预览图上点击行号和列号进行微调
3. 添加或移除不需要的过道

## 交互提示

### 视觉反馈
- **行号/列号悬停时**：背景变灰，文字变蓝，表示可点击
- **已标记为过道的行号/列号**：蓝色背景，蓝色文字，表示已激活
- **座位悬停时**：座位放大并显示阴影
- **过道座位**：显示虚线框，表示这是通道

### 实时统计
预览下方显示实时统计信息：
```
总座位: 96 | 实际: 48 | 过道: 48
```
- **总座位**：布局中的所有座位数
- **实际**：可用于参会人员的座位数
- **过道**：标记为过道的座位数

## 使用流程示例

### 场景1：创建中间有一条走道的会议室

```
1. 点击"创建座位区"
2. 输入区域名称（如"A区"）
3. 设置行数为8，列数为12
4. 点击快速模式中的"中"按钮
5. 预览中间会出现第6列为过道
6. 点击"保存"完成
```

### 场景2：需要左右两条走道

```
1. 点击"编辑座位区"或"创建座位区"
2. 点击快速模式中的"左右"按钮
3. 预览中会显示左侧和右侧为过道
4. 如需微调，可在预览中点击其他行号或列号添加/删除
5. 点击"保存"
```

### 场景3：复杂的自定义过道

```
1. 选择"中"快速模式作为基础
2. 再点击列号"3"添加左侧通道
3. 再点击列号"10"添加右侧通道
4. 再点击行号"2"和行号"7"添加两条横向通道
5. 预览显示网格式通道
6. 点击"保存"
```

## 常见问题

### Q: 点击行号/列号没有反应？
**A**: 
- 确保已选择一个快速模式（按钮会高亮）
- 检查鼠标是否在正确的行号或列号上
- 行号在左侧，列号在顶部

### Q: 如何只标记某几列为过道，而不是整列？
**A**: 
- 点击列号会将整列标记为过道（这是设计的功能）
- 如需更细粒度的控制，可以点击单个座位进行微调

### Q: 过道座位数显示有问题？
**A**: 
- 过道统计会实时更新
- 在"保存"前刷新预览可以看到最新数字
- 保存后，座位区会正确创建

### Q: 能否保存自定义的过道配置模板？
**A**: 
- 目前支持保存到座位区，每次编辑会保留之前的过道配置
- 快速模式是预设的常用配置

## 性能提示

- 对于超过100列的大规模座位区，交互可能会略有延迟
- 点击行号/列号会立即更新所有相关座位，响应时间<100ms
- 预览统计数字每次修改后实时计算

## 技术细节

### 支持的座位区配置
- **行数**：1-20
- **列数**：1-30
- **过道类型**：纵向（列）、横向（行）、组合

### 座位标记
- 被标记为过道的座位显示为虚线框
- 过道座位不会在排座时分配给参会人员
- 统计中的"实际"座位数不包括过道

### 配置保存
- 所有行号和列号的点击操作会自动转换为标准的过道配置格式
- 保存后可以在编辑时恢复
