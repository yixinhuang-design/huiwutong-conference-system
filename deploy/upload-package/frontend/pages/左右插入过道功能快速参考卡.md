# 左右插入过道功能 - 快速参考卡

## 🎯 功能概览

| 功能 | 触发方式 | 快捷菜单 | 效果 |
|-----|--------|--------|------|
| 上方插入过道 | 右键→行号 | ↑ 在上方插入过道 | 在该行上方插入整行过道 |
| 下方插入过道 | 右键→行号 | ↓ 在下方插入过道 | 在该行下方插入整行过道 |
| **左侧插入过道** | 右键→座位 | ← 在左边插入过道 | 在该列左侧插入整列过道 |
| **右侧插入过道** | 右键→座位 | → 在右边插入过道 | 在该列右侧插入整列过道 |

## 🔧 技术点

### 数据流

```
用户右键点击
    ↓
showSeatContextMenu() / showRowContextMenu()
    ↓
设置 contextMenu 对象 (visible=true, type='column'/'row')
    ↓
菜单显示 (v-if 条件满足)
    ↓
用户点击菜单项
    ↓
insertColumnAisle() / insertRowAisle()
    ↓
更新座位编号 + 保存数据 + 更新统计
    ↓
hideContextMenu()
```

### 列号转换关键

```javascript
// 座位的 col 是 1-based：1, 2, 3, 4, 5...
// 数组索引是 0-based：0, 1, 2, 3, 4...

// 左侧插入：在 colIndex - 1 位置
const insertPosition = colIndex - 1;

// 右侧插入：在 colIndex 位置
const insertPosition = colIndex;

// 示例：在第5列右侧插入
colIndex = 5
insertPosition = 5
array.splice(5, 0, aisleSeat)
// 原 seats[5] 成为 seats[6]，所以需要更新列号
```

## 📞 快速支持

| 问题 | 解决方案 |
|-----|--------|
| 菜单不显示 | F12 → Console 检查 contextMenu.visible |
| 过道位置错 | 检查座位的 col 属性值 |
| 编号没更新 | 刷新页面，查看 localStorage 数据 |
| 数据没保存 | 检查浏览器 localStorage 是否启用 |
| 多个座位区出错 | 验证 contextMenu.area 是否为正确区域 |
