## API 文档

### insertRowAisle(area, rowIndex, direction)

**功能：** 在指定行位置插入过道行

**参数：**
| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `area` | Object | 座位区对象 | `{ id: 'area_1', rows: [...] }` |
| `rowIndex` | Number | 行索引（从0开始） | `2` |
| `direction` | String | 插入方向 | `'above'` 或 `'below'` |

**返回值：** 无 (void)

**异常：**
- 如果 `area` 为 null/undefined，输出警告日志
- 如果 `rowIndex` 越界，输出警告日志

**示例：**
```javascript
// 在第3行（索引为2）下方插入过道
insertRowAisle(seatingAreas.value[0], 2, 'below');

// 在第1行（索引为0）上方插入过道
insertRowAisle(seatingAreas.value[0], 0, 'above');
```

**内部流程：**
```
1. 验证参数有效性
2. 根据direction确定insertPosition
   - direction='above' -> insertPosition = rowIndex
   - direction='below' -> insertPosition = rowIndex + 1
3. 获取列数 (colCount = area.rows[0].seats.length)
4. 创建新的过道行对象：
   {
     id: `${area.id}_aisle_row_${Date.now()}`,
     number: null,
     isAisle: true,
     seats: [
       { id, number: null, col, row: null, isAisle: true, ... },
       ...
     ]
   }
5. 使用splice()在指定位置插入
6. 重新编号所有非过道行（row.number = 1, 2, 3, ...）
7. 更新每行座位的row属性
8. 调用saveData()保存
9. 调用updateStatistics()更新统计
10. 输出成功日志
```

### insertColumnAisle(area, colIndex, direction)

**功能：** 在指定列位置插入过道列（影响所有行）

**参数：**
| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `area` | Object | 座位区对象 | `{ id: 'area_1', rows: [...] }` |
| `colIndex` | Number | 列索引（从1开始） | `3` |
| `direction` | String | 插入方向 | `'left'` 或 `'right'` |

**返回值：** 无 (void)

**异常：**
- 如果 `area` 为 null/undefined，输出警告日志
- 如果 `area.rows` 为空，输出警告日志

**示例：**
```javascript
// 在第3列左边插入过道
insertColumnAisle(seatingAreas.value[0], 3, 'left');

// 在第5列右边插入过道
insertColumnAisle(seatingAreas.value[0], 5, 'right');

// 在最后一列右边插入过道
const lastCol = area.rows[0].seats.length;
insertColumnAisle(area, lastCol + 1, 'right');
```

**内部流程：**
```
1. 验证参数有效性
2. 根据direction确定insertPosition
   - direction='left' -> insertPosition = colIndex - 1
   - direction='right' -> insertPosition = colIndex
3. 遍历area.rows中的每一行：
   a. 创建过道座位对象
      {
        id: `${area.id}_aisle_seat_${Date.now()}_${rowIdx}`,
        number: null,
        col: insertPosition + 1,
        row: row.isAisle ? null : row.number,
        isAisle: true,
        status: 'aisle'
      }
   b. 在该行的insertPosition位置插入座位
   c. 更新该行中insertPosition之后所有座位的col属性
4. 调用saveData()保存
5. 调用updateStatistics()更新统计
6. 输出成功日志
```

## 数据结构

### 座位区对象 (Area)

```javascript
{
  id: "area_1",                    // 唯一标识
  name: "普通区",                  // 区域名称
  type: "regular",                 // 'regular' | 'vip'
  venueType: "normal",             // 'normal' | 'ushape' | 'round'
  rows: [
    {
      id: "row_1",
      number: 1,                   // 排号（1, 2, 3...）
      isAisle: false,              // 是否为过道行
      seats: [
        {
          id: "seat_1_1",
          number: 1,               // 座位号（1, 2, 3...）
          col: 1,                  // 列号
          row: 1,                  // 行号
          status: "available",     // 'available' | 'occupied' | 'reserved' | 'aisle'
          type: "regular",         // 'regular' | 'vip'
          isAisle: false,          // 是否为过道座位
          isReserved: false,       // 是否为保留座位
          occupant: null           // 占用者信息 { name, department, ... }
        },
        // ... 更多座位
      ]
    },
    // ... 更多行
  ],
  aisleConfig: {
    vertical: [],                  // 纵向过道配置
    horizontal: []                 // 横向过道配置
  }
}
```

### 过道行对象

```javascript
{
  id: "area_1_aisle_row_1704067200000",
  number: null,                    // 过道行无编号
  isAisle: true,                   // 标记为过道行
  seats: [
    {
      id: "area_1_aisle_seat_1704067200000_0",
      number: null,                // 过道座位无编号
      col: 1,                       // 列号同步
      row: null,                    // 过道行的row属性为null
      status: "aisle",
      isAisle: true,
      occupant: null
    },
    // ... 其他列的过道座位
  ]
}
```

### 过道座位对象

```javascript
{
  id: "area_1_aisle_seat_1704067200000_0",
  number: null,                    // 无编号
  col: 3,                          // 列号正常
  row: null,                       // 若为行过道，row为null；若为列过道，row为行号
  status: "aisle",
  type: "regular",
  isAisle: true,
  isReserved: false,
  occupant: null
}
```

## 流程图

### 插入行过道的流程

```
开始
  ↓
验证参数
  ↓ (有效)
确定插入位置 (above/below)
  ↓
获取列数
  ↓
创建新的过道行
  ├─ id 生成（时间戳）
  ├─ number = null
  ├─ isAisle = true
  └─ seats 初始化
  ↓
插入到rows数组
  ↓
重新编号非过道行
  ├─ 遍历所有行
  ├─ 跳过isAisle=true的行
  └─ 顺序编号1,2,3...
  ↓
更新座位row属性
  ├─ 非过道行座位 → row=行号
  └─ 过道行座位 → row=null
  ↓
保存数据 (saveData)
  ↓
更新统计 (updateStatistics)
  ↓
输出日志
  ↓
结束
```

### 插入列过道的流程

```
开始
  ↓
验证参数
  ↓ (有效)
确定插入位置 (left/right)
  ↓
对每一行 (遍历area.rows)
  ├─ 创建过道座位对象
  │  ├─ id 生成
  │  ├─ number = null
  │  ├─ col = insertPosition + 1
  │  ├─ row = row.isAisle ? null : row.number
  │  └─ isAisle = true
  ├─ 在该行插入座位
  └─ 更新该行后续座位的col属性
  ↓
保存数据 (saveData)
  ↓
更新统计 (updateStatistics)
  ↓
输出日志
  ↓
结束
```

## 性能考虑

### 时间复杂度

| 操作 | 复杂度 | 说明 |
|------|-------|------|
| insertRowAisle | O(n) | n = 当前行数，需要重新编号 |
| insertColumnAisle | O(n×m) | n×m = 总座位数，需要更新所有座位 |

### 空间复杂度

| 操作 | 复杂度 | 说明 |
|------|-------|------|
| 新建过道行 | O(m) | m = 列数 |
| 新建过道座位 | O(n) | n = 行数 |

### 优化建议

1. **批量插入过道**：如需添加多条过道，建议一次性执行，而非多次调用
   ```javascript
   // 不好的做法
   insertRowAisle(area, 2, 'below');
   insertRowAisle(area, 5, 'below');
   insertRowAisle(area, 8, 'below');
   // 每次调用都会重新编号，共3×n次操作
   
   // 更好的做法
   // 一次性创建多条过道，然后一起插入
   const aisleRows = [...];
   area.rows.splice(insertPos1, 0, aisleRows[0]);
   // 然后统一调用重新编号函数
   ```

2. **避免频繁保存**：函数内已调用saveData()，无需重复保存
   ```javascript
   // 不好的做法
   insertRowAisle(area, 2, 'below');
   saveData();  // 重复保存
   
   // 正确做法
   insertRowAisle(area, 2, 'below');
   // 函数内已保存，无需重复调用
   ```

3. **虚拟滚动**：对于超大座位区（1000+座位），考虑使用虚拟滚动优化渲染性能

## 错误处理

### 已实现的错误处理

```javascript
// 1. 参数验证
if (!area || !area.rows || rowIndex < 0 || rowIndex >= area.rows.length) {
  console.warn('[insertRowAisle] 无效的行索引或区域:', rowIndex, area);
  return;
}

if (!area || !area.rows || area.rows.length === 0) {
  console.warn('[insertColumnAisle] 无效的区域或列索引:', area);
  return;
}

// 2. 日志输出
console.log(`[insertRowAisle] 已在第 ${rowIndex} 行的${direction === 'above' ? '上方' : '下方'}插入过道行`);
```

### 建议的额外错误处理

```javascript
// 添加try-catch保护
try {
  insertRowAisle(area, rowIndex, 'below');
} catch (error) {
  console.error('插入过道行失败:', error);
  showNotification('插入过道行失败，请重试', 'error');
}

// 验证direction参数
const validDirections = ['above', 'below', 'left', 'right'];
if (!validDirections.includes(direction)) {
  console.warn('无效的direction参数:', direction);
  return;
}

// 验证最大行/列数（防止过大的布局）
const MAX_ROWS = 100;
const MAX_COLS = 50;
if (area.rows.length >= MAX_ROWS) {
  console.warn('行数已达到最大限制');
  showNotification('行数已达到最大限制（100行）', 'warning');
  return;
}
```

### 异常场景处理

| 场景 | 当前处理 | 建议处理 |
|------|---------|---------|
| area 为 null | 警告日志 + 返回 | 显示用户提示 |
| rowIndex 越界 | 警告日志 + 返回 | 显示用户提示 |
| rows 为空 | 警告日志 + 返回 | 显示用户提示 |
| 超过最大行数 | 无限制 | 验证并提示 |
| 保存失败 | 无处理 | 添加重试机制 |

## 相关函数依赖

### saveData()
- **功能**：将座位数据保存到localStorage
- **调用时机**：在insertRowAisle和insertColumnAisle执行完成后
- **重要性**：必需，否则数据不会被持久化

### updateStatistics()
- **功能**：更新座位统计信息
- **调用时机**：在insertRowAisle和insertColumnAisle执行完成后
- **重要性**：必需，否则统计信息不准确

### getRowStyle()
- **功能**：获取行的样式
- **关系**：用于U型和圆形布局的位置计算
- **影响**：插入过道后可能需要重新计算位置
