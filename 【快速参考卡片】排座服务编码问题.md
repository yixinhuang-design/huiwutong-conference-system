## 💡 常见问题

### Q1: 怎样知道文件编码是否正确?
```
A: 在VS Code中:
   1. 右下角显示 "UTF-8" 表示正确
   2. 如果显示 "GB2312" 或其他，右击修改为 "UTF-8"
   3. 保存时会提示选择编码，选择 "UTF-8"
```

### Q2: 复制的代码有很多行，怎样确保完整?
```
A: 在markdown中:
   1. 找到 ```java 开始
   2. 找到最后一行代码
   3. 找到对应的 ``` 结束
   4. 确保 import 和 package 也复制了
```

### Q3: 编译失败，显示"cannot find symbol"?
```
A: 检查:
   1. import 语句是否完整
   2. 字段名是否正确拼写
   3. 是否少了某个字段
   4. 对比markdown中的完整代码
```

### Q4: 能否用脚本自动化恢复?
```
A: 可以但复杂:
   1. 需要编写代码提取markdown脚本
   2. 手动复制更快更安全
   3. 预计时间: 手动2小时 vs 脚本4小时 (含调试)
```
