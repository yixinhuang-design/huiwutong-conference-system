## 🔐 权限控制

### 后端权限验证

```java
private Result<Void> checkSuperAdminPermission(Long userId) {
    if (!isSuperAdmin(userId)) {
        log.warn("[权限拒绝] 用户ID:{} 尝试访问超管功能但权限不足", userId);
        return Result.fail(ResultCode.FORBIDDEN, "仅超级管理员可访问租户管理功能");
    }
    return null;
}
```

**权限要求**: 仅超级管理员（super_admin）可访问

## 📞 常见问题

### Q1: 页面加载时显示"加载失败"

**解决方案**:
1. 检查后端服务是否正在运行（http://localhost:8081）
2. 检查浏览器控制台错误信息
3. 检查CORS跨域配置
4. 确认数据库连接正常

### Q2: 创建客户时出现"403 Forbidden"

**解决方案**:
1. 确认用户是超级管理员
2. 检查Authorization header中的token是否有效
3. 查看后端日志中的权限验证信息

### Q3: 修改不生效

**解决方案**:
1. 检查API返回code是否为200
2. 检查数据库连接是否正常
3. 确认修改操作返回正确的JSON响应
4. 刷新页面，检查数据库中的值

### Q4: 删除失败

**解决方案**:
1. 检查是否有级联约束限制删除
2. 确认DELETE权限配置
3. 检查数据库中是否存在该租户的关联数据
