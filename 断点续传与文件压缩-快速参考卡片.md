## 🎯 完整示例代码

### 后端初始化
```java
// Controller中
@PostMapping("/init")
public Result<UploadProgressResponse> init(
    @RequestParam String fileName,
    @RequestParam Long totalSize,
    @RequestParam String fileHash,
    @RequestParam Long scheduleId) {
    
    String uploadId = UUID.randomUUID().toString();
    UploadProgressResponse response = 
        chunkUploadService.initializeUpload(
            uploadId, fileName, totalSize, fileHash, scheduleId);
    
    return Result.success(response);
}
```

### 前端完整流程
```javascript
async function uploadWithChunks(file, scheduleId) {
    try {
        // 1. 验证
        const validation = FileUploadUtils.validateFile(file);
        if (!validation.valid) {
            throw new Error(validation.message);
        }

        // 2. 计算哈希
        const fileHash = await FileUploadUtils.calculateFileHash(file);
        
        // 3. 初始化
        const initRes = await ChunkUploadAPI.initializeUpload(
            file.name, file.size, fileHash, scheduleId);
        const uploadId = initRes.data.uploadId;
        
        // 4. 分块上传
        const chunks = FileUploadUtils.splitFileToChunks(file);
        for (let i = 0; i < chunks.length; i++) {
            const chunkHash = await FileUploadUtils.calculateChunkHash(chunks[i]);
            
            await ChunkUploadAPI.uploadChunk(
                uploadId, i, chunks.length, 
                file.size, fileHash, chunkHash,
                file.name, scheduleId, chunks[i]);
            
            console.log(`上传进度: ${((i+1)/chunks.length*100).toFixed(2)}%`);
        }
        
        // 5. 完成
        const finishRes = await ChunkUploadAPI.finishUpload(
            uploadId, scheduleId, '');
        
        console.log('上传完成:', finishRes.data);
        return finishRes.data;
        
    } catch (error) {
        console.error('上传失败:', error.message);
        throw error;
    }
}
```

## 🐛 常见问题排查

### 问题1: "分块索引无效"
**原因**: 分块总数计算错误  
**解决**: 检查 `chunkSize` 配置，确保与客户端一致（默认5MB）

### 问题2: "上传任务不存在或已过期"
**原因**: uploadId过期或错误  
**解决**: 检查是否超过24小时，重新初始化上传

### 问题3: "文件大小超过限制"
**原因**: 文件>100MB  
**解决**: 修改 `max-file-size` 配置（不建议过大）

### 问题4: "分块上传失败"
**原因**: 网络中断  
**解决**: 自动重试3次，可手动取消重新上传

### 问题5: 无法访问"/api/schedule/chunk/init"
**原因**: 编译未包含ChunkUploadController  
**解决**: 检查ChunkUploadController.java是否正确放在src下，重新编译
