## 🎯 项目概览

智能会务系统是一个基于 Spring Cloud 微服务架构的现代化会议管理平台，支持：

- 📝 **多渠道报名**：通过 OCR 识别、智能审核实现高效报名
- 📣 **精准通知**：短信、邮件、微信等多渠道推送
- 👥 **会务协同**：群组协作、任务分派、实时通讯
- 🪑 **智能排座**：AI 算法优化座位分配
- 🤖 **AI 助教**：OCR、语音识别、智能问答
- 🗺️ **位置导航**：室内外导航、AR 指引
- 📊 **数据指挥**：实时监控、预警提示、数据分析

## 📋 核心功能说明

### 1. 认证与授权 (8081)
- JWT Token 生成与验证
- 用户登录、登出、Token 刷新
- RBAC 权限控制
- 多租户支持

### 2. 通知服务 (8083)
- 多渠道通知（短信、邮件、微信）
- 个性化推送
- 催报催收
- 通知统计

### 3. 协同服务 (8084)
- 群组管理与自动创建
- WebSocket 实时群聊
- 任务分派与跟踪
- 自助资料补充

### 4. 排座服务 (8085)
- 座位导入（Excel/图片）
- 智能分配算法
- 座位平面图展示
- 动态调整与导出

### 5. AI 服务 (8086)
- OCR 身份证识别
- 语音识别
- 智能问答
- 会议纪要生成

### 6. 导航服务 (8087)
- 室外导航（高德地图集成）
- 室内定位（蓝牙信标）
- AR 实景导航
- 路径规划

### 7. 数据服务 (8088)
- 实时监控面板
- 预警提示系统
- 数据分析报告
- 签到率预测
