### 关键设计模式

#### 1. **多租户隔离模式**
```java
// 所有查询都通过此方法实现隔离
private Entity getEntityByIdAndTenant(Long id) {
    Long tenantId = TenantContextHolder.getTenantId();
    return mapper.selectOne(
        new LambdaQueryWrapper<Entity>()
            .eq(Entity::getId, id)
            .eq(Entity::getTenantId, tenantId)
    );
}
```

#### 2. **容量管理模式**
```java
// 分配前检查容量
if (entity.getAssignedCount() >= entity.getCapacity()) {
    throw new SeatingBusinessException("容量已满");
}
// 执行分配
entity.setAssignedCount(entity.getAssignedCount() + 1);
```

#### 3. **唯一性约束模式**
```java
// 检查唯一性
if (checkXxxExists(conferenceId, xxxValue, excludeId)) {
    throw new SeatingBusinessException("已存在");
}
```

#### 4. **事务一致性模式**
```java
@Transactional(rollbackFor = Exception.class)
public void updateMethod() {
    // 自动处理事务，异常时回滚
}
```
