## 页面结构标准

### 1. 页面导航栏
```html
<nav class="page-navigation">
    <div class="nav-content">
        <a href="../index.html" class="nav-back">
            <i class="fas fa-arrow-left"></i>
            <span>返回首页</span>
        </a>
        <h1 class="nav-title">页面标题</h1>
        <div class="nav-actions">
            <button class="btn-primary" @click="action">
                <i class="fas fa-plus"></i>
                操作按钮
            </button>
        </div>
    </div>
</nav>
```

**要点**：
- 使用 `.page-navigation` 包裹
- 返回链接使用 `.nav-back` 类
- 标题使用 `.nav-title` 类
- 操作按钮组使用 `.nav-actions` 类

### 2. 主内容区域
```html
<div class="page-with-nav">
    <!-- 页面内容 -->
</div>
```

**要点**：
- 使用 `.page-with-nav` 包裹所有页面内容
- 设置最大宽度：`max-width: 1400px; margin: 0 auto; padding: 0 var(--space-4);`

### 3. 页面头部
```html
<div class="page-header glass-card">
    <div>
        <h2>📋 页面标题</h2>
        <div class="conference-context">
            <span class="context-label">当前会议:</span>
            <span class="context-value">{{ conference.name }} ({{ conference.id }})</span>
            <span class="context-label">其他信息:</span>
            <span class="context-value">值</span>
        </div>
    </div>
    <div class="page-actions">
        <button class="btn-outline" @click="action">
            <i class="fas fa-arrow-left"></i>
            返回
        </button>
        <button class="btn-primary" @click="action">
            <i class="fas fa-save"></i>
            主要操作
        </button>
    </div>
</div>
```

**要点**：
- 使用 `.glass-card` 类实现玻璃态效果
- 标题使用 emoji 图标增强视觉效果
- 会议上下文使用 `.conference-context` 类
- 页面操作使用 `.page-actions` 类

## 组件样式标准

### 1. 统计卡片

#### HTML结构
```html
<div class="glass-card">
    <h3>📊 统计概览</h3>
    <div class="stats-grid">
        <div class="stat-card">
            <div class="stat-icon">
                <i class="fas fa-icon-name"></i>
            </div>
            <div class="stat-content">
                <div class="stat-value">{{ value }}</div>
                <div class="stat-label">标签文本</div>
            </div>
        </div>
        <div class="stat-card success">
            <!-- 成功状态 -->
        </div>
        <div class="stat-card warning">
            <!-- 警告状态 -->
        </div>
        <div class="stat-card info">
            <!-- 信息状态 -->
        </div>
        <div class="stat-card danger">
            <!-- 危险状态 -->
        </div>
    </div>
</div>
```

#### 状态变体
- 默认：`.stat-card`
- 成功：`.stat-card.success` - 绿色
- 警告：`.stat-card.warning` - 黄色
- 信息：`.stat-card.info` - 蓝色
- 危险：`.stat-card.danger` - 红色

### 2. 筛选器

```html
<div class="tenant-filters glass-card">
    <div class="filter-group">
        <label>状态筛选:</label>
        <select v-model="filters.status" @change="filterData">
            <option value="">全部状态</option>
            <option value="active">启用</option>
            <option value="disabled">禁用</option>
            <option value="locked">锁定</option>
        </select>
    </div>
    <div class="filter-actions">
        <button class="btn-outline" @click="resetFilters">重置</button>
    </div>
</div>
```

### 3. 表格

```html
<div class="tenant-list glass-card">
    <table class="data-table">
        <thead>
            <tr>
                <th width="50">
                    <input type="checkbox" v-model="selectAll" @change="toggleSelectAll">
                </th>
                <th>列名1</th>
                <th>列名2</th>
            </tr>
        </thead>
        <tbody>
```


#### 状态变体
- `status-active` - 启用（绿色）
- `status-disabled` - 禁用（灰色）
- `status-locked` - 锁定（红色）

## 交互设计标准

### 1. 按钮交互
```javascript
// 确认对话框
const confirmed = await this.$showConfirm(
    '确定要执行此操作吗？',
    '确认标题',
    'warning' // success, error, info, warning
);

// 提示消息
this.$showToast('操作成功', 'success');

// 加载状态
this.$showLoading('处理中...');
// ... 执行操作
this.$hideLoading();
```

### 2. 表单验证
```javascript
async validateCurrentStep() {
    let isValid = true;
    let errorMessage = '';

    if (!this.formData.name || !this.formData.name.trim()) {
        isValid = false;
        errorMessage = '请输入名称';
    }

    if (!isValid) {
        this.$showToast(errorMessage, 'error');
        return false;
    }

    return true;
}
```

### 3. 错误处理
```javascript
try {
    this.$showLoading('处理中...');
    await apiCall();
    this.$hideLoading();
    this.$showToast('操作成功', 'success');
} catch (error) {
    this.$hideLoading();
    this.$showToast('操作失败，请重试', 'error');
}
```
