## 📝 修改说明

### 修改1: 添加主席台座位区初始化 (第 4882 行)

**原始代码**:
```javascript
seatingAreas.value = [
    createSeatingArea('area_a', 'A区 (VIP区)', 'vip', 5, 10, 0),
    createSeatingArea('area_b', 'B区', 'regular', 10, 15, 0),
    createSeatingArea('area_c', 'C区', 'regular', 10, 15, 0)
];
```

**修改后**:
```javascript
seatingAreas.value = [
    createSeatingArea('area_stage', '主席台', 'stage', 1, 8, 0),  // ← 新增主席台区
    createSeatingArea('area_a', 'A区 (VIP区)', 'vip', 5, 10, 0),
    createSeatingArea('area_b', 'B区', 'regular', 10, 15, 0),
    createSeatingArea('area_c', 'C区', 'regular', 10, 15, 0)
];
```

**含义**: 
- 主席台为 1 行 8 列的座位区
- 座位类型为 'stage'，便于样式和功能区分

## 🎯 功能特点

### 主席台座位配置
- **位置**: 显示在所有座位区的最前面
- **容量**: 8 个座位（1 行 8 列）
- **类型**: 'stage' 类型，样式独特
- **功能**: 支持完整的座位分配功能

### 支持的操作
- ✅ **座位分配**: 可将参会人员分配到主席台
- ✅ **座位管理**: 支持所有批量操作（清空、交换、插入、删除等）
- ✅ **座位编号**: 自动编号为 1-8
- ✅ **日程切换**: 每个日程都有独立的主席台座位配置
- ✅ **数据保存**: 主席台座位配置随日程数据保存

### 数据持久化
- 主席台座位分配保存在 localStorage
- 每个日程的主席台座位配置独立存储
- 刷新页面后数据保持不变

## 🔧 技术细节

### 座位区创建参数
```javascript
createSeatingArea(
    id: 'area_stage',           // 区域ID
    name: '主席台',              // 显示名称
    type: 'stage',               // 区域类型
    rows: 1,                     // 1 行
    cols: 8,                     // 8 列
    angle: 0                     // 旋转角度
)
```

### 座位类型数据结构
```javascript
{
    id: 'area_stage_1',          // 座位唯一ID
    number: 1,                   // 座位编号
    row: 1,                      // 行号
    col: 1,                      // 列号
    x: 1,                        // X坐标
    y: 1,                        // Y坐标
    status: 'available',         // 可用状态
    type: 'regular',             // 座位类型
    isAisle: false,              // 非过道座位
    occupant: null,              // 占用者信息
    position: 'center'           // 位置标识
}
```

## 🚀 使用指南

### 分配主席台座位的步骤

1. **打开页面**
   - 访问排座管理页面
   - 自动加载默认日程和座位配置

2. **查看主席台**
   - 页面顶部显示"主席台"座位区
   - 显示 8 个可用座位

3. **分配座位**
   - 使用自动排座功能 → 自动分配包括主席台
   - 或手动点击座位 → 分配参会人员

4. **编辑座位**
   - 点击已分配的座位可修改
   - 使用批量操作编辑多个座位

5. **保存数据**
   - 点击"保存"按钮
   - 主席台座位配置自动保存

### 日程切换时的主席台

1. 点击"切换日程"按钮
2. 选择新日程
3. 新日程的主席台座位会单独保存
4. 切换回原日程，主席台座位配置保持不变
