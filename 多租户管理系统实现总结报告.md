## 💡 关键设计决策说明

### 1. 权限检查方式
**决策**: 在每个 API 方法中显式调用权限检查

**理由**:
- 清晰易懂，不容易漏掉
- 支持方法级别的细粒度权限
- 便于调试和日志记录

**替代方案**:
- ❌ 使用 Spring Security 注解（@PreAuthorize） - 不够灵活
- ❌ 使用 AOP 拦截器 - 隐式，难以追踪

### 2. 前后端通信格式
**决策**: 使用 JSON 格式，统一的 Result<T> 响应包装

**理由**:
- 现代标准，所有 JavaScript 框架都支持
- 自动序列化/反序列化
- 易于前端处理

**格式示例**:
```json
{
  "code": 200,
  "message": "操作成功",
  "data": { ... }
}
```

### 3. 错误处理
**决策**: 返回 HTTP 200 + 业务错误码，而不是 HTTP 400/500

**理由**:
- 便于前端统一处理
- 不破坏 HTTP 语义
- 支持国际化错误信息

### 4. 前端状态管理
**决策**: 使用 localStorage 存储 Token 和租户 ID

**理由**:
- 简单直接
- 适合小型应用
- 无需引入复杂的状态管理库

**注意**: 生产环境应考虑使用 sessionStorage 或 HttpOnly Cookie

### 5. 移动端适配
**决策**: 使用纯 HTML/CSS/JavaScript，响应式设计

**理由**:
- 无需构建工具，开箱即用
- 适合快速原型
- 易于集成到现有项目

## 📖 代码示例

### 后端：创建租户的完整流程
```java
@PostMapping("/create")
public Result<Map<String, Object>> createTenant(@RequestBody TenantCreateRequest request) {
    // 1. 获取当前用户
    Long currentUserId = TenantContextHolder.getUserId();
    log.info("[超管操作] 用户ID:{} 创建租户: {}", currentUserId, request.getTenantName());
    
    // 2. 权限检查
    Result<Void> permissionCheck = checkSuperAdminPermission(currentUserId);
    if (permissionCheck != null) {
        return (Result<Map<String, Object>>) (Object) permissionCheck;
    }
    
    // 3. 参数验证
    if (request.getTenantCode() == null || request.getTenantCode().isEmpty()) {
        return Result.fail(ResultCode.PARAM_ERROR, "租户编码不能为空");
    }
    
    // 4. 重复检查
    SysTenant existing = tenantService.getTenantByCode(request.getTenantCode());
    if (existing != null) {
        return Result.fail(ResultCode.DUPLICATE_ERROR, "租户编码已存在");
    }
    
    // 5. 创建实体
    SysTenant newTenant = new SysTenant();
    newTenant.setTenantCode(request.getTenantCode());
    newTenant.setTenantName(request.getTenantName());
    newTenant.setDescription(request.getDescription());
    newTenant.setStatus("active");
    newTenant.setCreatedBy(currentUserId);
    
    // 6. 保存数据
    SysTenant created = tenantService.createTenant(newTenant);
    
    // 7. 返回结果
    Map<String, Object> result = new HashMap<>();
    result.put("id", created.getId());
    result.put("tenantCode", created.getTenantCode());
    result.put("tenantName", created.getTenantName());
    result.put("status", created.getStatus());
    
    return Result.ok("租户创建成功", result);
}
```
