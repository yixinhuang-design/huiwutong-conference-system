## 🔧 常见问题排查

### 问题1: 编译错误 - "缺少导入"

**症状**: `error: 不存在的符号: 类 XxxException`

**解决方案**:
```bash
# 检查是否遗漏了导入语句
# 在代码顶部应该有以下导入:
import jakarta.validation.constraints.*;
import io.swagger.v3.oas.annotations.media.Schema;
import lombok.*;
```

### 问题2: 中文显示乱码

**症状**: 看到 `？` 而不是中文注释

**解决方案**:
```bash
# 验证文件编码为UTF-8
# 在VS Code中: 右下角点击编码 -> 选择UTF-8

# 或者用命令验证:
file -i DiningCreateRequest.java
# 应该显示: UTF-8
```

### 问题3: 多租户隔离错误

**症状**: `error: TenantContextHolder无法解析`

**解决方案**:
```bash
# 确保依赖已正确配置
# 在pom.xml中检查是否有:
<dependency>
    <groupId>com.conference</groupId>
    <artifactId>conference-common</artifactId>
</dependency>
```

### 问题4: MyBatis Plus 异常

**症状**: `error: BaseMapper不存在`

**解决方案**:
```xml
<!-- 在pom.xml中添加或更新: -->
<dependency>
    <groupId>com.baomidou</groupId>
    <artifactId>mybatis-plus-spring-boot3-starter</artifactId>
    <version>3.5.5</version>
</dependency>
```
