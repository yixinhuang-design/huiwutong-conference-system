## 📞 问题排查快速指南

### 问题1：API返回401

**症状**：请求被拒绝，返回401 Unauthorized

**解决步骤**：
1. 检查Token是否获取：`console.log(uni.getStorageSync('token'))`
2. 检查Token是否过期：登录重新获取
3. 检查API地址是否正确
4. 检查请求头是否包含Authorization

### 问题2：创建日程失败

**症状**：点击保存没有反应或显示错误

**解决步骤**：
1. 检查必填字段是否已填（title必填）
2. 检查时间顺序：startTime < endTime
3. 打开DevTools查看网络请求
4. 查看后端日志了解错误原因

### 问题3：日程列表为空

**症状**：打开页面看不到任何日程

**解决步骤**：
1. 检查是否加载成功：查看网络请求
2. 检查meetingId是否正确
3. 检查权限：是否有权访问该会议
4. 检查数据库：日程表中是否有数据

### 问题4：UI显示错误

**症状**：页面样式混乱或功能按钮不显示

**解决步骤**：
1. 清除浏览器缓存
2. 强制刷新页面：Ctrl+Shift+R
3. 检查浏览器控制台错误
4. 重新启动开发服务器
