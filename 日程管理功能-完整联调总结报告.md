## 📞 技术支持

### 常见问题

**Q: API返回401错误？**
```
A: 确保Token正确保存和传递
  - 检查 localStorage 中是否有 'token' 字段
  - 检查 schedule-api.js 中的 getAuthToken() 函数
```

**Q: 创建日程时显示"参数错误"？**
```
A: 检查必填字段
  - title（日程标题）必填
  - startTime < endTime（时间顺序）
  - location（地点）建议填充
```

**Q: App页面显示本地数据？**
```
A: 说明API调用失败，检查：
  1. 后端服务是否运行（http://localhost:8084）
  2. API基础URL是否正确
  3. 网络连接是否正常
  4. Token是否有效
```

**Q: 搜索功能无法工作？**
```
A: 管理后台需要在前端实现过滤逻辑
  - API返回全部数据
  - 前端执行客户端过滤
```
