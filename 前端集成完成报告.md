#### 前端关键代码

**培训创建逻辑：**
```javascript
async createTraining() {
    // 构建请求数据
    const trainingData = {
        meetingCode: 'TRAIN_' + Date.now(),
        meetingName: this.form.trainingName,
        meetingType: 'training',
        description: this.form.description,
        startTime: this.form.startDate + 'T09:00:00',
        endTime: this.form.endDate + 'T17:00:00',
        registrationStart: this.form.registrationStartTime,
        registrationEnd: this.form.registrationEndTime,
        venueName: this.form.location,
        venueAddress: this.form.location,
        coverImageUrl: this.form.coverImage,
        registrationConfig: JSON.stringify({
            mode: this.form.registrationType,
            auditRequired: this.form.auditRequired,
            schedules: this.form.schedules
        }),
        checkinConfig: JSON.stringify({
            mode: this.form.checkinMode,
            address: this.form.checkinAddress
        }),
        notificationConfig: JSON.stringify({
            channels: this.form.notificationChannels,
            template: this.form.notificationTemplate
        })
    };

    // 获取租户ID和Token
    const token = localStorage.getItem('token') || '';
    const userInfo = localStorage.getItem('userInfo');
    let tenantId = null;
    if (userInfo) {
        try {
            const parsedUserInfo = JSON.parse(userInfo);
            tenantId = parsedUserInfo.tenantId;
        } catch (e) {
            console.warn('解析用户信息失败:', e);
        }
    }

    // 调用后端 API
    const apiUrl = 'http://localhost:8084/api/meeting/create';
    const response = await fetch(apiUrl, {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json',
            'Authorization': `Bearer ${token}`,
            'X-Tenant-Id': tenantId || '2027317834622709762'
        },
        body: JSON.stringify(trainingData)
    });

    if (!response.ok) {
        throw new Error(`API 错误: ${response.status}`);
    }

    const result = await response.json();

    // 后端返回格式: { code: 0, message: "success", data: {...} }
    if (result.code === 0 && result.data) {
        localStorage.removeItem('trainingDraft');
        alert('培训创建成功！会议ID: ' + result.data.id);
        window.location.href = 'training.html';
    } else {
        throw new Error(result.message || '创建失败');
    }
}
```

### 3.2 后端租户隔离验证

**后端TenantContextHolder机制：**
```java
@Service
public class MeetingServiceImpl {
    @Override
    public Meeting createMeeting(Meeting meeting) {
        // 从上下文获取租户ID
        Long tenantId = TenantContextHolder.getTenantId();
        if (tenantId == null) {
            throw new BusinessException("未获取到租户信息");
        }
        
        // 设置租户ID
        meeting.setTenantId(tenantId);
        
        // 保存会议信息
        this.save(meeting);
        return meeting;
    }
    
    @Override
    public IPage<Meeting> getMeetingList(Page<Meeting> page, Long tenantId, ...) {
        // 所有查询都包含租户ID过滤
        LambdaQueryWrapper<Meeting> wrapper = new LambdaQueryWrapper<>();
        wrapper.eq(Meeting::getTenantId, tenantId)
               .eq(Meeting::getDeleted, 0);
        return this.page(page, wrapper);
    }
}
```

**数据库索引优化：**

CREATE UNIQUE INDEX uk_tenant_code ON conf_meeting(tenant_id, meeting_code, deleted);

## 六、已知问题和限制

### 6.1 当前限制

1. **端口绑定问题**
   - **问题：** 后端服务在有360安全卫士的环境下无法绑定端口
   - **影响：** 需要在干净的环境中测试
   - **解决方案：** 临时禁用360或使用其他测试环境

2. **工作人员添加API未实现**
   - **问题：** `POST /api/meeting/{id}/staff` 端点未实现
   - **影响：** 管理后台创建会议时工作人员添加会失败
   - **解决方案：** 前端已添加错误捕获，不影响主流程

3. **前端未实现完整的表单验证**
   - **问题：** 部分必填字段验证逻辑不完整
   - **影响：** 可能提交不完整的数据到后端
   - **解决方案：** 依赖后端验证，前端逐步完善

### 6.2 待优化项

1. **Token管理**
   - 当前使用localStorage存储，存在XSS风险
   - 建议：使用HttpOnly Cookie + CSRF Token

2. **错误提示用户体验**
   - 当前使用alert()原生弹窗
   - 建议：使用Toast或Modal组件

3. **图片上传**
   - 当前使用Base64编码
   - 建议：实现文件上传到OSS

4. **分页功能**
   - 学员端列表未实现分页
   - 建议：添加滚动加载或分页组件
