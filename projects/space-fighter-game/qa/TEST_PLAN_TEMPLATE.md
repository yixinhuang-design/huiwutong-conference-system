## 3. 性能测试标准

### 3.1 帧率要求
- PC端：稳定60 FPS
- 移动端：稳定≥30 FPS

### 3.2 内存要求
- 内存占用 < 100MB
- 无明显内存泄漏

### 3.3 加载时间
- 首次加载 < 2秒
- 游戏启动 < 1秒
