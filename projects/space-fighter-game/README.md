## 项目架构

```
space-fighter-game/
├── index.html           # 游戏主页面
├── css/
│   └── style.css       # 样式文件
├── js/
│   ├── game.js         # 游戏核心逻辑
│   ├── player.js       # 玩家类
│   ├── enemy.js        # 敌人类
│   ├── bullet.js       # 子弹类
│   ├── particle.js     # 粒子系统
│   └── utils.js        # 工具函数
├── assets/
│   ├── images/         # 图片资源
│   └── sounds/         # 音效资源
└── docs/               # 文档
```

## 协作流程

1. **产品经理** → 定义需求 → `product/PRD.md`
2. **技术经理** → 架构设计 → `tech/ARCHITECTURE.md`
3. **技术经理** → 编码实现 → `js/` 各模块
4. **测试经理** → 测试验证 → `qa/TEST_REPORT.md`
5. **运营经理** → 发布准备 → `ops/RELEASE_CHECKLIST.md`
