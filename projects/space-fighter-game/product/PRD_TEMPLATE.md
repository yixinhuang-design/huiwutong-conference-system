## 1. 产品概述

### 1.1 项目背景
董事长指示开发一个太空战机单页面小游戏，用于验证AI团队协作能力。

### 1.2 目标用户
- 喜欢快节奏射击游戏的玩家
- 碎片时间娱乐的用户
- 科幻风格爱好者

### 1.3 核心价值
- **爽快：** 流畅的射击体验和视觉冲击
- **简单：** 易上手，难精通
- **酷炫：** 霓虹科幻风格

---

## 2. 功能需求

### 2.1 核心玩法（P0 - 必须实现）

#### 玩家控制
- [ ] PC端：WASD或方向键移动，空格键射击
- [ ] 移动端：触摸拖动移动，自动射击
- [ ] 移动边界限制

#### 射击系统
- [ ] 自动发射子弹
- [ ] 子弹飞行速度和频率可调
- [ ] 击中敌人产生爆炸特效

#### 敌人系统
- [ ] 普通敌机：直线飞行
- [ ] 快速敌机：追踪玩家
- [ ] Boss敌机：高血量，发射弹幕

#### 碰撞检测
- [ ] 子弹击中敌人 → 敌人扣血/死亡
- [ ] 敌人撞击玩家 → 玩家扣血
- [ ] 精确的碰撞盒计算

#### 游戏循环
- [ ] 开始界面：标题 + 开始按钮
- [ ] 游戏进行：实时战斗
- [ ] 游戏结束：分数显示 + 重新开始

### 2.2 进阶功能（P1 - 应该实现）

#### 道具系统
- [ ] 武器升级道具（双发、三发、散射）
- [ ] 护盾道具（无敌3秒）
- [ ] 生命恢复道具

#### 关卡系统
- [ ] 无限模式：难度逐渐增加
- [ ] 关卡模式：3-5个预设关卡
- [ ] 每5波出现一个Boss

#### 计分系统
- [ ] 实时分数显示
- [ ] 击杀不同敌人得分不同
- [ ] 连击加成
- [ ] 本地最高分记录

### 2.3 增强功能（P2 - 可以延后）

- [ ] 多种玩家飞船可选
- [ ] 成就系统
- [ ] 音效和背景音乐
- [ ] 排行榜（需要后端）
- [ ] 多人对战模式

## 3. 用户体验设计

### 3.1 视觉风格
- **主题：** 霓虹科幻
- **配色：** 深空黑背景 + 霓虹蓝/粉/紫
- **特效：** 粒子爆炸、光晕、拖尾

### 3.2 交互流程
```
打开游戏
  ↓
开始界面（标题 + START按钮）
  ↓
游戏进行（战斗画面 + HUD）
  ↓
玩家死亡
  ↓
结束界面（最终分数 + 最高分 + RETRY按钮）
  ↓
重新开始 / 关闭游戏
```

### 3.3 界面布局
- **顶部：** 分数、生命值、关卡
- **中央：** 游戏主画面
- **底部：** 操作提示（PC/移动端）

---

## 4. 技术需求

### 4.1 性能要求
- 60 FPS流畅运行
- 支持主流现代浏览器
- 移动端响应式（320px - 4K）

### 4.2 技术栈
- HTML5 Canvas
- 原生JavaScript（无框架依赖）
- CSS3动画
