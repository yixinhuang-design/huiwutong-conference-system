## 📞 问题排查

### 问题 1: CSS 文件未加载
**症状**: 弹窗没有样式，显示为空白  
**解决**:
1. 检查浏览器控制台是否有 404 错误
2. 验证文件路径: `../css/seating-function-modal.css`
3. 刷新页面，清除浏览器缓存 (Ctrl+Shift+Delete)

### 问题 2: 样式冲突
**症状**: 某些样式被覆盖，颜色或大小不对  
**解决**:
1. 检查是否有其他 CSS 文件与冲突
2. 在浏览器 DevTools 中检查样式层级
3. 使用 `!important` 临时修复 (后期移除)

### 问题 3: 响应式不工作
**症状**: 在手机上显示不正确  
**解决**:
1. 检查 HTML head 中是否有 viewport meta 标签
2. 按 F12 打开响应式设计模式
3. 调整浏览器宽度，测试各断点

### 问题 4: 深色模式无效
**症状**: 深色模式下颜色不对  
**解决**:
1. 系统设置中切换深色模式
2. 或在浏览器 DevTools 中强制深色模式
3. 检查 `@media (prefers-color-scheme: dark)` 规则
