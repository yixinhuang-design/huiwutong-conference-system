# 流程描述汇总 - 第二部分（流程07-12）- 提炼版

## 流程07：催报催收流程

开始

  ↓

1. 管理员进入催报催收页面（仅针对 限定式报名 ）

  ↓

2. 查询缺报和未报名单位（针对 限定式报名）

  ├─ 未报缺报单位列表（单位名称、应报名人数、已报名人数、缺报名人数、报名率、单位管理员姓名、联系电话）

  ├─ 底部一键全选


3. 选择催报对象

  ├─ 选择未报名单位（已报名数为0）

  ├─ 选择缺报名单位（已报名数不为0，但小于应报名人数）

  └─ 自定义查询（按名称查询某个单位）

  ↓

4. 选择催报模板

  ├─ 温和催报模板（友好提醒，语气温和）

  ├─ 紧急催报模板（强调重要性，语气紧急）

  └─ 最后催报模板（最后期限提醒）

  ↓

5. 配置催报渠道

  ├─ 短信

  ├─ 邮件

  ├─ 微信

  └─ APP/小程序推送

  ↓

6. 设置催报时间

  ├─ 立即发送

  ├─ 定时发送（设置具体时间）

  └─ 多轮催报计划（设置催报轮次、催报间隔、每轮使用的模板和渠道）

  ↓

7. 发送催报通知

  ├─ 批量发送

  ├─ 后台处理

  └─ 发送状态跟踪

  ↓

8. 查看催报效果

  ├─ 催报后报名数统计

  ├─ 催报转化率（催报后报名数/催报发送数）

  └─ 催报记录查询（管理员姓名、轮次、渠道、发送时间、报名状态）

  ↓

9. 多轮催报

  ├─ 设置催报轮次（1-5轮）

  ├─ 设置催报间隔（小时）

  └─ 自动执行多轮催报（已报名人员自动跳过后续催报）

  ↓

结束

---

## 流程08：自动组群管理 流程

开始

  ↓

1. 会议人员审核之后并且在手册生成完毕后（触发自动组群，自动组建1个全体人员群、1个工作人员群，如参会人员进行了分组，则自动建好各分组群）

  ↓

2. 进入系统自动创建的本次会议的群列表

  ├─ 显示群名称、创建时间、群内人数、消息数、群管理员姓名

  ├─ 修改群组设置（群组名称、群管理员增删－工作人员自动为群管理员、设置群公告）

  ├─ 添加/编辑群任务

  ├─ 发送欢迎消息（会议信息、群组规则）

  ↓

3. 群组管理

  ├─ 查看群组列表（群组名称、成员数、消息数）

  ├─ 管理群组成员（添加成员、移除成员、设置成员角色）

  ├─ 发送群组消息（文本消息、图片/文件、通知消息）

  └─ 查看消息历史（搜索消息、导出消息记录）

  ↓

结束

---

## 流程09：任务分派流程

开始

  ↓

1. 管理员进入任务管理页面

  ↓

2. 创建任务/编辑/删除

  ├─ 填写任务基本信息（任务标题、任务描述、任务类型、任务优先级、完成截止时间、上传任务提示图片/附件资料）

  ├─ 选择任务执行者（选择任务负责人、主执行人、副执行人（均可设多个））

  ├─ 配置任务流程（选择任务流程模板、配置任务节点、设置审批流程、设置任务状态流转规则）

  └─ 分派任务（发送任务通知、在群组内分发任务）

  ↓

3. 任务执行

  ├─ 执行者接收任务

  ├─ 查看任务详情（任务标题、任务描述、任务类型、优先级、完成截止时间、任务图片/附件资料）

  ├─ 执行任务（视任务类型进行相应操作，或上传任务完成的现场图片或描述）

  └─ 自动更新任务状态（进度百分比、完成状态执行记录）

  ↓

4. 任务进度跟踪

  ├─ 查看任务进度

  ├─ 查看任务执行记录（执行者、执行时间、执行内容）

  ├─ 任务进度通知/任务完成通知

  └─ 任务超时提醒

  ↓

5. 任务类型

  ├─ 签到任务（位置+点击方式/或实时拍照视频方式，或打印表格签到，接龙式确认）

  ├─ 就寝任务（主执行人为每个学员，位置+点击方式/或实时拍照视频方式）

  ├─ 问卷任务（请学员填写问卷调查表）

  ├─ 评价任务（请学员对课程或老师进行评价打分或满意度测评）

  ├─ 现场任务（请工作人员或学员去现场完成某几项任务，并拍照描述反馈完成状态）

  ├─ 征集任务（请某些人或全体学员反馈，是否用餐、出行方式、是否带车等等，系统能自动汇总分类统计）

  └─ 其他任务（保留可增类型）

  ↓

6. 任务统计（按 客户－会议 来区分，不将不同会议的任务放在一起展示）

  ├─ 查看任务列表（任务名称、任务完成率、任务完成进度、任务完成状态、任务完成现场图、未完成人员、一键催促

  ├─ 查看任务统计报表（任务总数、已完成数、进行中数、已超时数）

  └─ 导出任务数据

  ↓

结束

---

## 流程11：智能排座流程

开始

  ↓

1. 管理员进入排座管理页面

  ↓

2. 设置会场信息（支持多种会议类型）

  ├─ 设置会场名称

  ├─ 选择会议室类型
  │   ├─ 普通式会议（教室式座位布局）
  │   ├─ U型会议（U型桌位布局）
  │   └─ 圆桌会议（圆桌式布局）
  
  ├─ 创建座位区
  │   ├─ 配置座位区名称
  │   ├─ 配置座位区类型
  │   ├─ 配置布局大小（行数、列数、座位间距）
  │   ├─ 配置布局角度（支持任意角度旋转）
  │   ├─ 配置排座方式（横向/纵向/自定义）
  │   ├─ 配置座位数量
  │   ├─ 设置固定座位（指定特定座位不可分配）
  │   └─ 设置过道位置（标记过道区域）

  └─ 设置特殊座位标识
      ├─ VIP座位
      ├─ 专用座位
      └─ 其他特殊需求座位
  
  └─ 保存座位区配置

  ↓


5. 座位区排座（快速排座）

  ├─ 方式一：导入Excel名单快速排座
  │   ├─ 上传参会人员Excel名单（姓名对应在行号、列号）
  │   ├─ 系统自动按配置的排座方式分配
  │   └─ 支持EXCEL数据原样导入排座（名单所见即所得）
  
  ├─ 方式二：按规则自动排座
  │   ├─ 设置排座规则（按单位、按职级、按名册顺序等）
  │   ├─ 设置各规则的优先级
  │   └─ 系统自动按优先级的规则分配座位
  
  ├─ 方式三：按分区自动排座
      ├─ 将座位划分为多个小区域
      ├─ 按序号或分组分配到不同区域
      ├─ 系统自动按优先级的规则在各区域内分配座位
  
  └─ 预览排座结果
      ├─ 显示分配结果（姓名、座位号、座位位置）
      ├─ 显示空座位统计
      └─ 显示特殊座位分配情况

  ↓

6. 全局座位任意调整

  ├─ 手动调整座位
  │   ├─ 选择参会人员
  │   ├─ 选择目标座位
  │   ├─ 拖拽调整座位位置
  │   └─ 实时更新座位分配
  │   └─支持插入或删除，支持插入或删除后其余自动调整，或插入删除后其余保持不变
  
  ├─ 批量调整座位
  │   ├─ 选择多个参会人员
  │   ├─ 批量移动到指定区域
  │   └─ 批量交换座位
  
  ├─ 剩余座位快速填充
  │   ├─ 识别未分配座位
  │   ├─ 选择待分配人员
  │   └─ 一键填充剩余座位
  
  └─ 支持EXCEL名单原样排座
      ├─ 导入包含座位号的Excel名单
      ├─ 系统按Excel中的座位号原样分配
      └─ 确保分配结果与Excel完全一致

  ↓

7. 查看座位安排图

  ├─ 显示座位平面图
  │   ├─ 可视化座位布局
  │   ├─ 显示座位编号
  │   └─ 显示座位状态（已分配/未分配/固定）
  
  ├─ 显示座位分配情况
  │   ├─ 已分配座位高亮显示
  │   ├─ 显示参会者姓名
  │   └─ 显示座位位置信息
  
  ├─ 显示空座位
  │   ├─ 未分配座位标识
  │   └─ 空座位统计
  
  └─ 显示特殊座位标识
      ├─ VIP座位标识
      ├─ 专用座位标识
      └─ 固定座位标识

  ↓

8. 生成全局导示图（用于LED大屏显示）

  ├─ 生成座位区全局导示图
  │   ├─ 系统自动生成完整座位布局图
  │   ├─ 支持显示座位图标
  │   ├─ 支持显示参会者姓名
  │   ├─ 支持显示座位编号
  │   └─ 支持自定义显示内容
  
  ├─ 配置导示图样式
  │   ├─ 设置显示比例
  │   ├─ 设置字体大小
  │   └─ 设置颜色方案
  
  └─ 导出导示图
      ├─ 导出高清图片格式
      ├─ 用于签到处LED屏幕显示
      └─ 让参会者快速知悉座位信息

  ↓

9. 导出座位表

  ├─ 导出Excel格式
  │   ├─ 包含（姓名、座位号、座位位置）
  
  ├─ 导出PDF格式
  │   ├─ 生成标准格式座位表
  │   └─ 支持打印
  
  └─ 分享座位表
      ├─ 生成分享链接
      ├─ 生成二维码
      └─ 支持在线查看

  ↓

10. 手机查座功能（参会者自助查询）

  ├─ 生成会议专属二维码
  │   ├─ 每个会议生成单独的二维码
  │   ├─ 可打印到会议物料
  │   └─ 可放置在现场导示牌
  
  ├─ 参会者扫码查座
  │   ├─ 手机扫描二维码
  │   ├─ 输入姓名或手机号查询
  │   └─ 系统快速定位座位
  
  ├─ 查座结果展示
  │   ├─ 座位信息高亮显示
  │   ├─ 座位位置定位显示（在座位布局图中标记）
  │   ├─ 显示座位编号
  │   ├─ 显示座位区域
  │   └─ 显示周围座位信息
  
  └─ 支持参会企业广告投放
      ├─ 在查座页面显示企业广告
      ├─ 支持图片广告
      ├─ 支持文字广告
      └─ 可配置广告位置和样式

  ↓

11. 创建辅助安排信息

  ├─ 创建乘车安排
  │   ├─ 车辆类型（小车/中巴/大巴）
  │   ├─ 车号
  │   ├─ 出发地
  │   ├─ 目的地
  │   └─ 出发时间
  
  ├─ 创建住宿安排
  │   ├─ 酒店名称
  │   ├─ 房间号
  │   ├─ 房间类型
  │   ├─ 入住时间
  │   └─ 退房时间
  
  ├─ 创建餐饮安排
  │   ├─ 用餐时间
  │   ├─ 用餐地点
  │   └─ 用餐类型
  
  └─ 创建其他辅助安排
      ├─ 接送站安排
      ├─ 参观安排
      └─ 讨论安排

  ↓

12. 分配辅助安排

  ├─ 智能分配乘车
  │   ├─ 根据车辆类型自动安排人数
  │   ├─ 生成分配结果（姓名、车号、出发地、目的地、出发时间）
  │   └─ 支持手动调整
  
  ├─ 智能分配住宿
  │   ├─ 根据单住/双住等需求自动分配房间
  │   ├─ 生成分配结果（姓名、房间号、入住时间、退房时间）
  │   └─ 支持手动调整
  
  ├─ 智能讨论安排（也可在添加日程处链接进去设置）
  │   ├─ 根据讨论室数量及各自人数容量、学员分组规则自动分配讨论室号
  │   ├─ 生成分配结果（姓名、单位及职务、讨论室号、讨论室位置、讨论开始时间）
  │   └─ 

  ├─ 自动关联参会者信息，供系统推送通知、本人APP查询时提供 订制化显示
  │   ├─ 座位信息
  │   ├─ 车号信息
  │   ├─ 住宿信息
  │   ├─ 讨论信息
  │   └─ 其他安排信息


  ↓

13. 发送座位和辅助安排APP推送通知

  ├─ 发送座位信息给参会者，精准的点对点通知
  │   ├─ 本人姓名、对应座位号、开始时间、地点
  │   ├─ 支持查看全局座位图

  
  ├─ 发送车号信息给参会者，精准的点对点通知
  │   ├─ 本人姓名、车号、出发地、目的地、出发时间
  │   ├─ 支持查看本车号全部人员表
  
  ├─ 发送住宿信息给参会者
  │   ├─ 酒店名称、本人姓名、房间号、入住时间、退房时间

  ├─ 发送讨论信息给参会者
  │   ├─ 本人姓名、单位及职务、讨论室号、讨论室位置、讨论开始时间
  │   ├─ 支持查看本讨论室全部人员表
  
  └─ 支持非注册用户接收短信通知
      ├─ 二维码（扫码查询）
      └─ 短信/邮件通知（通过链接查询相关信息）

  ↓

结束

## 流程12：中途座位调整流程

开始

  ↓

1. 管理员进入中途座位调整页面

  ↓

2. 选择调整方式

  ├─ 手动调整单个座位

  ├─ 批量调整座位

  ├─ 临时增减座位

  └─ 自动重新分配

  ↓

3. 执行座位调整

  ├─ 选择调整对象（姓名）

  ├─ 选择目标座位（座位号、座位位置）

  ├─ 填写变更原因

  ├─ 确认调整

  └─ 更新座位分配

  ↓

4. 座位变更通知

  ├─ 发送变更通知给相关人员（姓名、原座位号、新座位号、变更原因）

  └─ 通知渠道：短信/APP/微信/小程序

  ↓

5. 座位变更记录

  ├─ 记录座位变更（姓名、原座位号、新座位号、变更原因、变更时间、操作人）


  ↓

6. 查看变更历史

  ├─ 查看座位变更记录（时间、人员、原座位、新座位、原因）

  └─ 可以撤销变更（恢复到变更前的座位，发送撤销通知）

  ↓

结束
