## 📝 使用指南

### 基本工作流程

#### 1. 创建座位布局
```
1. 点击"添加区域"按钮
2. 配置区域信息（名称、类型、行列数）
3. 配置角度、过道、固定座位
4. 查看实时预览
5. 保存配置
```

#### 2. 导入参会人员
```
方式一：从报名系统导入
1. 在报名管理页面审核通过报名
2. 点击"从报名系统导入"

方式二：Excel导入
1. 准备Excel文件（支持两种格式）
2. 选择导入模式
3. 点击"导入Excel"
```

#### 3. 执行智能排座
```
1. 选择排座算法（智能推荐/按部门/按职级/按分区）
2. 点击"执行排座"
3. 系统自动分配座位
```

#### 4. 手动调整（可选）
```
1. 点击选择要移动的参会者
2. 点击目标座位
3. 确认移动或交换
```

#### 5. 辅助安排（可选）
```
1. 添加车辆/房间/讨论室
2. 点击"智能分配所有"
3. 查看分配结果
```

#### 6. 导出和分享
```
1. 生成LED导示图（用于现场大屏）
2. 导出座位表（Excel/PDF）
3. 生成查座二维码
4. 分享查座链接
```
