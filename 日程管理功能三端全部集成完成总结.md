   formatScheduleData(data) {

## 🎓 技术亮点

### 1. **多租户设计**
```java
// 所有查询自动过滤租户数据
@Select("SELECT * FROM conf_schedule WHERE tenant_id = #{tenantId}")
List<Schedule> selectByTenantId(String tenantId);
```

### 2. **状态机管理**
```javascript
// 状态转换规则
0 → 1: 草稿 → 发布
1 → 2: 发布 → 进行中（自动）
2 → 3: 进行中 → 完成（自动）
* → 4: 任何状态 → 取消
```

### 3. **错误处理分层**
```
Controller(400/401/403/500)
↓
Service(业务异常)
↓
Mapper(数据库异常)
↓
最终用户提示
```

### 4. **前端降级策略**
```
API 加载失败 → 使用模拟数据 → 允许继续操作（只读）
复制 API 失败 → 降级到本地复制 → 用户手动修改
```

### 5. **实时状态同步**
```
操作 → API 调用 → 响应数据 → 更新本地列表 → UI 自动刷新
全程追踪，确保前后端数据一致性
```
