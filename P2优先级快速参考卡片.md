## 关键特性检查表

### ✅ 多租户隔离
- [ ] 所有查询都过滤 `tenant_id`
- [ ] 使用 `TenantContextHolder.getTenantId()` 获取当前租户
- [ ] 更新/删除前验证所有权

### ✅ 容量管理
- [ ] 分配前检查 `assignedCount < capacity`
- [ ] 取消分配时递减 `assignedCount`
- [ ] 删除前检查 `assignedCount == 0`

### ✅ 唯一性约束
- [ ] 房间号在会议内唯一
- [ ] 餐厅名称在会议内唯一
- [ ] 车牌号在会议内唯一

### ✅ 异常处理

- [ ] 业务规则验证

### ✅ 事务管理

## DTO 字段速查

### Accommodation
```
Create:  conferenceId*, roomNumber*, roomType, capacity*, address, phone
Update:  roomNumber, roomType, capacity, address, phone (都可选)
Response: id, conferenceId, roomNumber, roomType, capacity, assignedCount, 
          availableBeds, address, phone, createdAt, updatedAt
```

### Dining
```
Create:  conferenceId*, restaurantName*, mealType, capacity*, location, mealTime, remarks
Update:  restaurantName, mealType, capacity, location, mealTime, remarks (都可选)
Response: id, conferenceId, restaurantName, mealType, capacity, assignedCount,
          availableSeats, location, mealTime, remarks, createdAt, updatedAt
```

### Transport
```
Create:  conferenceId*, licensePlate*, vehicleType, capacity*, departure, destination, 
         departureTime, driver, contactPhone, remarks
Update:  licensePlate, vehicleType, capacity, departure, destination, departureTime,
         driver, contactPhone, remarks (都可选)
Response: id, conferenceId, licensePlate, vehicleType, capacity, assignedCount,
          availableSeats, departure, destination, departureTime, driver, 
          contactPhone, remarks, createdAt, updatedAt
```

## 故障排除

| 问题 | 原因 | 解决 |
|------|------|------|
| 409 Conflict | 房间号/餐厅名/车牌号重复 | 检查唯一性约束 |
| 400 Bad Request | 容量已满 | 选择其他有空余的资源 |
| 403 Forbidden | 无权访问 | 检查租户ID隔离 |
| 404 Not Found | 资源不存在 | 检查ID是否正确 |
| 422 Unprocessable | 有人员分配 | 先取消所有分配再删除 |
