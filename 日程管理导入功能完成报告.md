| 格式 | 扩展名 | 说明 |

| Excel | .xlsx, .xls | 推荐格式，支持复杂数据结构 |

**导入能力**：

- 📝 详细的导入结果报告

## 📝 文件格式规范

### TXT格式示例

```
# 日程导入模板
# 格式：日期|时间|内容|地点|主持人|主讲人

2026-06-15|09:00-10:30|开幕式|主会场A厅|张三|李四
2026-06-15|10:45-12:00|主题演讲|主会场A厅|张三|王五
2026-06-15|14:00-15:30|分组讨论|分会场B1|赵六|钱七
```

**格式说明**：
- 每行一个日程
- 字段用竖线(|)分隔
- 支持#开头的注释行
- 空行自动跳过
- 时间可以是单点（09:00）或范围（09:00-10:30）

### Excel格式示例

| 日期 | 时间 | 内容 | 地点 | 主持人 | 主讲人 |
|-----|------|------|------|--------|--------|
| 2026-06-15 | 09:00-10:30 | 开幕式 | 主会场A厅 | 张三 | 李四 |
| 2026-06-15 | 10:45-12:00 | 主题演讲 | 主会场A厅 | 张三 | 王五 |

**格式说明**：
- 第一行为表头（必须包含列名）
- 日期格式：YYYY-MM-DD
- 时间格式：HH:MM-HH:MM 或 HH:MM
- 支持Excel日期数值格式

## 🔧 技术实现

### 核心方法

| 方法 | 功能 | 说明 |
|-----|------|------|
| `showImportModal()` | 显示导入弹窗 | 初始化导入状态 |
| `handleImportFile()` | 处理文件选择 | 根据格式调用解析器 |
| `parseTxtFile()` | 解析TXT文件 | FileReader + 字符串分割 |
| `parseExcelFile()` | 解析Excel文件 | SheetJS库 |
| `validateImportData()` | 验证数据 | 检查格式和必填项 |
| `calculateImportStats()` | 统计数据 | 计算有效/无效/重复 |
| `executeImport()` | 执行导入 | 批量创建日程 |
| `downloadTemplate()` | 下载模板 | 生成TXT模板文件 |

### 依赖库

```html
<!-- SheetJS - 用于解析Excel文件 -->
<script src="https://cdn.bootcdn.net/ajax/libs/xlsx/0.18.5/xlsx.full.min.js"></script>
```

### 数据流程

```
用户选择文件
    ↓
handleImportFile()
    ↓
判断文件类型
    ├─→ .txt → parseTxtFile()
    └─→ .xlsx/.xls → parseExcelFile()
    ↓
validateImportData() - 验证每条数据
    ↓
calculateImportStats() - 统计有效/无效/重复
    ↓
显示预览表格
    ↓
用户点击"导入"
    ↓
executeImport() - 批量创建
    ├─ 循环每条有效数据
    ├─ 调用 ScheduleAPI.createSchedule()
    ├─ 统计成功/失败数量
    └─ 更新日程列表
    ↓
显示导入结果
```

## 📊 导入统计

### 统计指标

| 指标 | 说明 | 示例 |
|-----|------|------|
| 总计 | 文件中的总条数 | 10条 |
| 有效 | 验证通过的条数 | 8条 |
| 无效 | 验证失败的条数 | 1条 |
| 重复 | 与已有日程重复 | 1条 |

### 重复判断规则

```
重复 = 标题相同 + 日期相同 + 开始时间相同
```

**示例**：
- 已存在：`开幕式 | 2026-06-15 | 09:00`
- 导入数据：`开幕式 | 2026-06-15 | 09:00`
- 判断结果：✗ 重复

## 📚 使用指南

### 快速开始（3步）

1. **下载模板**
   - 点击"导入"按钮
   - 点击"导入模板"链接
   - 保存模板文件到本地

2. **编辑数据**
   - 用Excel/文本编辑器打开模板
   - 按照格式填写日程信息
   - 保存文件

3. **导入数据**
   - 点击"导入"按钮
   - 选择编辑好的文件
   - 查看预览和统计
   - 点击"导入"完成

### 注意事项

⚠️ **日期格式**：必须是 `YYYY-MM-DD` 格式（如：2026-06-15）

⚠️ **时间格式**：必须是 `HH:MM` 或 `HH:MM-HH:MM` 格式（如：09:00 或 09:00-10:30）

⚠️ **必填字段**：日期、时间、内容为必填项

⚠️ **文件编码**：TXT文件必须使用 UTF-8 编码

⚠️ **数据重复**：相同标题+日期+时间的日程会被跳过
