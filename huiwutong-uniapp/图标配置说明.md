# Font Awesome 图标配置说明

## 问题诊断

如果您在页面上看到占位符而不是图标，可能是以下原因：

1. **网络问题** - CDN 无法访问
2. **字体文件未加载** - 字体文件下载失败
3. **UniApp 兼容性** - 某些平台不支持 webfont

## 解决方案

### 方案1：使用本地图标组件（推荐）

对于 UniApp 项目，建议使用 iconfont 或组件化图标：

```vue
<!-- 创建图标组件 -->
<template>
  <text :class="iconClass" :style="{ fontSize: size, color: color }"></text>
</template>

<script>
export default {
  props: {
    name: String,
    size: {
      type: String,
      default: '48rpx'
    },
    color: {
      type: String,
      default: '#333'
    }
  },
  computed: {
    iconClass() {
      return `fa fa-${this.name}`
    }
  }
}
</script>
```

### 方案2：下载字体文件到本地

1. 下载 Font Awesome 字体文件：
   - 访问 https://fontawesome.com/download
   - 下载 webfonts 版本
   - 将字体文件放到 `static/fonts/` 目录

2. 在 `styles/font-awesome.scss` 中修改路径：

```scss
@font-face {
  font-family: 'Font Awesome 6 Free';
  src: url('/static/fonts/fa-solid-900.woff2') format('woff2');
}
```

### 方案3：使用 Unicode 字符（临时方案）

如果字体无法加载，可以使用 Unicode 字符直接显示：

```javascript
// 在 data 中定义
const iconMap = {
  'fa-check': '\f00c',
  'fa-times': '\f00d',
  'fa-home': '\f015',
  'fa-user': '\f007'
}

// 在模板中使用
<text style="font-family: 'Font Awesome 6 Free'; font-weight: 900">
  {{ iconMap['fa-check'] }}
</text>
```

### 方案4：使用 iconfont（推荐用于生产环境）

1. 访问 https://www.iconfont.cn/
2. 创建项目并添加图标
3. 下载字体文件
4. 在项目中引入

```scss
@font-face {
  font-family: 'iconfont';
  src: url('/static/fonts/iconfont.ttf') format('truetype');
}

.iconfont {
  font-family: 'iconfont' !important;
}
```

## 测试步骤

1. 访问图标测试页面：`/pages/common/icon-test`
2. 检查图标是否正常显示
3. 如果不显示，使用开发者工具检查：
   - 字体文件是否加载（Network 面板）
   - CSS 类名是否正确（Elements 面板）
   - 是否有网络错误（Console 面板）

## 常见问题

### Q1: H5 端正常，小程序端不显示

**原因**: 小程序不支持外部字体链接

**解决**: 使用本地字体文件或 iconfont

### Q2: 部分图标显示，部分不显示

**原因**: 字体文件不完整或版本不匹配

**解决**: 确保使用 Font Awesome 6.x 完整版

### Q3: 图标显示为方块

**原因**: 字体文件未加载或类名错误

**解决**: 检查网络连接和类名拼写

## 快速修复

如果急需显示图标，可以使用 emoji 或图片替代：

```vue
<!-- 使用 emoji -->
<text>✓</text>

<!-- 使用图片 -->
<image src="/static/icons/check.png" style="width: 48rpx; height: 48rpx;"></image>
```
