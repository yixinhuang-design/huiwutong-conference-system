#### 1. 平台适配工具 (`utils/platform.js`)

完整的多平台适配工具函数库，包含以下功能：

**基础平台检测**
- `getPlatformInfo()` - 获取当前平台信息（H5/小程序/App）
- `getSystemInfo()` - 获取系统详细信息
- `getStatusBarHeight()` - 获取状态栏高度
- `getSafeArea()` - 获取安全区域
- `isNotchScreen()` - 判断是否为刘海屏

**导航与路由**
- `navigateTo(url)` - 平台特定的页面跳转
- `navigateBack(delta)` - 平台特定的页面返回

**剪贴板与外部链接**
- `setClipboard(data)` - 复制到剪贴板（H5/小程序/App不同实现）
- `openExternalLink(url)` - 打开外部链接
- `makePhoneCall(phoneNumber)` - 拨打电话

**媒体与文件**
- `chooseImage(options)` - 选择图片
- `scanCode()` - 扫码功能
- `saveImageToPhotos(filePath)` - 保存图片到相册

**位置与地图**
- `getLocation()` - 获取位置信息
- `openLocation(lat, lng, name, address)` - 打开地图

**支付功能**
- `requestPayment(orderInfo)` - 跨平台支付

**分享功能**
- `share(options)` - 平台特定的分享实现

**其他功能**
- `getNetworkType()` - 获取网络状态
- `vibrate(short)` - 震动反馈
- `setNavigationBarTitle(title)` - 设置导航栏标题
- `setNavigationBarColor(frontColor, bgColor)` - 设置导航栏颜色

#### 2. 平台适配 Mixin (`mixins/platform.js`)

提供响应式的平台信息，可直接在组件中使用：

**数据属性**
- `platform` - 当前平台标识
- `isH5` / `isMP` / `isApp` - 平台判断
- `isIOS` / `isAndroid` - 系统判断
- `statusBarHeight` - 状态栏高度
- `safeArea` / `safeAreaTop` / `safeAreaBottom` - 安全区域信息
- `customNavHeight` - 自定义导航栏高度
- `isSmallScreen` / `isLargeScreen` - 屏幕尺寸判断

**方法**
- `initPlatformInfo()` - 初始化平台信息
- `getPlatformClass()` - 获取平台相关样式类

## 📝 注意事项

### 1. 条件编译使用
```javascript
// #ifdef H5
console.log('只在H5平台执行')
// #endif

// #ifdef MP-WEIXIN
console.log('只在微信小程序执行')
// #endif

// #ifdef APP-PLUS
console.log('只在App平台执行')
// #endif

// #ifndef H5
console.log('在非H5平台执行')
// #endif
```

### 2. 平台差异处理
- **H5**: 不支持扫码、保存图片等原生功能，需要提供替代方案
- **小程序**: 受限于包大小，必须使用分包
- **App**: 需要处理安全区域、权限等问题

### 3. 性能优化建议
- 长列表必须使用虚拟列表
- 图片必须使用懒加载
- 合理使用缓存
- 避免频繁的DOM操作
