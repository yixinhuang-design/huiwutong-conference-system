## 🔧 每个页面的优化步骤

### 1. emoji替换为Font Awesome
```bash
# 查找页面中的emoji
grep -n "🔒\|📱\|👁️\|📢\|🏠\|👤\|📝\|⚙️\|🔔\|📊\|🎯\|📁\|🔍\|➕\|❌\|✅\|⭐\|⚠️" pages/xxx/xxx.vue
```

替换规则：
| emoji | Font Awesome | 用途 |
|-------|--------------|------|
| 🔒 | fa-lock | 密码、锁定 |
| 📱 | fa-mobile-alt | 手机、电话 |
| 👁️ | fa-eye | 查看、显示 |
| 👁️‍🗨️ | fa-eye-slash | 隐藏 |
| 📢 | fa-bullhorn | 通知、提醒 |
| 🏠 | fa-home | 首页、住宿 |
| 👤 | fa-user | 用户、人员 |
| 📝 | fa-edit | 编辑、填写 |
| ⚙️ | fa-cog | 设置 |
| 🔔 | fa-bell | 消息、通知 |
| 📊 | fa-chart-bar | 数据、统计 |
| 🎯 | fa-bullseye | 目标、主题 |
| 📁 | fa-folder | 文件夹、资料 |
| 🔍 | fa-search | 搜索、查找 |
| ➕ | fa-plus | 添加、新建 |
| ❌ | fa-times | 关闭、删除 |
| ✅ | fa-check | 完成、确认 |
| ⭐ | fa-star | 收藏、评价 |
| ⚠️ | fa-exclamation | 警告、注意 |

### 2. 统一卡片样式
添加到卡片的class：
```vue
<view class="card fade-in" style="border-radius: 24rpx;">
```

或在scoped style中：
```scss
.card {
  border-radius: 24rpx !important; /* 匹配app原型的12px */
}
```

### 3. 使用统一的颜色变量
替换硬编码颜色为变量：
```scss
/* 之前 */
background: linear-gradient(135deg, #3b82f6, #1d4ed8);

/* 之后 */
background: $icon-gradient-blue;
```

### 4. 统一状态标签
使用渐变背景的状态标签：
```vue
<view class="status-chip status-accent">进行中</view>
<view class="status-chip status-positive">已完成</view>
<view class="status-chip status-warn">待处理</view>
<view class="status-chip status-neutral">已归档</view>
```

### 5. 统一图标样式
使用渐变色图标类：
```vue
<view class="feature-icon-lg blue">
<view class="feature-icon-lg cyan">
<view class="feature-icon-lg green">
<view class="feature-icon-lg orange">
<view class="feature-icon-lg purple">
```

### 6. 添加fade-in动画
确保所有卡片和列表项有动画：
```vue
<view class="card fade-in">
<view class="list-item fade-in">
```
