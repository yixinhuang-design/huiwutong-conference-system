# 图标显示问题快速修复指南

## 问题现状

页面显示占位符而不是图标，说明 **Font Awesome 字体文件未正确加载**。

## 快速修复（3种方案）

### ⚡ 方案1：使用 Emoji 备用（最简单）

修改 `App.vue`，添加 Emoji 备用样式：

```vue
<style lang="scss">
@import './styles/common.scss';
@import './styles/font-awesome.scss';
@import './styles/icon-fallback.scss'; // 添加这行

/* 如果 Font Awesome 未加载，使用 emoji 备用 */
text.fa:not(:has(*)) {
  font-family: sans-serif !important;
}

text.fa-check:empty::before {
  content: '✓';
}

text.fa-times:empty::before {
  content: '✕';
}

text.fa-star:empty::before {
  content: '★';
}

text.fa-home:empty::before {
  content: '🏠';
}

text.fa-user:empty::before {
  content: '👤';
}
</style>
```

### 🔧 方案2：下载字体文件到本地（推荐）

1. **下载字体文件**

```bash
# 在项目根目录执行
cd G:\huiwutong新版合集\huiwutong-uniapp

# 创建字体目录
mkdir -p static/fonts

# 下载 Font Awesome 字体文件
curl -o static/fonts/fa-solid-900.woff2 \
  https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/webfonts/fa-solid-900.woff2

curl -o static/fonts/fa-solid-900.ttf \
  https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/webfonts/fa-solid-900.ttf

curl -o static/fonts/fa-brands-400.woff2 \
  https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/webfonts/fa-brands-400.woff2

curl -o static/fonts/fa-regular-400.woff2 \
  https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/webfonts/fa-regular-400.woff2
```

2. **修改 font-awesome.scss 中的路径**

```scss
@font-face {
  font-family: 'Font Awesome 6 Free';
  font-style: normal;
  font-weight: 900;
  src: url('/static/fonts/fa-solid-900.woff2') format('woff2'),
       url('/static/fonts/fa-solid-900.ttf') format('truetype');
}
```

## 临时解决方案

如果急需上线，可以使用以下方式临时替代：

### 1. 使用图片图标

```vue
<template>
  <image src="/static/icons/check.png" style="width: 48rpx; height: 48rpx;"></image>
</template>
```

### 2. 使用 base64 图标

```vue
<script>
export default {
  data() {
    return {
      icons: {
        check: 'data:image/svg+xml;base64,...',
        close: 'data:image/svg+xml;base64,...'
      }
    }
  }
}
</script>

<template>
  <image :src="icons.check" style="width: 48rpx; height: 48rpx;"></image>
</template>
```

### 3. 使用纯 CSS 图标

```vue
<template>
  <view class="icon icon-check"></view>
</template>

<style>
.icon-check {
  width: 48rpx;
  height: 48rpx;
  position: relative;
}

.icon-check::after {
  content: '';
  position: absolute;
  left: 16rpx;
  top: 4rpx;
  width: 12rpx;
  height: 24rpx;
  border: solid #333;
  border-width: 0 4rpx 4rpx 0;
  transform: rotate(45deg);
}
</style>
```
