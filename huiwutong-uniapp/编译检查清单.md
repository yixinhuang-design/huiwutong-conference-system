## 🚀 编译前检查

### 1. 依赖安装
```bash
npm install
# 或
yarn install
# 或
pnpm install
```

### 2. 检查Node版本
建议使用 Node.js 14+ 版本

### 3. 运行项目

**H5端**
```bash
npm run dev:h5
# 访问: http://localhost:8080
```

**微信小程序**
```bash
npm run dev:mp-weixin
# 使用微信开发者工具打开 dist/dev/mp-weixin 目录
```

**App端**
```bash
npm run dev:app
# 使用 HBuilderX 运行到模拟器或真机
```

### 问题1: "根目录缺少 index.html"

### 问题2: 样式文件引用错误
✅ **已解决** - 已创建 styles/common.scss 和 uni.scss

### 问题3: 页面路径错误
检查 pages.json 中的路径是否与实际文件路径一致

### 问题4: 组件引用错误
确保组件路径使用 `@/` 别名，例如：
```javascript
import NavBar from '@/components/nav-bar/nav-bar.vue'
```

## 🔧 开发建议

### 1. 使用条件编译
在需要区分平台的代码中使用条件编译：
```javascript
// #ifdef H5
console.log('H5平台')
// #endif

// #ifdef MP-WEIXIN
console.log('微信小程序')
// #endif

// #ifdef APP-PLUS
console.log('App平台')
// #endif
```

### 2. 使用平台适配工具
```javascript
import { getPlatformInfo } from '@/utils/platform'

const platformInfo = getPlatformInfo()
if (platformInfo.isH5) {
  // H5特定逻辑
}
```

### 3. 性能优化
- 长列表使用虚拟列表组件
- 图片使用懒加载组件
- 合理使用缓存

### 4. 分包加载
项目已配置分包，无需额外处理。编译器会自动分包。

## 📱 平台特定注意事项

### H5
- 路由模式使用 hash 模式
- 不支持原生扫码、保存相册等功能
- 使用 Intersection Observer 实现图片懒加载

### 微信小程序
- 需配置 appid（在 manifest.json 中）
- 包大小限制：主包2MB，总包20MB
- 已启用分包优化
- 需配置服务器域名白名单

### App
- iOS 需配置隐私描述
- Android 需配置权限
- 支持原生功能调用
