## 🎨 设计规范遵循

### guestbook.html 设计亮点

#### 颜色系统
- **头像渐变**: 蓝色(#2563eb)、绿色(#16a34a)、紫色(#9333ea)、橙色(#f97316)、青色(#0891b2)
- **强调色**: 蓝色渐变 (135deg, #2563eb 0%, #1e40af 100%)
- **背景色**: 浅灰色 (#f8f9fa, #f9f9f9)

#### 组件结构
1. **页面头部**
   - 状态栏 (时间、信号、电量)
   - 返回按钮 + 页面标题卡片
   - 功能说明卡片

2. **核心功能区**
   - 论坛统计卡片 (3列网格: 总留言数、参与人数、互动数)
   - 发言框 (输入框 + 话题标签按钮 + 发表按钮)
   - 帖子列表 (6个示例帖子)

3. **帖子卡片设计**
   ```
   ┌─ 头部 (40px圆形头像 + 用户信息 + 时间)
   ├─ 正文 (灰色背景文本框)
   ├─ 标签 (话题标签组)
   └─ 操作栏 (点赞 + 回复按钮)
   ```

4. **底部导航** (5个nav-item)
   - 首页、沟通、留言(active)、往期、我的

#### 交互体验
- **点赞功能**: 可切换liked状态，数字更新
- **发言框**: 支持多个标签选择，提交后清空
- **动画效果**: fade-in淡入动画 (300ms延迟递增)
- **响应式设计**: 移动端375×812px全适配
