## 🔧 手动调整Grid位置

### 步骤1: 打开座位区编辑

在Grid布局视图下：
1. 找到要编辑的座位区
2. 点击该座位区上的"编辑"按钮
3. 编辑模态框打开

### 步骤2: 找到Grid定位部分

在编辑模态框中，向下滚动找到 **"Grid多块布局定位"** 部分：

```
┌─────────────────────────────────────┐
│ 📍 Grid多块布局定位                   │
├─────────────────────────────────────┤
│ Grid列位置 (当前: 1 / span 4)        │
│ ┌──────────────────────────┬────┐  │
│ │ 1 / span 4              │自动 │  │
│ └──────────────────────────┴────┘  │
│                                     │
│ Grid行位置 (当前: 1 / span 2)        │
│ ┌──────────────────────────┬────┐  │
│ │ 1 / span 2              │自动 │  │
│ └──────────────────────────┴────┘  │
│                                     │
│ 💡 提示: 使用自动定位可根据...      │
└─────────────────────────────────────┘
```

### 步骤3: 手动修改Grid位置

#### 格式说明

**Grid列位置**: `起始列 / span 列数`
- 例: `1 / span 6` = 从第1列开始，占用6列
- 例: `7 / span 4` = 从第7列开始，占用4列

**Grid行位置**: `起始行 / span 行数`
- 例: `1 / span 2` = 从第1行开始，占用2行
- 例: `3 / span 3` = 从第3行开始，占用3行

#### 修改示例

假设要将某个座位区移到右上角：

```
当前: Grid列位置 = "1 / span 4"
修改为: "17 / span 6"  (右侧)

当前: Grid行位置 = "1 / span 2"
保持: "1 / span 2"     (顶部)
```

### 步骤4: 保存并预览

1. 修改完成后点击"保存"
2. 返回座位区列表
3. 点击"应用"重新应用Grid布局
4. 预览新的位置

### 步骤5: 重置为自动计算

如果手动设置不满意，点击 **"自动"** 按钮：

```
┌──────────────────────────┬────┐
│ 1 / span 4              │自动 │ ← 点这里
└──────────────────────────┴────┘
```

**效果**:
- Grid定位重置为自动计算
- 基于座位数重新计算最优大小

## ❌ 常见问题排查

### Q: 点击"Grid多块"没有反应？

**排查步骤**:
1. 打开浏览器控制台（F12）
2. 检查是否有错误信息
3. 刷新页面后重试
4. 检查是否有JS错误

### Q: 座位区显示错乱？

**排查步骤**:
1. 检查浏览器宽度是否足够（至少1024px）
2. 清除浏览器缓存
3. 尝试其他浏览器
4. 检查座位区数据是否完整

### Q: 手动编辑Grid位置不生效？

**排查步骤**:
1. 检查输入格式是否正确（如："1 / span 4"）
2. 确认数值不超出范围（列：1-20，行：1-10）
3. 点击"保存"确实是保存了
4. 点击"应用"重新应用布局

### Q: 切换回流式布局后座位区错位？

**排查步骤**:
1. 刷新页面
2. 在流式布局下重新拖拽座位区顺序
3. 手动保存数据

## 🎓 高级用法

### 自定义Grid大小

修改`layoutConfigs.conferenceHall.gridTemplate`：

```javascript
// 改为30列×15行
gridTemplate: {
    columns: 30,
    rows: 15,
    gap: 15
}
```

### 修改自动定位规则

编辑`calculateAreaGridSize()`函数中的映射：

```javascript
if (totalSeats <= 40) {
    // 自定义小区定义
    return { cols: 4, rows: 2 };
}
```

### 批量编辑Grid位置

通过编程方式修改：

```javascript
// 让所有座位区宽度相同
seatingAreas.value.forEach((area, i) => {
    area.gridLayout.gridColumn = `${i*5+1} / span 5`;
    area.gridLayout.gridRow = '1 / span 2';
});
```
