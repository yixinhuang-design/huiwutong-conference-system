## ✅ 问题1：centerOutward 编号算法修复

### 问题描述
中心扩展（centerOutward）编号方式显示不正确，特别是对于主席台（8列1行）的编号。

### 原因分析
偶数列的centerOutward算法逻辑有误，导致座位号计算错误。

### 修复方案

**文件**: seating-mgr.html (Lines 6035-6087)

修正了centerOutward算法：

#### 奇数列（如7列）
- 中心列（第4列）编号为 1
- 左侧从中心向外：2, 3, 4...
- 右侧从中心向外：2, 3, 4...
- 最终编号：7-5-3-1-2-4-6

#### 偶数列（如8列）
- 中心两列（第4列和第5列）
- 左侧编号从高到低：7-5-3-1
- 右侧编号从低到高：2-4-6-8
- 最终编号：7-5-3-1-2-4-6-8

### 修复代码
```javascript
case 'centerOutward':
    const colCount = area.rows?.[0]?.seats?.length || 1;
    let seatNum = actualCol;
    
    if (colCount % 2 === 1) {
        // 奇数列逻辑
        const centerCol = Math.ceil(colCount / 2);
        if (actualCol === centerCol) {
            seatNum = 1;
        } else if (actualCol < centerCol) {
            const distance = centerCol - actualCol;
            seatNum = 1 + distance * 2;
        } else {
            const distance = actualCol - centerCol;
            seatNum = distance * 2;
        }
    } else {
        // 偶数列逻辑（修复）
        const centerLeft = Math.floor(colCount / 2);
        const centerRight = centerLeft + 1;
        
        if (actualCol <= centerLeft) {
            const distance = centerLeft - actualCol;
            seatNum = colCount - distance;
        } else {
            const distance = actualCol - centerRight;
            seatNum = centerRight + distance;
        }
    }
    
    return startNumber + (actualRow - 1) * colCount + seatNum - 1;
```

## ⚙️ 技术细节

### centerOutward 算法原理

**目标**: 从中心向两侧扩展编号，中心号码最小

**对于8列位置**:
```
位置: 1  2  3  4 | 5  6  7  8
号码: 7  5  3  1 | 2  4  6  8
     ← 左边 中心→ ←右边 →
```

**算法逻辑**:
- 左侧（第1-4列）: 从右向左递减 (7,5,3,1)
- 右侧（第5-8列）: 从左向右递增 (2,4,6,8)

### 多块布局切换原理

```javascript
// switchLayout() 函数
const switchLayout = (layoutName) => {
    currentLayoutMode.value = layoutName;  // 记录当前模式
    
    // 更新座位区布局信息
    seatingAreas.value.forEach((area, index) => {
        area.gridLayout.displayLayout = layoutName === 'flowLayout' ? 'flow' : 'grid';
        
        // 如果是Grid模式，分配定位
        if (layoutName === 'conferenceHall') {
            assignGridPosition(area, index);
        }
    });
};
```
