## 🎓 关键技术点

### CSS Grid 核心概念

```css
/* 定义网格 */
display: grid;
grid-template-columns: repeat(20, 1fr);  /* 20等宽列 */
gap: 15px;                                /* 间距 */

/* 元素定位 */
grid-column: 7 / span 8;  /* 从第7列开始，跨8列 */
grid-row: 1;              /* 第1行 */
```

### Vue 3 响应式数据

