#### 3️⃣ 修改点3：座位编号系统 (第5748-5805行)

**目的**: 实现5种灵活的座位编号算法

**核心函数**:
```javascript
generateOptimizedSeatNumber(seat, area) {
  if (!seat || !area) return seat?.number || '';
  
  const mode = area.numberingMode || 'leftToRight';
  const start = (area.startNumber || 1) - 1;  // 调整为0基
  const {row = 1, col = 1} = seat;
  const {rows = 10, cols = 10} = area;
  
  let seatNum = 0;
  
  switch(mode) {
    case 'leftToRight':
      // 从左到右：(行-1)×列数 + 列
      seatNum = (row - 1) * cols + col;
      break;
      
    case 'rightToLeft':
      // 从右到左：(行-1)×列数 + (列数-列+1)
      seatNum = (row - 1) * cols + (cols - col + 1);
      break;
      
    case 'centerOutward':
      // 中心向外：按离中心距离排序
      // 计算到中心的距离
      const centerRow = (rows + 1) / 2;
      const centerCol = (cols + 1) / 2;
      const distance = Math.abs(row - centerRow) + Math.abs(col - centerCol);
      
      // 先按距离分组，再按行列编号
      let count = 0;
      for (let d = 0; d <= distance; d++) {
        for (let r = 1; r <= rows; r++) {
          for (let c = 1; c <= cols; c++) {
            const dist = Math.abs(r - centerRow) + Math.abs(c - centerCol);
            if (dist === d) {
              count++;
              if (r === row && c === col) {
                seatNum = count;
                break;
              }
            }
          }
          if (seatNum > 0) break;
        }
        if (seatNum > 0) break;
      }
      break;
      
    case 'alternatingLTRStart':
      // 蛇形①：奇数行L→R，偶数行R→L
      seatNum = (row - 1) * cols;
      if (row % 2 === 1) {
        seatNum += col;  // 奇数行：左→右
      } else {
        seatNum += (cols - col + 1);  // 偶数行：右→左
      }
      break;
      
    case 'alternatingRTLStart':
      // 蛇形②：奇数行R→L，偶数行L→R
      seatNum = (row - 1) * cols;
      if (row % 2 === 1) {
        seatNum += (cols - col + 1);  // 奇数行：右→左
      } else {
        seatNum += col;  // 偶数行：左→右
      }
      break;
      
    default:
      seatNum = seat.number || '';
  }
  
  return seatNum + start;
}
```

**5种编号方式详解**:

| 方式 | 名称 | 算法 | 应用场景 |
|------|------|------|---------|
| `leftToRight` | 左→右 | (行-1)×列数 + 列 | 标准线性编号 |
| `rightToLeft` | 右→左 | (行-1)×列数 + (列数-列+1) | 镜像编号 |
| `centerOutward` | 中心扩展 | 按离中心距离排序 | VIP/主宾席位 |
| `alternatingLTRStart` | 蛇形① | 奇偶行交替 | 走道配置 |
| `alternatingRTLStart` | 蛇形② | 逆向蛇形 | 高级布局 |

## 📐 数学原理

### 编号算法验证

#### 示例：2行×3列座位区

**数据结构**:
```
位置 (row, col):
(1,1) (1,2) (1,3)
(2,1) (2,2) (2,3)
```

**编号结果对比**:

```
leftToRight (左→右):
1  2  3
4  5  6

rightToLeft (右→左):
3  2  1
6  5  4

centerOutward (中心向外):
按距离排序，中心 (1.5, 1.5)
最近: (1,2)=1, (1,3)=2, (2,1)=3, (2,2)=4, (2,3)=5, (1,1)=6

alternatingLTRStart (蛇形①):
1  2  3      <- 奇数行：L→R
6  5  4      <- 偶数行：R→L

alternatingRTLStart (蛇形②):
3  2  1      <- 奇数行：R→L
4  5  6      <- 偶数行：L→R
```
