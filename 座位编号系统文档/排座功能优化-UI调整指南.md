# 排座功能 - UI界面调整座位编号指南

**更新日期**: 2026年2月28日  
**功能**: 通过UI界面直接调整座位编号方式  
**难度**: ⭐ 非常简单  

---

## 🎯 快速开始

### 步骤1: 打开座位区编辑面板

1. 在排座页面，找到要调整的**座位区**
2. 点击座位区名称旁的**编辑按钮** (🖊️ 铅笔图标)
3. 会弹出**座位区配置窗口**

### 步骤2: 找到编号方式设置

在配置窗口中，找到**"排座方式"**部分：

```
┌─────────────────────────────────────┐
│     排座方式                        │
├─────────────────────────────────────┤
│ 布局方向: [横向排列 ▼]             │
│                                     │
│ 座位编号方式: [左→右 ▼]            │
│   ├─ 左→右 (标准线性编号)          │
│   ├─ 右→左 (镜像编号)              │
│   ├─ 中心扩展 (VIP优先)            │
│   ├─ 蛇形① (奇偶行交替)           │
│   └─ 蛇形② (逆向交替)             │
│                                     │
│ 起始编号:  [1] [-10] [+10]        │
│   (座位编号将从此数字开始)         │
│                                     │
│ 预览: 当前设置为 "左→右" 编号      │
│      从 1 开始                      │
└─────────────────────────────────────┘
```

### 步骤3: 选择编号方式

**点击下拉菜单**，选择想要的编号方式：

| 选项 | 效果 | 适用场景 |
|------|------|---------|
| **左→右** | 1 2 3 / 4 5 6 | 标准编号 |
| **右→左** | 3 2 1 / 6 5 4 | 镜像编号 |
| **中心扩展** | 从中心向外 | VIP座位区 |
| **蛇形①** | 奇偶行交替 | 走道设计 |
| **蛇形②** | 逆向交替 | 高级布局 |

### 步骤4: 调整起始编号（可选）

如果要从特定号码开始编号：

```
默认: 从1开始 → 座位号为 1, 2, 3, ...
改为100: 从100开始 → 座位号为 100, 101, 102, ...
改为1001: 从1001开始 → 座位号为 1001, 1002, 1003, ...
```

**调整方法**:
- 📝 直接输入数字到输入框
- ➕ 点击 **[+10]** 增加10
- ➖ 点击 **[-10]** 减少10

### 步骤5: 预览和保存

1. 查看下方的**预览**信息，确认设置无误：
   ```
   预览: 当前设置为 "中心扩展" 编号，从 100 开始
   ```

2. 点击窗口底部的**保存按钮**

3. 座位区将按新的编号方式重新显示座位号

## 📊 5种编号方式详解

### 1️⃣ 左→右 (leftToRight)

**效果**: 从左到右，从上到下依次编号

```
1  2  3
4  5  6
7  8  9
```

**适用**: 标准会议座位

**特点**: 最直观，最常用

---

### 2️⃣ 右→左 (rightToLeft)

**效果**: 从右到左，从上到下依次编号

```
3  2  1
6  5  4
9  8  7
```

**适用**: 某些国家的阅读习惯

**特点**: 与左→右相反

---

### 3️⃣ 中心扩展 (centerOutward)

**效果**: 从中间向四周扩展，中间座位号最小

```
8  5  2
7  4  1
10 6  3
```

**适用**: VIP区域、主宾座位

**特点**: 突出中心位置的重要性

---

### 4️⃣ 蛇形① (alternatingLTRStart)

**效果**: 奇数行从左→右，偶数行从右→左

```
1  2  3    ← 第1行：左→右
6  5  4    ← 第2行：右→左
7  8  9    ← 第3行：左→右
```

**适用**: 有中间走道的座位布局

**特点**: 适合U型、走廊式布局

---

### 5️⃣ 蛇形② (alternatingRTLStart)

**效果**: 奇数行从右→左，偶数行从左→右

```
3  2  1    ← 第1行：右→左
4  5  6    ← 第2行：左→右
9  8  7    ← 第3行：右→左
```

**适用**: 高级布局配置

**特点**: 与蛇形①相反

## 💡 常见场景应用

### 场景1: 标准会议座位

```
配置建议:
- 编号方式: 左→右
- 起始编号: 1
- 说明: 最直观，参会者容易找到座位
```

```
配置建议:
- 编号方式: 中心扩展
- 起始编号: 1001
- 说明: 突出中心VIP座位，编号以1001开头彰显尊贵
```

```

- 编号方式: 蛇形①
```

```
配置建议:
- 分会场1: 左→右, 起始编号 1
- 分会场2: 左→右, 起始编号 101
- 分会场3: 左→右, 起始编号 201
- 说明: 不同分会场使用不同的起始编号，易于区分
```

## ⚡ 快速操作技巧

### 技巧1: 快速修改起始编号

```
要改为100: 直接输入100
要改为1000: 输入1000
要增加10: 点击 [+10] 按钮
要减少10: 点击 [-10] 按钮
```

### 技巧2: 对比效果

1. 选择一种编号方式
2. 查看预览信息
3. 如不满意，重新选择另一种
4. 可以多次尝试，直到满意为止

### 技巧3: 批量修改

如果有多个座位区需要调整：

1. 逐个打开每个座位区的编辑面板
2. 选择相同的编号方式
3. 为不同座位区设置不同的起始编号（避免重号）
4. 逐个保存

## ❓ 常见问题

### Q1: 修改编号方式后，座位位置会改变吗？

**A**: 不会。修改编号方式只改变**座位号的显示**，不改变座位的物理位置。座位的行列位置保持不变。

### Q2: 起始编号可以从0开始吗？

**A**: 当前最小值是1。如确实需要从0开始，可以修改代码中的最小值限制。

### Q3: 修改后如何撤销？

**A**: 
- **保存前**: 点击取消或关闭窗口，不保存即可
- **保存后**: 重新编辑，选择原来的配置，再次保存

### Q4: 不同座位区可以用相同的编号方式吗？

**A**: 可以，但要注意起始编号不要重复。例如：
- A区: 1-99号
- B区: 100-199号
- C区: 200-299号

### Q5: 编号方式会影响智能排座功能吗？

**A**: 不会。智能排座按座位的行列位置分配，不受编号方式影响。

### Q6: 座位搜索会受编号方式影响吗？

**A**: 会。搜索座位号时，需要使用当前编号方式的座位号。例如，如果使用"右→左"方式，座位号会是3、2、1而不是1、2、3。

## 🎓 配置最佳实践

### ✅ 推荐做法

1. **统一性**: 同一会议的座位区使用相同的编号方式
2. **可读性**: 优先选择"左→右"（最直观）
3. **避免重号**: 为不同座位区设置不同的起始编号
4. **易于查找**: 考虑参会者的习惯和视觉流向
5. **提前规划**: 编辑配置时先核实尺寸和布局

### ❌ 避免做法

1. **混乱编号**: 同一会议不同座位区使用不同编号方式
2. **重号**: 多个座位区编号重叠
3. **过大起始号**: 起始号过大增加参会者困惑
4. **频繁改动**: 避免编号确定后再改动
