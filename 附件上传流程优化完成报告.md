## 📝 代码修改详情

### 1. 优化 `handleFileUpload()` 方法

**优化前（需要手动保存）**：
```javascript
async handleFileUpload(event) {
    const files = event.target.files;
    if (!files || files.length === 0) return;

    // 检查是否已创建日程（需要scheduleId）
    if (!this.scheduleForm.id) {
        this.$showToast('请先保存日程，再上传课件', 'warning');
        event.target.value = '';
        return;  // 阻止上传
    }

    // 上传逻辑...
}
```

**优化后（自动保存）**：
```javascript
async handleFileUpload(event) {
    const files = event.target.files;
    if (!files || files.length === 0) return;

    // 优化：如果是新建模式，先自动保存草稿获取scheduleId
    if (!this.scheduleForm.id) {
        // 验证必填项
        if (!this.scheduleForm.title || !this.scheduleForm.startTime || 
            !this.scheduleForm.endTime || !this.scheduleForm.location) {
            this.$showToast('请先填写日程主题、时间和地点', 'warning');
            event.target.value = '';
            return;
        }

        // 验证时间
        const start = new Date(this.scheduleForm.startTime);
        const end = new Date(this.scheduleForm.endTime);
        if (start >= end) {
            this.$showToast('结束时间必须晚于开始时间', 'warning');
            event.target.value = '';
            return;
        }

        // 自动保存草稿
        try {
            this.$showLoading('正在保存日程信息...');
            
            const response = await ScheduleAPI.createSchedule(
                this.conference.id, 
                this.scheduleForm
            );
            
            if (response && response.data) {
                // 更新scheduleForm的ID，现在可以上传附件了
                this.scheduleForm.id = response.data.id;
                this.isEditMode = true; // 切换为编辑模式
                this.$showToast('日程信息已保存，正在上传课件...', 'success');
            } else {
                throw new Error('保存日程失败');
            }
        } catch (error) {
            this.$hideLoading();
            console.error('保存日程失败:', error);
            this.$showToast('保存日程失败，请重试', 'error');
            event.target.value = '';
            return;
        }
    }

    // 上传附件逻辑...
}
```

### 2. 优化 `saveSchedule()` 错误提示

**优化前**：
```javascript
catch (error) {
    this.$hideLoading();
    console.error('保存日程失败:', error);
    this.$showToast('操作失败，请重试', 'error');
}
```

**优化后**：
```javascript
catch (error) {
    this.$hideLoading();
    console.error('保存日程失败:', error);
    
    // 更详细的错误提示
    const errorMsg = error.response?.data?.message || error.message || '操作失败，请重试';
    this.$showToast('保存失败：' + errorMsg, 'error');
}
```

## 📊 技术实现细节

### 状态流转图

```
新建日程模式（isEditMode = false）
    ↓
用户点击上传附件
    ↓
验证必填项
    ✅ 通过 → 继续
    ❌ 失败 → 提示并返回
    ↓
验证时间逻辑
    ✅ 通过 → 继续
    ❌ 失败 → 提示并返回
    ↓
自动调用 createSchedule()
    ↓
获得 scheduleId
    ↓
切换为编辑模式（isEditMode = true）
    ↓
上传附件到服务器
    ↓
更新本地附件列表
    ↓
显示成功提示
```

### 关键技术点

1. **条件判断**
```javascript
if (!this.scheduleForm.id) {
    // 新建模式：先保存，再上传
} else {
    // 编辑模式：直接上传
}
```

2. **模式切换**
```javascript
this.isEditMode = true; // 自动切换为编辑模式
```

3. **错误回滚**
```javascript
catch (error) {
    event.target.value = ''; // 清空文件选择，允许重试
    return; // 阻止后续上传流程
}
```
