## 常见问题排查

### Q1：Legend不显示
**可能原因**：
- `currentLayoutMode !== 'conferenceHall'`（使用了其他布局模式）
- CSS类`.seat-legend-global`的`display: none;`

**解决方案**：
1. 确认当前是Grid多块模式(conferenceHall)
2. 检查浏览器开发者工具，看是否有display:none
3. 检查条件`v-if="seatingAreas.length > 0 && currentLayoutMode === 'conferenceHall'"`

### Q2：768px宽度仍然是Flex-column
**可能原因**：
- 浏览器缓存未清除
- CSS修改未生效

**解决方案**：
1. 清除浏览器缓存（Ctrl+Shift+Delete）
2. 硬刷新页面（Ctrl+Shift+R）
3. 检查开发者工具Network，确认seating-mgr.html是新版本

### Q3：表单标题仍显示旧名称
**可能原因**：
- 前端代码未更新
- Vue组件缓存

**解决方案**：
1. 清除浏览器缓存
2. 重新启动开发服务器
3. 检查seating-mgr.html中的标题是否已修改

### Q4：表单中图标不显示
**可能原因**：
- Font Awesome库未加载
- 图标类名错误

**解决方案**：
1. 检查是否引入了Font Awesome库
2. 验证图标类名（如`fas fa-expand`、`fas fa-sort`等）
3. 查看浏览器控制台是否有加载错误
